INSERTPS — 插入标量单精度浮点值

操作码/指令 Op/En 64/32位模式支持 CPUID 特征标志 描述
66 0F 3A 21 /r ib INSERTPS xmm1, xmm2/m32, imm8 A V/V SSE4_1 将由 imm8 选择的来自 xmm2/m32 的单精度浮点值插入到 xmm1 中由 imm8 指定的目标元素位置，并根据 imm8 的指示将 xmm1 中的目标元素清零。
VEX.128.66.0F3A.WIG 21 /r ib VINSERTPS xmm1, xmm2, xmm3/m32, imm8 B V/V AVX 将由 imm8 选择的来自 xmm3/m32 的单精度浮点值插入，并与 xmm2 中的值在由 imm8 指定的目标元素位置进行合并，写入结果并根据 imm8 的指示将 xmm1 中的目标元素清零。
EVEX.128.66.0F3A.W0 21 /r ib VINSERTPS xmm1, xmm2, xmm3/m32, imm8 C V/V AVX512F 将由 imm8 选择的来自 xmm3/m32 的单精度浮点值插入，并与 xmm2 中的值在由 imm8 指定的目标元素位置进行合并，写入结果并根据 imm8 的指示将 xmm1 中的目标元素清零。

描述¶

（寄存器源形式）

将单精度标量浮点元素复制到 128 位向量寄存器中。立即数操作数有三个字段，其中 ZMask 位指定目的操作数的哪些元素将被设置为零，Count_D 位指定目的操作数的哪个元素将被该标量值覆盖，对于向量寄存器源，Count_S 位指定要复制源操作数的哪个元素。当标量源是内存操作数时，Count_S 位会被忽略。

（内存源形式）

从 32 位内存位置加载浮点元素，并将其写入立即数操作数的 Count_D 位所指示的位置的第一个源操作数中。存储到目的操作数中，并根据立即数操作数的 ZMask 位将目的操作数中的对应元素清零。

128 位传统 SSE 版本：第一个源寄存器是 XMM 寄存器。第二个源操作数可以是 XMM 寄存器或 32 位内存位置。目的操作数与第一个源 XMM 寄存器相同，并且对应寄存器目的高位 (MAXVL-1:128) 保持不变。

VEX.128 和 EVEX 编码版本：目的寄存器和第一个源寄存器是 XMM 寄存器。第二个源操作数可以是 XMM 寄存器或 32 位内存位置。对应寄存器目的的高位 (MAXVL-1:128) 清零。

如果 VINSERTPS 编码为 VEX.L= 1，则尝试执行编码为 VEX.L= 1 的指令将导致 #UD 异常。

操作¶

VINSERTPS (VEX.128 and EVEX Encoded Version)¶

IF (SRC = REG) THEN COUNT_S := imm8[7:6]
    ELSE COUNT_S := 0
COUNT_D := imm8[5:4]
ZMASK := imm8[3:0]
CASE (COUNT_S) OF
    0: TMP := SRC2[31:0]
    1: TMP := SRC2[63:32]
    2: TMP := SRC2[95:64]
    3: TMP := SRC2[127:96]
ESAC;
CASE (COUNT_D) OF
    0: TMP2[31:0] := TMP
        TMP2[127:32] := SRC1[127:32]
    1: TMP2[63:32] := TMP
        TMP2[31:0] := SRC1[31:0]
        TMP2[127:64] := SRC1[127:64]
    2: TMP2[95:64] := TMP
        TMP2[63:0] := SRC1[63:0]
        TMP2[127:96] := SRC1[127:96]
    3: TMP2[127:96] := TMP
        TMP2[95:0] := SRC1[95:0]
ESAC;
IF (ZMASK[0] = 1) THEN DEST[31:0] := 00000000H
    ELSE DEST[31:0] := TMP2[31:0]
IF (ZMASK[1] = 1) THEN DEST[63:32] := 00000000H
    ELSE DEST[63:32] := TMP2[63:32]
IF (ZMASK[2] = 1) THEN DEST[95:64] := 00000000H
    ELSE DEST[95:64] := TMP2[95:64]
IF (ZMASK[3] = 1) THEN DEST[127:96] := 00000000H
    ELSE DEST[127:96] := TMP2[127:96]
DEST[MAXVL-1:128] := 0

    ELSE DEST[31:0] := TMP2[31:0]
