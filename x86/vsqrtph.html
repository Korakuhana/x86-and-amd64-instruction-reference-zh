VSQRTPH — 计算打包 FP16 值的平方根

操作码/指令 Op/En 64/32位模式支持 CPUID 功能标志 描述

描述¶

此指令对源操作数中的值执行打包 FP16 平方根计算，并将打包 FP16 结果存储在目的操作数中。在写掩码控制下更新目的操作数中的元素。

操作¶

VSQRTPH dest{k1}, src¶

VL = 128, 256 or 512
KL := VL/16
FOR i := 0 to KL-1:
    IF k1[i] or *no writemask*:
        IF SRC is memory and (EVEX.b = 1):
            tsrc := src.fp16[0]
        ELSE:
            tsrc := src.fp16[i]
        DEST.fp16[i] := SQRT(tsrc)
    ELSE IF *zeroing*:
        DEST.fp16[i] := 0
    //else DEST.fp16[i] remains unchanged
DEST[MAXVL-1:VL] := 0
