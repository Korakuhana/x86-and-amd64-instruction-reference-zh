PMOVMSKB — 移动字节掩码

操作码/指令 Op/En 64/32位模式支持 CPUID 特征标志 描述
NP 0F D7 /r1 PMOVMSKB reg, mm RM V/V SSE 将 mm 的字节掩码移至 reg 中。r32 或 r64 的高位清零
66 0F D7 /r PMOVMSKB reg, xmm RM V/V SSE2 将 xmm 的字节掩码移至 reg 中。r32 或 r64 的高位清零
VEX.128.66.0F.WIG D7 /r VPMOVMSKB reg, xmm1 RM V/V AVX 将 xmm1 的字节掩码移至 reg 中。r32 或 r64 的高位用零填充。
VEX.256.66.0F.WIG D7 /r VPMOVMSKB reg, ymm1 RM V/V AVX2 将 ymm1 的 32 位掩码移至 reg 中。r64 的高位用零填充。

1. 请参阅《Intel® 64 和 IA-32 架构软件开发人员手册》第 2A 卷第 2.5 节“Intel® AVX 和 Intel® SSE 指令异常分类”中的注释，以及《Intel® 64 和 IA-32 架构软件开发人员手册》第 3B 卷第 23.25.3 节“操作 MMX 寄存器的传统 SIMD 指令的异常条件”中的注释。

指令操作数编码¶

Op/En 操作数 1 操作数 2 操作数 3 操作数 4
RM ModRM:reg (w) ModRM:r/m (r) N/A N/A

描述¶

由源操作数（第二个操作数）的每个字节的最高有效位创建一个掩码，并将结果存储在目的操作数（第一个操作数）的低位字节或字中。

对于 64 位源操作数，字节掩码为 8 位；对于 128 位源操作数，字节掩码为 16 位；对于 256 位源操作数，字节掩码为 32 位。目的操作数是通用寄存器。

在 64 位模式下，当与 REX.R 前缀结合使用时，该指令可以访问其他寄存器（XMM8-XMM15，R8-R15）。在 64 位模式下，默认操作数大小为 64 位。

传统 SSE 版本：源操作数是 MMX 技术寄存器。

128位传统 SSE 版本：源操作数是 XMM 寄存器。

VEX.128 编码版本：源操作数是 XMM 寄存器。

VEX.256 编码版本：源操作数是 YMM 寄存器。

注意：VEX.vvvv 已保留，且必须为 1111b。

操作¶

PMOVMSKB（配合 64 位源操作数和 r32）¶

r32[0] := SRC[7];
r32[1] := SRC[15];
(* Repeat operation for bytes 2 through 6 *)
r32[7] := SRC[63];
r32[31:8] := ZERO_FILL;

(V)PMOVMSKB（配合 128 位源操作数和 r32）¶

r32[0] := SRC[7];
r32[1] := SRC[15];
(* Repeat operation for bytes 2 through 14 *)
r32[15] := SRC[127];
r32[31:16] := ZERO_FILL;

VPMOVMSKB（配合 256 位源操作数和 r32）¶

r32[0] := SRC[7];
r32[1] := SRC[15];
(* Repeat operation for bytes 3rd through 31*)
r32[31] := SRC[255];

PMOVMSKB（配合 64 位源操作数和 r64）¶

r64[0] := SRC[7];
r64[1] := SRC[15];
(* Repeat operation for bytes 2 through 6 *)
r64[7] := SRC[63];
r64[63:8] := ZERO_FILL;

(V)PMOVMSKB（配合 128 位源操作数和 r64）¶

r64[0] := SRC[7];
r64[1] := SRC[15];
(* Repeat operation for bytes 2 through 14 *)
r64[15] := SRC[127];
r64[63:16] := ZERO_FILL;

VPMOVMSKB（配合 256 位源操作数和 r64）¶

r64[0] := SRC[7];
r64[1] := SRC[15];
(* Repeat operation for bytes 2 through 31*)
r64[31] := SRC[255];
r64[63:32] := ZERO_FILL;

受影响的标志位¶

无。

数值异常¶

无。

其他异常¶

请参阅表 2-24，“第 7 类异常条件”，此外：

#UD 如果 VEX.vvvv ≠ 1111B。
