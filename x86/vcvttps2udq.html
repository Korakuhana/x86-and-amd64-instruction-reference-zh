VCVTTPS2UDQ — 使用截断将打包的单精度浮点值转换为打包的无符号双字整数值

描述¶

使用截断将源操作数中的打包单精度浮点值转换为目的操作数中的打包无符号双字整数。

当转换不精确时，返回截断的值（向零舍入）。如果转换后的结果无法在目的格式中表示，则会引发浮点无效异常；如果该异常被屏蔽，则返回整数值 2w – 1，其中 w 代表目的格式中的位数。

EVEX 编码版本：源操作数是 ZMM/YMM/XMM 寄存器、512/256/128 位内存位置，或从 32 位内存位置广播的 512/256/128 位向量。目的操作数是根据写掩码 k1 有条件更新的 ZMM/YMM/XMM 寄存器。

注意：EVEX.vvvv 保留且必须为 1111b，否则指令会导致 #UD。

操作¶

VCVTTPS2UDQ（EVEX 编码版本）当源操作数为寄存器时¶

(KL, VL) = (4, 128), (8, 256), (16, 512)
FOR j := 0 TO KL-1
    i := j * 32
    IF k1[j] OR *no writemask*
        THEN DEST[i+31:i] :=
            Convert_Single_Precision_Floating_Point_To_UInteger_Truncate(SRC[i+31:i])
        ELSE
            IF *merging-masking* ; merging-masking
                THEN *DEST[i+31:i] remains unchanged*
                ELSE ; zeroing-masking
                    DEST[i+31:i] := 0
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0

VCVTTPS2UDQ（EVEX 编码版本）当源操作数为内存源时¶

(KL, VL) = (4, 128), (8, 256), (16, 512)
FOR j := 0 TO KL-1
    i := j * 32
    IF k1[j] OR *no writemask*
        THEN
            IF (EVEX.b = 1)
                THEN
                    DEST[i+31:i] :=
            Convert_Single_Precision_Floating_Point_To_UInteger_Truncate(SRC[31:0])
                ELSE
                    DEST[i+31:i] :=
            Convert_Single_Precision_Floating_Point_To_UInteger_Truncate(SRC[i+31:i])
            FI;
        ELSE
            IF *merging-masking* ; merging-masking
                THEN *DEST[i+31:i] remains unchanged*
                ELSE ; zeroing-masking
                    DEST[i+31:i] := 0
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0
