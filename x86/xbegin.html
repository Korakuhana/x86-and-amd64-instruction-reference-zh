XBEGIN — 事务开启

操作码/指令 Op/En 64/32位模式支持 CPUID 功能标志 描述
C7 F8 XBEGIN rel16 A V/V RTM 指定 RTM 区域的开始。提供 16 位相对偏移量以计算后备指令地址，在 RTM 终止后，执行将在此地址处恢复。
C7 F8 XBEGIN rel32 A V/V RTM 指定 RTM 区域的开始。提供 32 位相对偏移量以计算后备指令地址，在 RTM 终止后，执行将在此地址处恢复。

指令操作数编码¶

Op/En 操作数 1 操作数 2 操作数 3 操作数 4
A Offset N/A N/A N/A

描述¶

XBEGIN 指令指定 RTM 代码区域的开始。如果逻辑处理器尚未处于事务执行中，则 XBEGIN 指令使逻辑处理器过渡到事务执行中。使逻辑处理器过渡到事务执行中的 XBEGIN 指令被称为最外层 XBEGIN 指令。该指令还指定了一个相对偏移量，用于计算事务终止后后备代码路径的地址。（与到相对偏移量的近跳转不同，使用 16 位操作数大小不会导致该地址被截断为 16 位。）

在 RTM 终止时，逻辑处理器将丢弃在 RTM 执行期间执行的所有架构寄存器和内存更新，并恢复到与最外层 XBEGIN 指令对应的架构状态。终止后的后备地址是从最外层 XBEGIN 指令计算得出的。

在暂停读取地址跟踪区域中执行 XBEGIN 会导致事务终止。

操作¶

XBEGIN¶

IF RTM_NEST_COUNT < MAX_RTM_NEST_COUNT AND SUSLDTRK_ACTIVE = 0
    THEN
        RTM_NEST_COUNT++
        IF RTM_NEST_COUNT = 1 THEN
            IF 64-bit Mode
                THEN
                    IF OperandSize = 16
                        THEN fallbackRIP := RIP + SignExtend64(rel16);
                        ELSE fallbackRIP := RIP + SignExtend64(rel32);
                    FI;
                    IF fallbackRIP is not canonical
                        THEN #GP(0);
                    FI;
                ELSE
                    IF OperandSize = 16
                        THEN fallbackEIP := EIP + SignExtend32(rel16);
                        ELSE fallbackEIP := EIP + rel32;
                    FI;
                    IF fallbackEIP outside code segment limit
                        THEN #GP(0);
                    FI;
            FI;
            RTM_ACTIVE := 1
            Enter RTM Execution (* 记录寄存器状态，开始跟踪内存状态 *)
        FI; (* RTM_NEST_COUNT = 1 *)
    ELSE (* RTM_NEST_COUNT = MAX_RTM_NEST_COUNT OR SUSLDTRK_ACTIVE = 1 *)
        GOTO RTM_ABORT_PROCESSING
FI;
(* 对于在 RTM 执行期间遇到的任何 RTM 终止条件 *)
RTM_ABORT_PROCESSING:
    Restore architectural register state
    Discard memory updates performed in transaction
    Update EAX with status
    RTM_NEST_COUNT := 0
    RTM_ACTIVE := 0
    SUSLDTRK_ACTIVE := 0
    IF 64-bit mode
        THEN
            RIP := fallbackRIP
        ELSE
            EIP := fallbackEIP
    FI;
END

受影响的标志位¶

无。

XBEGIN unsigned int _xbegin( void );

SIMD 浮点异常¶

无。

保护模式异常¶

#UD 如果 CPUID.(EAX=7, ECX=0):EBX.RTM[bit 11]=0。
如果使用了 LOCK 前缀。
#GP(0) 如果后备地址在 CS 段之外。

实地址模式异常¶

#GP(0) 如果后备地址超出 0000H 到 FFFFH 的地址空间。
#UD 如果 CPUID.(EAX=7, ECX=0):EBX.RTM[bit 11]=0。
如果使用了 LOCK 前缀。

虚拟-8086 模式异常¶

#GP(0) 如果后备地址超出 0000H 到 FFFFH 的地址空间。
#UD 如果 CPUID.(EAX=7, ECX=0):EBX.RTM[bit 11]=0。
如果使用了 LOCK 前缀。

兼容模式异常¶

与保护模式下的异常相同。

64位模式异常¶

#UD 如果 CPUID.(EAX=7, ECX=0):EBX.RTM[bit 11] = 0。
如果使用了 LOCK 前缀。
#GP(0) 如果后备地址为非规范形式。
