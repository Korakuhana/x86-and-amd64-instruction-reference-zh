XTEST — 测试是否处于事务执行中

操作码/指令 Op/En 64/32位模式支持 CPUID 功能标志 描述
NP 0F 01 D6 XTEST ZO V/V HLE or RTM 测试是否在事务区域中执行。

指令操作数编码¶

Op/En 操作数 1 操作数 2 操作数 3 操作数 4
ZO N/A N/A N/A N/A

描述¶

XTEST 指令查询事务执行状态。如果该指令在以事务方式执行的 RTM 区域或以事务方式执行的 HLE 区域内部执行，则清除 ZF 标志，否则设置该标志。

操作¶

XTEST¶

IF (RTM_ACTIVE = 1 OR HLE_ACTIVE = 1)
    THEN
        ZF := 0
    ELSE
        ZF := 1
FI;

受影响的标志位¶

如果该指令以事务方式执行，则清除 ZF 标志；否则将其设置为 1。CF、OF、SF、PF 和 AF 标志被清零。

Intel C/C++ 编译器等价内部函数¶

XTEST int _xtest( void );

SIMD 浮点异常¶

无。

其他异常¶

#UD 如果 CPUID.(EAX=7, ECX=0):EBX.HLE[bit 4] = 0 且 CPUID.(EAX=7, ECX=0):EBX.RTM[bit 11] = 0。
如果使用了 LOCK 前缀。
