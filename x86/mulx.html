MULX — 无影响标志位的无符号乘法

操作码/指令 Op/En 64/32位模式 CPUID 特征标志 描述
VEX.LZ.F2.0F38.W0 F6 /r MULX r32a, r32b, r/m32 RVM V/V BMI2 将 r/m32 与 EDX 进行无符号乘法，不影响算术标志位。
VEX.LZ.F2.0F38.W1 F6 /r MULX r64a, r64b, r/m64 RVM V/N.E. BMI2 将 r/m64 与 RDX 进行无符号乘法，不影响算术标志位。

指令操作数编码¶

Op/En 操作数 1 操作数 2 操作数 3 操作数 4
RVM ModRM:reg (w) VEX.vvvv (w) ModRM:r/m (r) RDX/EDX is implied 64/32 bits source

描述¶

对隐式源操作数（EDX/RDX）与指定的源操作数（第三个操作数）进行无符号乘法，并将结果的低半部分存储在第二个目的操作数（第二个操作数）中，将结果的高半部分存储在第一个目的操作数（第一个操作数）中，而不读取或写入算术标志位。这使得软件能够将带进位加法操作和乘法操作交织在一起，从而实现高效的编程。

如果第一个和第二个操作数相同，则它将包含乘法结果的高半部分。

此指令在实模式和虚拟-8086模式下不受支持。如果不在64位模式下，操作数大小始终为32位。在64位模式下，64位操作数大小需要 VEX.W1。在非64位模式下，VEX.W1 将被忽略。尝试在 VEX.L 不等于 0 的情况下执行此指令将导致 #UD 异常。

操作¶

// DEST1: ModRM:reg
// DEST2: VEX.vvvv
IF (OperandSize = 32)
    SRC1 := EDX;
    DEST2 := (SRC1*SRC2)[31:0];
    DEST1 := (SRC1*SRC2)[63:32];
ELSE IF (OperandSize = 64)
    SRC1 := RDX;
    DEST2 := (SRC1*SRC2)[63:0];
    DEST1 := (SRC1*SRC2)[127:64];
FI
