SAVEPREVSSP — 保存前一个影子栈指针

操作码/指令 Op/En 64/32位模式支持 CPUID 特征标志 描述
F3 0F 01 EA (mod!=11, /5, RM=010) SAVEPREVSSP ZO V/V CET_SS 在前一个影子栈上保存一个恢复影子栈（restore-shadow-stack）令牌。

指令操作数编码¶

Op/En 操作数 1 操作数 2 操作数 3 操作数 4
ZO N/A N/A N/A N/A

描述¶

在下一个 8 字节对齐的边界处，在前一个影子栈上压入一个恢复影子栈（restore-shadow-stack）令牌。前一个 SSP 从当前影子栈顶部的 previous-ssp 令牌中获取。

操作¶

IF CPL = 3
    IF (CR4.CET & IA32_U_CET.SH_STK_EN) = 0
        THEN #UD; FI;
ELSE
    IF (CR4.CET & IA32_S_CET.SH_STK_EN) = 0
        THEN #UD; FI;
FI;
IF SSP not aligned to 8 bytes
    THEN #GP(0); FI;
(* 从当前影子栈弹出 “previous-ssp” 令牌 *)
previous_ssp_token = ShadowStackPop8B(SSP)
(* 如果 CF 标志指示当前影子栈上存在对齐空洞，则弹出该对齐空洞 *)
(* 请注意，对齐空洞必须为零，且仅在传统/兼容模式下才能存在 *)
IF RFLAGS.CF == 1 AND (IA32_EFER.LMA AND CS.L)
    #GP(0)
FI;
IF RFLAGS.CF == 1
    must_be_zero = ShadowStackPop4B(SSP)
    IF must_be_zero != 0 THEN #GP(0)
FI;
(* 前一个 SSP 令牌必须设置位 1 *)
IF ((previous_ssp_token & 0x02) == 0)
    THEN #GP(0); (* 位 1 为 0 *)
IF ((IA32_EFER.LMA AND CS.L) = 0 AND previous_ssp_token [63:32] != 0)
THEN #GP(0); FI; (* 如果为兼容/传统模式且 SSP 不在 4G 内 *)
(* 将 previous_ssp_token 中的 Prev SSP 保存到旧影子栈的下一个 8 字节对齐地址处 *)
old_SSP = previous_ssp_token & ~0x03
temp := (old_SSP | (IA32_EFER.LMA & CS.L));
Shadow_stack_store 4 bytes of 0 to (old_SSP - 4)
old_SSP := old_SSP & ~0x07;
Shadow_stack_store 8 bytes of temp to (old_SSP - 8)

受影响的标志位¶

无。

C/C++ 编译器等价内部函数¶

SAVEPREVSSP void _saveprevssp(void);

保护模式异常¶

#UD 如果使用了 LOCK 前缀。
如果 CR4.CET = 0。
如果 CPL = 3 且 IA32_U_CET.SH_STK_EN = 0。
如果 CPL < 3 且 IA32_S_CET.SH_STK_EN = 0。
#GP(0) 如果 SSP 未 8 字节对齐。
如果影子栈上的对齐空洞不为 0。
如果 previous-ssp 令牌的位 1 未设置为 1。
如果处于 32 位/兼容模式，且 previous-ssp 令牌中记录的 SSP 超出 4G。
#PF(错误码) 如果发生缺页异常。

实地址模式异常¶

#UD 在实地址模式下无法识别 SAVEPREVSSP 指令。

虚拟-8086 模式异常¶

#UD 在虚拟-8086 模式下无法识别 SAVEPREVSSP 指令。

兼容模式异常¶

与保护模式下的异常相同。

64位模式异常¶

#UD 如果使用了 LOCK 前缀。
如果 CR4.CET = 0。
如果 CPL = 3 且 IA32_U_CET.SH_STK_EN = 0。
如果 CPL < 3 且 IA32_S_CET.SH_STK_EN = 0。
#GP(0) 如果 SSP 未 8 字节对齐。
如果进位标志 (CF) 被设置。
如果 previous-ssp 令牌的位 1 未设置为 1。
#PF(错误码) 如果发生缺页异常。
