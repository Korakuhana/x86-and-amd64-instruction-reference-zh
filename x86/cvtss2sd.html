CVTSS2SD — 将标量单精度浮点值转换为标量双精度浮点值

操作码/指令 Op / En 64/32位模式支持 CPUID 特征标志 描述
F3 0F 5A /r CVTSS2SD xmm1, xmm2/m32 A V/V SSE2 将 xmm2/m32 中的一个单精度浮点值转换为 xmm1 中的一个双精度浮点值。
VEX.LIG.F3.0F.WIG 5A /r VCVTSS2SD xmm1, xmm2, xmm3/m32 B V/V AVX 将 xmm3/m32 中的一个单精度浮点值转换为一个双精度浮点值，并与 xmm2 的高位合并。
EVEX.LLIG.F3.0F.W0 5A /r VCVTSS2SD xmm1 {k1}{z}, xmm2, xmm3/m32{sae} C V/V AVX512F 在写掩码 k1 下，将 xmm3/m32 中的一个单精度浮点值转换为一个双精度浮点值，并与 xmm2 的高位合并。

描述¶

将“源转换”操作数中的单精度浮点值转换为目标操作数中的双精度浮点值。当“源转换”源操作数是 XMM 寄存器时，该单精度浮点值包含在该寄存器的低双字中。结果存储在目标操作数的低四字中。

128 位传统 SSE 版本：“源转换”源操作数（第二操作数）是 XMM 寄存器或内存位置。对应目标寄存器的位 (MAXVL-1:64) 保持不变。目标操作数是 XMM 寄存器。

VEX.128 和 EVEX 编码版本：“源转换”源操作数（第三操作数）可以是 XMM 寄存器或 32 位内存位置。第一源操作数和目标操作数是 XMM 寄存器。目标 XMM 寄存器的位 (127:64) 从第一源操作数中的对应位复制。目标寄存器的位 (MAXVL-1:128) 将被清零。

软件应确保将 VCVTSS2SD 编码为 VEX.L=0。将 VCVTSS2SD 编码为 VEX.L=1 可能会在不同的处理器世代中遇到不可预测的行为。

操作¶

VCVTSS2SD (EVEX Encoded Version)¶

IF k1[0] or *no writemask*
	THEN DEST[63:0] := Convert_Single_Precision_To_Double_Precision_Floating_Point(SRC2[31:0]);
	ELSE
		IF *merging-masking* ; merging-masking
			THEN *DEST[63:0] remains unchanged*
			ELSE ; zeroing-masking
				THEN DEST[63:0] = 0
		FI;
FI;
DEST[127:64] := SRC1[127:64]
DEST[MAXVL-1:128] := 0

VCVTSS2SD (VEX.128 Encoded Version)¶

DEST[63:0] := Convert_Single_Precision_To_Double_Precision_Floating_Point(SRC2[31:0])
DEST[127:64] := SRC1[127:64]
DEST[MAXVL-1:128] := 0

CVTSS2SD (128-bit Legacy SSE Version)¶

DEST[63:0] := Convert_Single_Precision_To_Double_Precision_Floating_Point(SRC[31:0]);
DEST[MAXVL-1:64] (Unmodified)
