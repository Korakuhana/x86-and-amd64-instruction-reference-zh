PSLLDQ — 双四字逻辑左移

操作码/指令 Op/En 64/32位模式支持 CPUID 特征标志 描述
66 0F 73 /7 ib PSLLDQ xmm1, imm8 A V/V SSE2 将 xmm1 左移 imm8 字节，同时移入 0。
VEX.128.66.0F.WIG 73 /7 ib VPSLLDQ xmm1, xmm2, imm8 B V/V AVX 将 xmm2 左移 imm8 字节，同时移入 0，并将结果存储在 xmm1 中。
VEX.256.66.0F.WIG 73 /7 ib VPSLLDQ ymm1, ymm2, imm8 B V/V AVX2 将 ymm2 左移 imm8 字节，同时移入 0，并将结果存储在 ymm1 中。
EVEX.128.66.0F.WIG 73 /7 ib VPSLLDQ xmm1,xmm2/ m128, imm8 C V/V AVX512VL AVX512BW 将 xmm2/m128 左移 imm8 字节，同时移入 0，并将结果存储在 xmm1 中。
EVEX.256.66.0F.WIG 73 /7 ib VPSLLDQ ymm1, ymm2/m256, imm8 C V/V AVX512VL AVX512BW 将 ymm2/m256 左移 imm8 字节，同时移入 0，并将结果存储在 ymm1 中。
EVEX.512.66.0F.WIG 73 /7 ib VPSLLDQ zmm1, zmm2/m512, imm8 C V/V AVX512BW 将 zmm2/m512 左移 imm8 字节，同时移入 0，并将结果存储在 zmm1 中。

描述¶

将目的操作数（第一个操作数）左移计数操作数（第二个操作数）指定的字节数。空出的低位字节被清除（设置为全 0）。如果计数操作数指定的值大于 15，则目的操作数将被设置为全 0。计数操作数是一个 8 位立即数。

128位传统 SSE 版本：源操作数和目的操作数相同。对应 YMM 目的寄存器的位（MAXVL-1:128）保持不变。

VEX.128 编码版本：源操作数和目的操作数是 XMM 寄存器。目的 YMM 寄存器的位（MAXVL-1:128）清零。

VEX.256 编码版本：源操作数是 YMM 寄存器。目的操作数是 YMM 寄存器。对应 ZMM 寄存器的位（MAXVL-1:256）清零。计数操作数同时应用于低 128 位和高 128 位通道。

EVEX 编码版本：源操作数是 ZMM/YMM/XMM 寄存器或 512/256/128 位内存位置。目的操作数是 ZMM/YMM/XMM 寄存器。计数操作数应用于每个 128 位通道。

操作¶

VPSLLDQ（EVEX.U1.512 编码版本）¶

TEMP := COUNT
IF (TEMP > 15) THEN TEMP := 16; FI
DEST[127:0] := SRC[127:0] << (TEMP * 8)
DEST[255:128] := SRC[255:128] << (TEMP * 8)
DEST[383:256] := SRC[383:256] << (TEMP * 8)
DEST[511:384] := SRC[511:384] << (TEMP * 8)
DEST[MAXVL-1:512] := 0

VPSLLDQ（VEX.256 和 EVEX.256 编码版本）¶

TEMP := COUNT
IF (TEMP > 15) THEN TEMP := 16; FI
DEST[127:0] := SRC[127:0] << (TEMP * 8)
DEST[255:128] := SRC[255:128] << (TEMP * 8)
DEST[MAXVL-1:256] := 0

VPSLLDQ（VEX.128 和 EVEX.128 编码版本）¶

TEMP := COUNT
IF (TEMP > 15) THEN TEMP := 16; FI
DEST := SRC << (TEMP * 8)
DEST[MAXVL-1:128] := 0

PSLLDQ（128位传统 SSE 版本）¶

TEMP := COUNT
IF (TEMP > 15) THEN TEMP := 16; FI
DEST := DEST << (TEMP * 8)
DEST[MAXVL-1:128] (Unmodified)
