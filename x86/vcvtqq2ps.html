VCVTQQ2PS — 将打包的四字整数转换为打包的单精度浮点值

操作码/指令 Op/En 64/32位模式支持 CPUID 特征标志 描述
EVEX.128.0F.W1 5B /r VCVTQQ2PS xmm1 {k1}{z}, xmm2/m128/m64bcst A V/V AVX512VL AVX512DQ 在写掩码 k1 的控制下，将 xmm2/mem 中的两个打包四字整数转换为 xmm1 中的打包单精度浮点值。
EVEX.256.0F.W1 5B /r VCVTQQ2PS xmm1 {k1}{z}, ymm2/m256/m64bcst A V/V AVX512VL AVX512DQ 在写掩码 k1 的控制下，将 ymm2/mem 中的四个打包四字整数转换为 xmm1 中的打包单精度浮点值。
EVEX.512.0F.W1 5B /r VCVTQQ2PS ymm1 {k1}{z}, zmm2/m512/m64bcst{er} A V/V AVX512DQ 在写掩码 k1 的控制下，将 zmm2/mem 中的八个打包四字整数转换为 ymm1 中的八个打包单精度浮点值。

指令操作数编码¶

Op/En 元组类型 操作数 1 操作数 2 操作数 3 操作数 4
A Full ModRM:reg (w) ModRM:r/m (r) N/A N/A

描述¶

将源操作数（第二个操作数）中的打包四字整数转换为目的操作数（第一个操作数）中的打包单精度浮点值。

源操作数是 ZMM/YMM/XMM 寄存器或 512/256/128 位内存位置。目的操作数是根据写掩码 k1 有条件更新的 YMM/XMM/XMM（低 64 位）寄存器。

EVEX.vvvv 保留且必须为 1111b，否则指令会导致 #UD。

操作¶

VCVTQQ2PS（EVEX 编码版本）当源操作数为寄存器时¶

(KL, VL) = (2, 128), (4, 256), (8, 512)
FOR j := 0 TO KL-1
    i := j * 64
    k := j * 32
    IF k1[j] OR *no writemask*
        THEN DEST[k+31:k] :=
            Convert_QuadInteger_To_Single_Precision_Floating_Point(SRC[i+63:i])
        ELSE
            IF *merging-masking* ; merging-masking
                THEN *DEST[k+31:k] remains unchanged*
                ELSE
                        ; zeroing-masking
                    DEST[k+31:k] := 0
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL/2] := 0

VCVTQQ2PS（EVEX 编码版本）当源操作数为内存源时¶

(KL, VL) = (2, 128), (4, 256), (8, 512)
FOR j := 0 TO KL-1
    i := j * 64
    k := j * 32
    IF k1[j] OR *no writemask*
        THEN
            IF (EVEX.b == 1)
                THEN
                    DEST[k+31:k] :=
            Convert_QuadInteger_To_Single_Precision_Floating_Point(SRC[63:0])
                ELSE
                    DEST[k+31:k] :=
            Convert_QuadInteger_To_Single_Precision_Floating_Point(SRC[i+63:i])
            FI;
        ELSE
            IF *merging-masking* ; merging-masking
                THEN *DEST[k+31:k] remains unchanged*
                ELSE ; zeroing-masking
                    DEST[k+31:k] := 0
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL/2] := 0
