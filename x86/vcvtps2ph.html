VCVTPS2PH — 将单精度浮点值转换为 16 位浮点值

描述¶

将源操作数中的打包单精度浮点值转换为半精度（16位）浮点值，并存储到目的操作数中。使用立即数字段 (imm8) 指定舍入模式。

欠溢结果（即极小值结果）被转换为反常数（渐近下溢数）。MXCSR.FTZ 被忽略。如果源元素相对于输入格式是反常数，且 DM 被屏蔽且 PM 或 UM 中至少有一个未被屏蔽，则会引发 SIMD 异常，并将 DE、UE 和 PE 置位。

立即数字节定义了几个控制舍入操作的位字段。RC 字段的影响和编码在表 5-13中列出。

VEX.128 版本：源操作数是 XMM 寄存器。目的操作数是 XMM 寄存器或 64 位内存位置。如果目的操作数是寄存器，则对应寄存器的高位 (MAXVL-1:64) 清零。

VEX.256 版本：源操作数是 YMM 寄存器。目的操作数是 XMM 寄存器或 128 位内存位置。如果目的操作数是寄存器，则对应寄存器的高位 (MAXVL-1:128) 清零。

注意：VEX.vvvv 和 EVEX.vvvv 保留（必须为 1111b）。

EVEX 编码版本：源操作数是 ZMM/YMM/XMM 寄存器。目的操作数是 YMM/XMM/XMM（低 64 位）寄存器或 256/128/64 位内存位置，根据写掩码 k1 有条件地更新。对应目的寄存器的高位 (MAXVL-1:256/128/64) 清零。

操作¶

vCvt_s2h(SRC1[31:0])
{
IF Imm[2] = 0
THEN ; 使用 Imm[1:0] 进行舍入控制，请参阅表 5-13
    RETURN Cvt_Single_Precision_To_Half_Precision_FP_Imm(SRC1[31:0]);
ELSE ; 使用 MXCSR.RC 进行舍入控制
    RETURN Cvt_Single_Precision_To_Half_Precision_FP_Mxcsr(SRC1[31:0]);
FI;
}

VCVTPS2PH（EVEX 编码版本）当目的操作数为寄存器时¶

(KL, VL) = (4, 128), (8, 256), (16, 512)
FOR j := 0 TO KL-1
    i := j * 16
    k := j * 32
    IF k1[j] OR *no writemask*
        THEN DEST[i+15:i] :=
            vCvt_s2h(SRC[k+31:k])
        ELSE
            IF *merging-masking*
                        ; merging-masking
                THEN *DEST[i+15:i] remains unchanged*
                ELSE
                        ; zeroing-masking
                    DEST[i+15:i] := 0
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL/2] := 0

VCVTPS2PH（EVEX 编码版本）当目的操作数为内存时¶

(KL, VL) = (4, 128), (8, 256), (16, 512)
FOR j := 0 TO KL-1
    i := j * 16
    k := j * 32
    IF k1[j] OR *no writemask*
        THEN DEST[i+15:i] :=
            vCvt_s2h(SRC[k+31:k])
        ELSE
            *DEST[i+15:i] remains unchanged*
                ; merging-masking
    FI;
ENDFOR

SIMD 浮点异常¶

无效、欠溢、溢出、精度、反常数（如果 MXCSR.DAZ=0）。
