描述¶

将第一源操作数（第二操作数）中各个数据元素（字/双字/四字）中的位，向右移位第二源操作数（第三操作数）中各个相应数据元素指定的计数移位数。随着数据元素中的位向右移位，空出的高位被设置为最高有效位（符号扩展）。

计数移位数分别在第二源操作数的每个数据元素中指定。如果第二源操作数对应数据元素中指定的无符号整数值大于 15（对于字）、31（对于双字）或 63（对于四字），则目的数据元素将被填充为源操作数对应元素的符号位。

VEX.128 编码版本：目的和第一源操作数是 XMM 寄存器。计数操作数可以是 XMM 寄存器或 128 位内存位置。对应目的寄存器的 bits (MAXVL-1:128) 清零。

VEX.256 编码版本：目的和第一源操作数是 YMM 寄存器。计数操作数可以是 YMM 寄存器或 256 位内存。对应目的寄存器的 bits (MAXVL-1:256) 清零。

EVEX.512/256/128 编码的 VPSRAVD/W：目的和第一源操作数是 ZMM/YMM/XMM 寄存器。计数操作数可以是 ZMM/YMM/XMM 寄存器、512/256/128 位内存位置，或从 32/64 位内存位置广播的 512/256/128 位向量。目的操作数在写掩码 k1 的控制下有条件地更新。

EVEX.512/256/128 编码的 VPSRAVQ：目的和第一源操作数是 ZMM/YMM/XMM 寄存器。计数操作数可以是 ZMM/YMM/XMM 寄存器或 512/256/128 位内存位置。目的操作数在写掩码 k1 的控制下有条件地更新。

操作¶

VPSRAVW (EVEX 编码版本)¶

(KL, VL) = (8, 128), (16, 256), (32, 512)
FOR j := 0 TO KL-1
    i := j * 16
    IF k1[j] OR *no writemask*
        THEN
            COUNT := SRC2[i+3:i]
            IF COUNT < 16
                THEN DEST[i+15:i] := SignExtend(SRC1[i+15:i] >> COUNT)
                ELSE
                    FOR k := 0 TO 15
                        DEST[i+k] := SRC1[i+15]
                    ENDFOR;
            FI
        ELSE
            IF *merging-masking* ; merging-masking
                THEN *DEST[i+15:i] remains unchanged*
                ELSE ; zeroing-masking
                    DEST[i+15:i] := 0
            FI
    FI;
ENDFOR;
DEST[MAXVL-1:VL] := 0;

VPSRAVD (VEX.128 版本)¶

COUNT_0 := SRC2[31 : 0]
    (* Repeat Each COUNT_i for the 2nd through 4th dwords of SRC2*)
COUNT_3 := SRC2[127 : 96];
DEST[31:0] := SignExtend(SRC1[31:0] >> COUNT_0);
    (* Repeat shift operation for 2nd through 4th dwords *)
DEST[127:96] := SignExtend(SRC1[127:96] >> COUNT_3);
DEST[MAXVL-1:128] := 0;

VPSRAVD (EVEX 编码版本)¶

(KL, VL) = (4, 128), (8, 256), (16, 512)
FOR j := 0 TO KL-1
    i := j * 32
    IF k1[j] OR *no writemask* THEN
            IF (EVEX.b = 1) AND (SRC2 *is memory*)
                THEN
                    COUNT := SRC2[4:0]
                    IF COUNT < 32
                        THEN DEST[i+31:i] := SignExtend(SRC1[i+31:i] >> COUNT)
                        ELSE
                            FOR k := 0 TO 31
                                DEST[i+k] := SRC1[i+31]
                            ENDFOR;
                    FI
                ELSE
                    COUNT := SRC2[i+4:i]
                    IF COUNT < 32
                        THEN DEST[i+31:i] := SignExtend(SRC1[i+31:i] >> COUNT)
                        ELSE
                            FOR k := 0 TO 31
                                DEST[i+k] := SRC1[i+31]
                            ENDFOR;
                    FI
            FI;
    ELSE
        IF *merging-masking*
                                    ; merging-masking
            THEN *DEST[31:0] remains unchanged*
            ELSE
                                    ; zeroing-masking
                DEST[31:0] := 0
            FI
    FI;
ENDFOR;
DEST[MAXVL-1:VL] := 0;

VPSRAVQ (EVEX 编码版本)¶

(KL, VL) = (2, 128), (4, 256), (8, 512)
FOR j := 0 TO KL-1
    i := j * 64
    IF k1[j] OR *no writemask* THEN
            IF (EVEX.b = 1) AND (SRC2 *is memory*)
                THEN
                    COUNT := SRC2[5:0]
                    IF COUNT < 64
                        THEN DEST[i+63:i] := SignExtend(SRC1[i+63:i] >> COUNT)
                        ELSE
                            FOR k := 0 TO 63
                                DEST[i+k] := SRC1[i+63]
                            ENDFOR;
                    FI
                ELSE
                    COUNT := SRC2[i+5:i]
                    IF COUNT < 64
                        THEN DEST[i+63:i] := SignExtend(SRC1[i+63:i] >> COUNT)
                        ELSE
                            FOR k := 0 TO 63
                                DEST[i+k] := SRC1[i+63]
                            ENDFOR;
                    FI
            FI;
    ELSE
        IF *merging-masking*
            THEN *DEST[63:0] remains unchanged*
            ELSE ; zeroing-masking
                DEST[63:0] := 0
            FI
    FI;
ENDFOR;
DEST[MAXVL-1:VL] := 0;
