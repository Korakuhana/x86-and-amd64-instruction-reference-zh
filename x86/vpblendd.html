VPBLENDD — 混合打包双字

操作码/指令 Op/En 64/32位模式支持 CPUID 功能标志 描述
VEX.128.66.0F3A.W0 02 /r ib VPBLENDD xmm1, xmm2, xmm3/m128, imm8 RVMI V/V AVX2 根据 imm8 中指定的掩码，从 xmm2 和 xmm3/m128 中选择双字（dword），并将值存储到 xmm1 中。
VEX.256.66.0F3A.W0 02 /r ib VPBLENDD ymm1, ymm2, ymm3/m256, imm8 RVMI V/V AVX2 根据 imm8 中指定的掩码，从 ymm2 和 ymm3/m256 中选择双字（dword），并将值存储到 ymm1 中。

指令操作数编码¶

Op/En 操作数 1 操作数 2 操作数 3 操作数 4
RVMI ModRM:reg (w) VEX.vvvv (r) ModRM:r/m (r) imm8

描述¶

根据立即数操作数（第三操作数）中的位，有条件地将来自源操作数（第二操作数）的双字元素写入目的操作数（第一操作数）。立即数位（位 7:0）形成一个掩码，该掩码决定目的操作数中的对应双字是否从源操作数复制。如果掩码中对应于双字的位为“1”，则复制该双字，否则目的操作数中的双字保持不变。

VEX.128 编码版本：第二源操作数可以是 XMM 寄存器或 128 位内存位置。第一源操作数和目的操作数是 XMM 寄存器。对应 YMM 寄存器的高位（MAXVL-1:128）清零。

VEX.256 编码版本：第一源操作数是 YMM 寄存器。第二源操作数是 YMM 寄存器或 256 位内存位置。目的操作数是 YMM 寄存器。

VPBLENDD (VEX.128 编码版本)¶

IF (imm8[0] == 1) THEN DEST[31:0] := SRC2[31:0]
ELSE DEST[31:0] := SRC1[31:0]
IF (imm8[1] == 1) THEN DEST[63:32] := SRC2[63:32]
ELSE DEST[63:32] := SRC1[63:32]
IF (imm8[2] == 1) THEN DEST[95:64] := SRC2[95:64]
ELSE DEST[95:64] := SRC1[95:64]

ELSE DEST[127:96] := SRC1[127:96]
DEST[MAXVL-1:128] := 0
