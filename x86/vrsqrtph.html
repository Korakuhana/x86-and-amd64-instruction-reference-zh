VRSQRTPH — 计算打包 FP16 值平方根的倒数

操作码/指令 Op/En 64/32位模式支持 CPUID 功能标志 描述
EVEX.128.66.MAP6.W0 4E /r VRSQRTPH xmm1{k1}{z}, xmm2/m128/m16bcst A V/V AVX512-FP16 AVX512VL 在写掩码 k1 控制下，计算 xmm2/m128/m16bcst 中打包 FP16 值平方根的近似倒数，并将结果存储在 xmm1 中。

指令操作数编码¶

描述¶

此指令对源操作数（第二操作数）中的 8/16/32 个打包 FP16 浮点值进行平方根近似倒数的 SIMD 计算，并将打包 FP16 浮点结果存储在目的操作数中。

该近似的最大相对误差小于 2−11 + 2−14。特殊情况请参阅表 5-38。

目的操作数元素在写掩码控制下有条件地更新。

操作¶

VRSQRTPH dest{k1}, src¶

VL = 128, 256 or 512
KL := VL/16
FOR i := 0 to KL-1:
    IF k1[i] or *no writemask*:
        IF SRC is memory and (EVEX.b = 1):
            tsrc := src.fp16[0]
        ELSE:
            tsrc := src.fp16[i]
        DEST.fp16[i] := APPROXIMATE(1.0 / SQRT(tsrc) )
    ELSE IF *zeroing*:
        DEST.fp16[i] := 0
    //else DEST.fp16[i] remains unchanged
DEST[MAXVL-1:VL] := 0
