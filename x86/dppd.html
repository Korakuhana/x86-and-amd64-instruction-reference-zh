DPPD — 打包双精度浮点值的点积

操作码/指令 操作/编码 64/32位模式支持 CPUID功能标志 描述
66 0F 3A 41 /r ib DPPD xmm1, xmm2/m128, imm8 RMI V/V SSE4_1 有选择地将 xmm1 中的打包双精度浮点值与 xmm2/m128 中的打包双精度浮点值相乘，相加并有选择地将结果打包双精度浮点值存储到 xmm1。
VEX.128.66.0F3A.WIG 41 /r ib VDPPD xmm1, xmm2, xmm3/m128, imm8 RVMI V/V AVX 有选择地将 xmm2 中的打包双精度浮点值与 xmm3/m128 中的打包双精度浮点值相乘，相加并有选择地将结果打包双精度浮点值存储到 xmm1。

指令操作数编码¶

操作/编码 操作数 1 操作数 2 操作数 3 操作数 4
RMI ModRM:reg (r, w) ModRM:r/m (r) imm8 N/A
RVMI ModRM:reg (w) VEX.vvvv (r) ModRM:r/m (r) imm8

描述¶

根据从立即数操作数（第三操作数）的位 [5:4] 中提取的掩码，有条件地将目标操作数（第一操作数）中的打包双精度浮点值与源操作数（第二操作数）中的打包双精度浮点值相乘。如果条件掩码位为零，则相应的乘法由值 0.0 代替，方式如《Intel® 64 and IA-32 Architectures Software Developer’s Manual》卷 1 第 12.8.4 节所述。

将两个生成的双精度值相加得到一个中间结果。根据立即数（第三操作数）的位 [1:0] 指定的广播掩码，将中间结果有条件地广播到目标操作数中。

如果广播掩码位为 “1”，则中间结果将复制到目标操作数中相应的四字元素中。如果广播掩码位为零，则目标操作数中对应的元素设置为零。

DPPD 遵循《Software Developer’s Manual》卷 1 表 4-7 中规定的 NaN 传递规则。这些规则不包括 NaN 的水平优先级划分。NaN 向目标操作数的水平传播以及这些 NaN 在目标操作数中的定位与实现相关。输入源上的 NaN 或计算生成的 NaN 将使至少一个 NaN 传播到目标操作数。

128 位传统 SSE 版本：第二个源可以是 XMM 寄存器或 128 位内存位置。目标操作数与第一个源操作数 XMM 寄存器相同，对应 YMM 寄存器目标操作数的高位（MAXVL-1:128）保持不变。

VEX.128 编码版本：第一个源操作数是 XMM 寄存器或 128 位内存位置。目标操作数是 XMM 寄存器。对应 YMM 寄存器目标操作数的高位（MAXVL-1:128）清零。

如果 VDPPD 编码为 VEX.L= 1，则尝试执行编码为 VEX.L= 1 的指令将导致 #UD 异常。

操作¶

DP_primitive (SRC1, SRC2)¶

IF (imm8[4] = 1)
    THEN Temp1[63:0] := DEST[63:0] * SRC[63:0]; // update SIMD exception flags
    ELSE Temp1[63:0] := +0.0; FI;
IF (imm8[5] = 1)
    THEN Temp1[127:64] := DEST[127:64] * SRC[127:64]; // update SIMD exception flags
    ELSE Temp1[127:64] := +0.0; FI;
/* if unmasked exception reported, execute exception handler*/
Temp2[63:0] := Temp1[63:0] + Temp1[127:64]; // update SIMD exception flags
/* if unmasked exception reported, execute exception handler*/
IF (imm8[0] = 1)
    THEN DEST[63:0] := Temp2[63:0];
    ELSE DEST[63:0] := +0.0; FI;
IF (imm8[1] = 1)
    THEN DEST[127:64] := Temp2[63:0];
    ELSE DEST[127:64] := +0.0; FI;

DPPD (128位传统 SSE 版本)¶

DEST[127:0] := DP_Primitive(SRC1[127:0], SRC2[127:0]);
DEST[MAXVL-1:128] (Unmodified)

VDPPD (VEX.128 编码版本)¶

DEST[127:0] := DP_Primitive(SRC1[127:0], SRC2[127:0]);
DEST[MAXVL-1:128] := 0

影响的标志寄存器¶

无。

Intel C/C++ 编译器等价内部函数¶

DPPD __m128d _mm_dp_pd ( __m128d a, __m128d b, const int mask);

SIMD 浮点异常¶

溢出、欠幅、无效、精度、非正规化。

每次加法和乘法操作分别确定异常。未屏蔽的异常将保持目标操作数不变。
