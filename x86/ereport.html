EREPORT — 创建 Enclave 的密码学报告

操作码/指令 操作/编码 64/32位模式支持 CPUID功能标志 描述
EAX = 00H ENCLU[EREPORT] IR V/V SGX1 此叶函数创建 Enclave 的密码学报告。

指令操作数编码¶

操作/编码 EAX RBX RCX RDX
IR EREPORT (输入) TARGETINFO 的地址 (输入) REPORTDATA 的地址 (输入) REPORT 写入的 OUTPUTDATA 的地址 (输入)

描述¶

此叶函数创建一个描述 Enclave 内容的密码学报告（REPORT）。该指令叶只能在 Enclave 内部执行。密码学报告可被其他 Enclave 用于确定该 Enclave 运行在同一个平台上。

RBX 包含将要认证 REPORT 输出的 Enclave 的 MRENCLAVE 值的有效地址，使用由该 Enclave 的 EGETKEY 命令交付的 REPORT 密钥。RCX 包含 64 字节 REPORTDATA 结构的有效地址，允许指令调用者将数据与调用该指令的 Enclave 相关联。RDX 包含指令输出 REPORT 的地址。

EREPORT 内存参数语义¶

TARGETINFO REPORTDATA OUTPUTDATA
Enclave 读取访问 Enclave 读取访问 Enclave 读/写访问

此指令叶执行以下操作：

1. 验证 3 个操作数（RBX、RCX、RDX）是否在 Enclave 内部。

2. 为目标 Enclave 计算报告密钥，如 RBX(TARGETINFO) 中的值所示。

3. 组装 Enclave SECS 数据以完成 REPORT 结构（包括使用 RCX (REPORTDATA) 操作数提供的数据）。

4. 计算 REPORT 结构的密码学哈希。

5. 将计算出的哈希添加到 REPORT 结构中。

6. 将完成의 REPORT 结构输出到 RDX (OUTPUTDATA) 中的地址。

如果操作数未正确对齐，指令将失败。

CR_REPORT_KEYID 用于提供密钥磨损保护，在平台启动时由 SGX TCB 内的受信任实体填充一个统计上唯一的值。

如果发生以下任何情况，指令将产生异常：

EREPORT 异常条件¶

有效地址未正确对齐。 内存地址未解析为 EPC 页面。
如果访问无效的 EPC 页面。 如果 EPC 页面被阻塞。
可能会发生页面错误。

(* Address verification for TARGETINFO (RBX) *)

THEN #PF(DS:RBX); FI;

IF (EPCM(DS:RBX).VALID = 0)

THEN #PF(DS:RBX); FI;

IF (EPCM(DS:RBX).BLOCKED = 1)

THEN #PF(DS:RBX); FI;

(* Check page parameters for correctness *)

IF ( (EPCM(DS:RBX).PT ≠ PT_REG) or (EPCM(DS:RBX).ENCLAVESECS ≠ CR_ACTIVE_SECS) or (EPCM(DS:RBX).PENDING = 1) or

(EPCM(DS:RBX).MODIFIED = 1) or (EPCM(DS:RBX).ENCLAVEADDRESS ≠ (DS:RBX & ~0FFFH) ) or (EPCM(DS:RBX).R = 0) )

THEN #PF(DS:RBX);

FI;

(* Verify RESERVED spaces in TARGETINFO are valid *)

IF (DS:RBX.RESERVED != 0)

THEN #GP(0); FI;

THEN #GP(0); FI;

THEN #PF(DS:RCX); FI;

IF (EPCM(DS:RCX).VALID = 0)

THEN #PF(DS:RCX); FI;

IF (EPCM(DS:RCX).BLOCKED = 1)

THEN #PF(DS:RCX); FI;

(* Check page parameters for correctness *)

IF ( (EPCM(DS:RCX).PT ≠ PT_REG) or (EPCM(DS:RCX).ENCLAVESECS ≠ CR_ACTIVE_SECS) or (EPCM(DS:RCX).PENDING = 1) or

(EPCM(DS:RCX).MODIFIED = 1) or (EPCM(DS:RCX).ENCLAVEADDRESS ≠ (DS:RCX & ~0FFFH) ) or (EPCM(DS:RCX).R = 0) )
