BLENDPD — 混合打包双精度浮点值

操作码/指令 Op/En 64/32位模式 CPUID 特征标志 描述
66 0F 3A 0D /r ib BLENDPD xmm1, xmm2/m128, imm8 RMI V/V SSE4_1 根据 imm8 中指定的掩码，从 xmm1 和 xmm2/m128 中选择打包双精度浮点值，并将结果存入 xmm1。
VEX.128.66.0F3A.WIG 0D /r ib VBLENDPD xmm1, xmm2, xmm3/m128, imm8 RVMI V/V AVX 根据 imm8 中的掩码，从 xmm2 和 xmm3/m128 中选择打包双精度浮点值，并将结果存入 xmm1。
VEX.256.66.0F3A.WIG 0D /r ib VBLENDPD ymm1, ymm2, ymm3/m256, imm8 RVMI V/V AVX 根据 imm8 中的掩码，从 ymm2 和 ymm3/m256 中选择打包双精度浮点值，并将结果存入 ymm1。

指令操作数编码¶

Op/En 操作数 1 操作数 2 操作数 3 操作数 4
RMI ModRM:reg (r, w) ModRM:r/m (r) imm8 N/A
RVMI ModRM:reg (w) VEX.vvvv (r) ModRM:r/m (r) imm8[3:0]

描述¶

根据来自第二源操作数（第三操作数）的双精度浮点值，条件性地与来自第一源操作数（第二操作数）的值进行合并，并将结果写入目的操作数（第一操作数）。立即数 [3:0] 位决定目的地中对应的双精度浮点值是从第二源还是第一源复制。如果掩码中对应于某个字（qword）的位为 “1”，则复制第二源操作数中的双精度浮点值，否则复制第一源操作数中的值。

128 位传统 SSE 版本：第二源操作数可以是 XMM 寄存器或 128 位内存位置。目标操作数与第一源操作数 XMM 寄存器相同，并且对应 YMM 寄存器目标的高位（MAXVL-1:128）保持不变。

VEX.128 编码版本：第一源操作数是 XMM 寄存器。第二源操作数是 XMM 寄存器或 128 位内存位置。目的操作数是 XMM 寄存器。对应 YMM 寄存器目标的高位（MAXVL-1:128）清零。

VEX.256 编码版本：第一源操作数是 YMM 寄存器。第二源操作数可以是 YMM 寄存器或 256 位内存位置。目的操作数是 YMM 寄存器。

BLENDPD (128-bit Legacy SSE Version)¶

IF (IMM8[0] = 0)THEN DEST[63:0] := DEST[63:0]
    ELSE DEST [63:0] := SRC[63:0] FI
IF (IMM8[1] = 0) THEN DEST[127:64] := DEST[127:64]
    ELSE DEST [127:64] := SRC[127:64] FI
DEST[MAXVL-1:128] (Unmodified)

VBLENDPD (VEX.128 Encoded Version)¶

IF (IMM8[0] = 0)THEN DEST[63:0] := SRC1[63:0]
    ELSE DEST [63:0] := SRC2[63:0] FI
IF (IMM8[1] = 0) THEN DEST[127:64] := SRC1[127:64]
    ELSE DEST [127:64] := SRC2[127:64] FI
DEST[MAXVL-1:128] := 0

VBLENDPD (VEX.256 Encoded Version)¶

IF (IMM8[0] = 0)THEN DEST[63:0] := SRC1[63:0]
    ELSE DEST [63:0] := SRC2[63:0] FI
IF (IMM8[1] = 0) THEN DEST[127:64] := SRC1[127:64]
    ELSE DEST [127:64] := SRC2[127:64] FI
IF (IMM8[2] = 0) THEN DEST[191:128] := SRC1[191:128]
    ELSE DEST [191:128] := SRC2[191:128] FI
IF (IMM8[3] = 0) THEN DEST[255:192] := SRC1[255:192]
    ELSE DEST [255:192] := SRC2[255:192] FI
