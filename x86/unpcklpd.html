描述¶

对第一源操作数和第二源操作数的低位双精度浮点值进行交错解包。

128 位传统 SSE 版本：第二源操作数可以是 XMM 寄存器或 128 位内存位置。目的操作数与第一源操作数 XMM 寄存器相同，对应 ZMM 寄存器目的的高位 (MAXVL-1:128) 保持不变。当从内存操作数解包时，实现可能仅提取适当的 64 位；但仍将强制执行 16 字节边界对齐和常规段检查。

VEX.128 编码版本：第一源操作数是 XMM 寄存器。第二源操作数可以是 XMM 寄存器或 128 位内存位置。目的操作数是 XMM 寄存器。对应 ZMM 寄存器目的的高位 (MAXVL-1:128) 清零。

VEX.256 编码版本：第一源操作数是 YMM 寄存器。第二源操作数可以是 YMM 寄存器或 256 位内存位置。目的操作数是 YMM 寄存器。

EVEX.512 编码版本：第一源操作数是 ZMM 寄存器。第二源操作数是 ZMM 寄存器、512 位内存位置，或从 64 位内存位置广播的 512 位向量。目的操作数是 ZMM 寄存器，在写掩码 k1 的控制下有条件地更新。

EVEX.256 编码版本：第一源操作数是 YMM 寄存器。第二源操作数是 YMM 寄存器、256 位内存位置，或从 64 位内存位置广播的 256 位向量。目的操作数是 YMM 寄存器，在写掩码 k1 的控制下有条件地更新。

EVEX.128 编码版本：第一源操作数是 XMM 寄存器。第二源操作数是 XMM 寄存器、128 位内存位置，或从 64 位内存位置广播的 128 位向量。目的操作数是 XMM 寄存器，在写掩码 k1 的控制下有条件地更新。

操作¶

VUNPCKLPD（EVEX 编码版本，当 SRC2 为寄存器时）¶

(KL, VL) = (2, 128), (4, 256), (8, 512)
IF VL >= 128
    TMP_DEST[63:0] := SRC1[63:0]
    TMP_DEST[127:64] := SRC2[63:0]
FI;
IF VL >= 256
    TMP_DEST[191:128] := SRC1[191:128]
    TMP_DEST[255:192] := SRC2[191:128]
FI;
IF VL >= 512
    TMP_DEST[319:256] := SRC1[319:256]
    TMP_DEST[383:320] := SRC2[319:256]
    TMP_DEST[447:384] := SRC1[447:384]
    TMP_DEST[511:448] := SRC2[447:384]
FI;
FOR j := 0 TO KL-1
    i := j * 64
    IF k1[j] OR *no writemask*
        THEN DEST[i+63:i] := TMP_DEST[i+63:i]
        ELSE
            IF *merging-masking*
                        ; merging-masking
                THEN *DEST[i+63:i] remains unchanged*
                ELSE *zeroing-masking*
                            ; zeroing-masking
                    DEST[i+63:i] := 0
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0

VUNPCKLPD（EVEX 编码版本，当 SRC2 为内存时）¶

(KL, VL) = (2, 128), (4, 256), (8, 512)
FOR j := 0 TO KL-1
    i := j * 64
    IF (EVEX.b = 1)
        THEN TMP_SRC2[i+63:i] := SRC2[63:0]
        ELSE TMP_SRC2[i+63:i] := SRC2[i+63:i]
    FI;
ENDFOR;
IF VL >= 128
    TMP_DEST[63:0] := SRC1[63:0]
    TMP_DEST[127:64] := TMP_SRC2[63:0]
FI;
IF VL >= 256
    TMP_DEST[191:128] := SRC1[191:128]
    TMP_DEST[255:192] := TMP_SRC2[191:128]
FI;
IF VL >= 512
    TMP_DEST[319:256] := SRC1[319:256]
    TMP_DEST[383:320] := TMP_SRC2[319:256]
    TMP_DEST[447:384] := SRC1[447:384]
    TMP_DEST[511:448] := TMP_SRC2[447:384]
FI;
FOR j := 0 TO KL-1
    i := j * 64
    IF k1[j] OR *no writemask*
        THEN DEST[i+63:i] := TMP_DEST[i+63:i]
        ELSE
            IF *merging-masking*
                        ; merging-masking
                THEN *DEST[i+63:i] remains unchanged*
                ELSE *zeroing-masking*
                            ; zeroing-masking
                    DEST[i+63:i] := 0
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0
