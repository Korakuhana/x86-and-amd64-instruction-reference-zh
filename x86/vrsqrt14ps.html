VRSQRT14PS — 计算打包单精度浮点值平方根的近似倒数

描述¶

此指令对源操作数（第二操作数）中的打包单精度浮点值执行平方根近似倒数的 SIMD 计算，并在写掩码控制下将打包单精度浮点结果存储在目的操作数（第一操作数）中。该近似的最大相对误差小于 2-14。

EVEX.512 编码版本：源操作数可以是 ZMM 寄存器、512 位内存位置，或从 32 位内存位置广播的 512 位向量。目的操作数是 ZMM 寄存器，在写掩码 k1 控制下有条件地更新。

EVEX.256 编码版本：源操作数是 YMM 寄存器、256 位内存位置，或从 32 位内存位置广播的 256 位向量。目的操作数是 YMM 寄存器，在写掩码 k1 控制下有条件地更新。

EVEX.128 编码版本：源操作数是 XMM 寄存器、128 位内存位置，或从 32 位内存位置广播的 128 位向量。目的操作数是 XMM 寄存器，在写掩码 k1 控制下有条件地更新。

VRSQRT14PS 指令不受 MXCSR 寄存器中舍入控制位的影响。当源值为 0.0 时，返回带有源值符号的 ∞。当源操作数是 +∞ 时，返回 +ZERO。仅当 MXCSR 中设置了 DAZ 位时，渐近下溢 (denormal) 源值才会被视为零。否则它将被正确处理，并使用指定的掩码响应执行近似计算。当源值是负值（非 0.0）时，返回浮点 QNaN_indefinite。当源值是 SNaN 或 QNaN 时，SNaN 被转换为 QNaN，或者返回源 QNaN。

MXCSR 异常标志不受此指令影响，且不报告浮点异常。

注意：EVEX.vvvv 是保留的，必须为 1111b，否则指令将引发 #UD。

操作¶

VRSQRT14PS (EVEX 编码版本)¶

(KL, VL) = (4, 128), (8, 256), (16, 512)
FOR j := 0 TO KL-1
    i := j * 32
    IF k1[j] OR *no writemask* THEN
            IF (EVEX.b = 1) AND (SRC *is memory*)
                THEN DEST[i+31:i] := APPROXIMATE(1.0/ SQRT(SRC[31:0]));
                ELSE DEST[i+31:i] := APPROXIMATE(1.0/ SQRT(SRC[i+31:i]));
            FI;
    ELSE
        IF *merging-masking* ; merging-masking
            THEN *DEST[i+31:i] remains unchanged*
            ELSE
                    ; zeroing-masking
                DEST[i+31:i] := 0
        FI;
    FI;
ENDFOR;
DEST[MAXVL-1:VL] := 0
