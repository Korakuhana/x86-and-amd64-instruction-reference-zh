AESENC — 执行一轮 AES 加密流程

描述¶

该指令使用来自第一源操作数的一个/两个/四个（取决于向量长度）128 位数据（状态）和来自第二源操作数的一个/两个/四个（取决于向量长度）轮密钥，执行单轮 AES 加密流程，并将结果存入目标操作数。

除了最后一轮加密，所有加密轮次都使用 AESENC 指令。对于最后一轮加密，使用 AESENCLAST 指令。

VEX 和 EVEX 编码版本允许三操作数（非破坏性）操作。指令的传统编码版本要求第一源操作数和目标操作数必须相同，且必须为 XMM 寄存器。

此指令的 EVEX 编码形式不支持内存故障抑制。

操作¶

AESENC

STATE := SRC1;
RoundKey := SRC2;
STATE := ShiftRows( STATE );
STATE := SubBytes( STATE );
STATE := MixColumns( STATE );
DEST[127:0] := STATE XOR RoundKey;
DEST[MAXVL-1:128] (Unmodified)

VAESENC (128b and 256b VEX Encoded Versions)

(KL,VL) = (1,128), (2,256)
FOR I := 0 to KL-1:
    STATE := SRC1.xmm[i]
    RoundKey := SRC2.xmm[i]
    STATE := ShiftRows( STATE )
    STATE := SubBytes( STATE )
    STATE := MixColumns( STATE )
    DEST.xmm[i] := STATE XOR RoundKey
DEST[MAXVL-1:VL] := 0

VAESENC (EVEX Encoded Version)

(KL,VL) = (1,128), (2,256), (4,512)
FOR i := 0 to KL-1:
    STATE := SRC1.xmm[i] // xmm[i] is the i’th xmm word in the SIMD register
    RoundKey := SRC2.xmm[i]
    STATE := ShiftRows( STATE )
    STATE := SubBytes( STATE )
    STATE := MixColumns( STATE )
    DEST.xmm[i] := STATE XOR RoundKey
DEST[MAXVL-1:VL] := 0
