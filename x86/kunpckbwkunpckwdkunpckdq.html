KUNPCKBW/KUNPCKWD/KUNPCKDQ — 掩码寄存器交错

操作码/指令 Op/En 64/32位模式支持 CPUID 特征标志 描述
VEX.L1.66.0F.W0 4B /r KUNPCKBW k1, k2, k3 RVR V/V AVX512F 交错合并 k2 和 k3 中的 8 位掩码，并将字结果写入 k1。
VEX.L1.0F.W0 4B /r KUNPCKWD k1, k2, k3 RVR V/V AVX512BW 交错合并 k2 和 k3 中的 16 位掩码，并将双字结果写入 k1。
VEX.L1.0F.W1 4B /r KUNPCKDQ k1, k2, k3 RVR V/V AVX512BW 交错合并 k2 和 k3 中的 32 位掩码，并将四字结果写入 k1。

指令操作数编码¶

Op/En 操作数 1 操作数 2 操作数 3
RVR ModRM:reg (w) VEX.1vvv (r) ModRM:r/m (r, ModRM:[7:6] 必须是 11b)

描述¶

将第二个和第三个操作数（源操作数）的低 8/16/32 位交错合并到第一个操作数（目的操作数）的低位部分，从低位字节开始。结果在目的操作数中零扩展。

操作¶

KUNPCKBW¶

DEST[7:0] := SRC2[7:0]
DEST[15:8] := SRC1[7:0]
DEST[MAX_KL-1:16] := 0

KUNPCKWD¶

DEST[15:0] := SRC2[15:0]
DEST[31:16] := SRC1[15:0]
DEST[MAX_KL-1:32] := 0

KUNPCKDQ¶

DEST[31:0] := SRC2[31:0]
DEST[63:32] := SRC1[31:0]
DEST[MAX_KL-1:64] := 0

Intel C/C++ 编译器等价内部函数¶

KUNPCKBW __mmask16 _mm512_kunpackb(__mmask16 a, __mmask16 b);

KUNPCKDQ __mmask64 _mm512_kunpackd(__mmask64 a, __mmask64 b);

KUNPCKWD __mmask32 _mm512_kunpackw(__mmask32 a, __mmask32 b);
