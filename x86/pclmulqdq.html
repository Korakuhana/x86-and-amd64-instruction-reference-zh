描述¶

根据立即数字节的值，对从第一个源操作数和第二个源操作数中选出的两个四字执行无进位乘法。根据表 4-13，使用位 4 和位 0 来选择要使用每个操作数的哪个 64 位半部分，立即数字节的其他位将被忽略。

此指令的 EVEX 编码形式不支持内存故障抑制。

1. SRC2 表示第二个源操作数，可以是寄存器或内存；SRC1 表示第一个源和目的操作数。

第一个源操作数和目的操作数相同，且必须是 ZMM/YMM/XMM 寄存器。第二个源操作数可以是 ZMM/YMM/XMM 寄存器或 512/256/128 位内存位置。对应 YMM 目的寄存器的位（VL_MAX-1:128）保持不变。

编译器和汇编器可以实现以下伪操作语法，以简化编程并生成 imm8 所需的编码。

操作¶

define PCLMUL128(X,Y): // helper function
    FOR i := 0 to 63:
        TMP [ i ] := X[ 0 ] and Y[ i ]
        FOR j := 1 to i:
            TMP [ i ] := TMP [ i ] xor (X[ j ] and Y[ i - j ])
        DEST[ i ] := TMP[ i ]
    FOR i := 64 to 126:
        TMP [ i ] := 0
        FOR j := i - 63 to 63:
            TMP [ i ] := TMP [ i ] xor (X[ j ] and Y[ i - j ])
        DEST[ i ] := TMP[ i ]
    DEST[127] := 0;
    RETURN DEST // 128b vector

PCLMULQDQ（SSE 版本）¶

IF imm8[0] = 0:
    TEMP1 := SRC1.qword[0]
ELSE:
    TEMP1 := SRC1.qword[1]
IF imm8[4] = 0:
    TEMP2 := SRC2.qword[0]
ELSE:
    TEMP2 := SRC2.qword[1]
DEST[127:0] := PCLMUL128(TEMP1, TEMP2)
DEST[MAXVL-1:128] (Unmodified)

(KL,VL) = (1,128), (2,256)
FOR i= 0 to KL-1:
    IF imm8[0] = 0:
        TEMP1 := SRC1.xmm[i].qword[0]
    ELSE:
        TEMP1 := SRC1.xmm[i].qword[1]
    IF imm8[4] = 0:
        TEMP2 := SRC2.xmm[i].qword[0]
    ELSE:
        TEMP2 := SRC2.xmm[i].qword[1]
    DEST.xmm[i] := PCLMUL128(TEMP1, TEMP2)
DEST[MAXVL-1:VL] := 0
