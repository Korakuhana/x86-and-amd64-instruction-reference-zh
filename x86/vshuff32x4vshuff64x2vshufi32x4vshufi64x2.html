描述¶

256 位版本：将第一源操作数（第二操作数）中的两个 128 位打包单精度浮点值之一移入目的操作数（第一操作数）的低 128 位；将第二源操作数（第三操作数）中的两个 128 位打包浮点值之一移入目的操作数的高 128 位。选择器操作数（第四操作数，即 imm8）决定哪些值移入目的操作数。

512 位版本：将第一源操作数（第二操作数）中的四个 128 位打包单精度浮点值中的两个移入目的操作数（第一操作数）的低 256 位；将第二源操作数（第三操作数）中的四个 128 位打包浮点值中的两个移入目的操作数的高 256 位。选择器操作数（第四操作数，即 imm8）决定哪些值移入目的操作数。

第一源操作数是向量寄存器。第二源操作数可以是 ZMM 寄存器、512 位内存位置，或从 32/64 位内存位置广播的 512 位向量。目的操作数是向量寄存器。

写掩码以 32/64 位数据元素的粒度更新目的操作数。

操作¶

Select2(SRC, control) {
CASE (control[0]) OF
    0: TMP := SRC[127:0];
    1: TMP := SRC[255:128];
ESAC;
RETURN TMP
}
Select4(SRC, control) {
CASE (control[1:0]) OF
    0: TMP := SRC[127:0];
    1: TMP := SRC[255:128];
    2: TMP := SRC[383:256];
    3: TMP := SRC[511:384];
ESAC;
RETURN TMP
}

VSHUFF32x4 (EVEX 版本)¶

(KL, VL) = (8, 256), (16, 512)
FOR j := 0 TO KL-1
    i := j * 32
    IF (EVEX.b = 1) AND (SRC2 *is memory*)
        THEN TMP_SRC2[i+31:i] := SRC2[31:0]
        ELSE TMP_SRC2[i+31:i] := SRC2[i+31:i]
    FI;
ENDFOR;
IF VL = 256
    TMP_DEST[127:0] := Select2(SRC1[255:0], imm8[0]);
    TMP_DEST[255:128] := Select2(SRC2[255:0], imm8[1]);
FI;
IF VL = 512
    TMP_DEST[127:0] := Select4(SRC1[511:0], imm8[1:0]);
    TMP_DEST[255:128] := Select4(SRC1[511:0], imm8[3:2]);
    TMP_DEST[383:256] := Select4(TMP_SRC2[511:0], imm8[5:4]);
    TMP_DEST[511:384] := Select4(TMP_SRC2[511:0], imm8[7:6]);
FI;
FOR j := 0 TO KL-1
    i := j * 32
    IF k1[j] OR *no writemask*
        THEN DEST[i+31:i] := TMP_DEST[i+31:i]
        ELSE
            IF *merging-masking*
                        ; merging-masking
                THEN *DEST[i+31:i] remains unchanged*
                ELSE *zeroing-masking*
                            ; zeroing-masking
                    THEN DEST[i+31:i] := 0
            FI;
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0
