描述¶

将 4、8 或 16 个单精度浮点值从源操作数（第二个操作数）移动到目的操作数（第一个操作数）。此指令可用于从 128 位、256 位或 512 位内存位置加载 XMM、YMM 或 ZMM 寄存器，将 XMM、YMM 或 ZMM 寄存器的内容存储到 128 位、256 位或 512 位内存位置中，或者在两个 XMM、两个 YMM 或两个 ZMM 寄存器之间移动数据。

当源或目的操作数是内存操作数时，操作数必须对齐在 16 字节（128 位版本）、32 字节（VEX.256 编码版本）或 64 字节（EVEX.512 编码版本）边界上，否则将产生一般保护性异常 (#GP)。对于 EVEX.512 编码版本，操作数必须对齐到内存操作数的大小。要在未对齐的内存位置之间移动单精度浮点值，请使用 VMOVUPS 指令。

注意：VEX.vvvv 和 EVEX.vvvv 是保留的，必须为 1111b，否则指令将产生 #UD 异常。

EVEX.512 编码版本：

将 512 位打包单精度浮点值从源操作数（第二个操作数）移动到目的操作数（第一个操作数）。此指令可用于从 512 位 float32 内存位置加载 ZMM 寄存器，将 ZMM 寄存器的内容存储到 float32 内存位置中，或者在两个 ZMM 寄存器之间移动数据。当源或目的操作数是内存操作数时，操作数必须对齐在 64 字节边界上，否则将产生一般保护性异常 (#GP)。要在未对齐的内存位置之间移动单精度浮点值，请使用 VMOVUPS 指令。

VEX.256 和 EVEX.256 编码版本：

将 256 位打包单精度浮点值从源操作数（第二个操作数）移动到目的操作数（第一个操作数）。此指令可用于从 256 位内存位置加载 YMM 寄存器，将 YMM 寄存器的内容存储到 256 位内存位置中，或者在两个 YMM 寄存器之间移动数据。当源或目的操作数是内存操作数时，操作数必须对齐在 32 字节边界上，否则将产生一般保护性异常 (#GP)。

128 位版本：

将 128 位打包单精度浮点值从源操作数（第二个操作数）移动到目的操作数（第一个操作数）。此指令可用于从 128 位内存位置加载 XMM 寄存器，将 XMM 寄存器的内容存储到 128 位内存位置中，或者在两个 XMM 寄存器之间移动数据。当源或目的操作数是内存操作数时，操作数必须对齐在 16 字节边界上，否则将产生一般保护性异常 (#GP)。要在未对齐的内存位置之间移动单精度浮点值，请使用 VMOVUPS 指令。

128 位传统 SSE 版本：对应 ZMM 目的寄存器的位 (MAXVL-1:128) 保持不变。

(E)VEX.128 编码版本：目的 ZMM 寄存器的位 (MAXVL-1:128) 被清零。

操作¶

VMOVAPS (EVEX Encoded Versions, Register-Copy Form)¶

(KL, VL) = (4, 128), (8, 256), (16, 512)
FOR j := 0 TO KL-1
    i := j * 32
    IF k1[j] OR *no writemask*
        THEN DEST[i+31:i] := SRC[i+31:i]
        ELSE
            IF *merging-masking*
                THEN *DEST[i+31:i] remains unchanged*
                ELSE DEST[i+31:i] := 0 ; zeroing-masking
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0

VMOVAPS (EVEX Encoded Versions, Store Form)¶

(KL, VL) = (4, 128), (8, 256), (16, 512)
FOR j := 0 TO KL-1
    i := j * 32
    IF k1[j] OR *no writemask*
        THEN DEST[i+31:i] :=
            SRC[i+31:i]
        ELSE *DEST[i+31:i] remains unchanged*
                ; merging-masking
ENDFOR;

VMOVAPS (EVEX Encoded Versions, Load Form)¶

(KL, VL) = (4, 128), (8, 256), (16, 512)
FOR j := 0 TO KL-1
    i := j * 32
    IF k1[j] OR *no writemask*
        THEN DEST[i+31:i] := SRC[i+31:i]
        ELSE
            IF *merging-masking*
                THEN *DEST[i+31:i] remains unchanged*
                ELSE DEST[i+31:i] := 0 ; zeroing-masking
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0

DEST[255:0] := SRC[255:0]
DEST[MAXVL-1:256] := 0

VMOVAPS (VEX.256 Encoded Version, Store-Form)¶

DEST[255:0] := SRC[255:0]
