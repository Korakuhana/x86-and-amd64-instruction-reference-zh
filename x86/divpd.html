DIVPD — 除以打包双精度浮点值

指令/指令格式 操作/编码 64/32位模式支持 CPUID功能标志 描述
66 0F 5E /r DIVPD xmm1, xmm2/m128 A V/V SSE2 将 xmm1 中的打包双精度浮点值除以 xmm2/mem 中的打包双精度浮点值。
VEX.128.66.0F.WIG 5E /r VDIVPD xmm1, xmm2, xmm3/m128 B V/V AVX 将 xmm2 中的打包双精度浮点值除以 xmm3/mem 中的打包双精度浮点值。
VEX.256.66.0F.WIG 5E /r VDIVPD ymm1, ymm2, ymm3/m256 B V/V AVX 将 ymm2 中的打包双精度浮点值除以 ymm3/mem 中的打包双精度浮点值。
EVEX.128.66.0F.W1 5E /r VDIVPD xmm1 {k1}{z}, xmm2, xmm3/m128/m64bcst C V/V AVX512VL AVX512F 根据写掩码 k1，将 xmm2 中的打包双精度浮点值除以 xmm3/m128/m64bcst 中的打包双精度浮点值，并将结果写入 xmm1。
EVEX.256.66.0F.W1 5E /r VDIVPD ymm1 {k1}{z}, ymm2, ymm3/m256/m64bcst C V/V AVX512VL AVX512F 根据写掩码 k1，将 ymm2 中的打包双精度浮点值除以 ymm3/m256/m64bcst 中的打包双精度浮点值，并将结果写入 ymm1。
EVEX.512.66.0F.W1 5E /r VDIVPD zmm1 {k1}{z}, zmm2, zmm3/m512/m64bcst{er} C V/V AVX512F 根据写掩码 k1，将 zmm2 中的打包双精度浮点值除以 zmm3/m512/m64bcst 中的打包双精度浮点值，并将结果写入 zmm1。

描述¶

对第一个源操作数中的打包双精度浮点值除以第二个源操作数（第三个操作数）中的双精度浮点值进行 SIMD 除法。结果写入目标操作数（第一个操作数）。

EVEX 编码版本：第一个源操作数（第二个操作数）是 ZMM/YMM/XMM 寄存器。第二个源操作数可以是 ZMM/YMM/XMM 寄存器、512/256/128 位内存位置或从 64 位内存位置广播的 512/256/128 位向量。目标操作数是 ZMM/YMM/XMM 寄存器，并根据写掩码 k1 条件式更新。

VEX.256 编码版本：第一个源操作数（第二个操作数）是 YMM 寄存器。第二个源操作数可以是 YMM 寄存器或 256 位内存位置。目标操作数是 YMM 寄存器。对应目标操作数的高位（MAXVL-1:256）清零。

VEX.128 编码版本：第一个源操作数（第二个操作数）是 XMM 寄存器。第二个源操作数可以是 XMM 寄存器或 128 位内存位置。目标操作数是 XMM 寄存器。对应目标操作数的高位（MAXVL-1:128）清零。

128 位传统 SSE 版本：第二个源操作数（第二个操作数）可以是 XMM 寄存器或 128 位内存位置。目标操作数与第一个源操作数相同。对应目标操作数的高位（MAXVL-1:128）保持不变。

操作¶

VDIVPD (EVEX 编码版本)¶

(KL, VL) = (2, 128), (4, 256), (8, 512)
IF (VL = 512) AND (EVEX.b = 1) AND SRC2 *is a register*
    THEN
        SET_ROUNDING_MODE_FOR_THIS_INSTRUCTION(EVEX.RC); ; refer to Table 15-4 in the Intel® 64 and IA-32 Architectures Software Developer’s Manual, Volume 1
    ELSE
        SET_ROUNDING_MODE_FOR_THIS_INSTRUCTION(MXCSR.RC);
FI;
FOR j := 0 TO KL-1
    i := j * 64
    IF k1[j] OR *no writemask*
        THEN
            IF (EVEX.b = 1) AND (SRC2 *is memory*)
                THEN
                    DEST[i+63:i] := SRC1[i+63:i] / SRC2[63:0]
                ELSE
                    DEST[i+63:i] := SRC1[i+63:i] / SRC2[i+63:i]
            FI;
        ELSE
            IF *merging-masking* ; merging-masking
                THEN *DEST[i+63:i] remains unchanged*
                ELSE ; zeroing-masking
                    DEST[i+63:i] := 0
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0

VDIVPD (VEX.256 编码版本)¶

DEST[63:0] := SRC1[63:0] / SRC2[63:0]
DEST[127:64] := SRC1[127:64] / SRC2[127:64]
DEST[191:128] := SRC1[191:128] / SRC2[191:128]
DEST[255:192] := SRC1[255:192] / SRC2[255:192]
DEST[MAXVL-1:256] := 0;

VDIVPD (VEX.128 编码版本)¶

DEST[63:0] := SRC1[63:0] / SRC2[63:0]
DEST[127:64] := SRC1[127:64] / SRC2[127:64]
DEST[MAXVL-1:128] := 0;

DIVPD (128位传统 SSE 版本)¶

DEST[63:0] := SRC1[63:0] / SRC2[63:0]
DEST[127:64] := SRC1[127:64] / SRC2[127:64]
DEST[MAXVL-1:128] (Unmodified)
