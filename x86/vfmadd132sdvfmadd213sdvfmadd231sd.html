描述¶

使用三个源操作数对低位双精度浮点值执行 SIMD 乘加计算，并将乘加结果写入目的操作数。目的操作数也是第一个源操作数。第一和第二个操作数是 XMM 寄存器。第三个源操作数可以是 XMM 寄存器或 64 位内存位置。

VFMADD132SD：将来自第一个源操作数的低位双精度浮点值与第三个源操作数中的低位双精度浮点值相乘，将无限精度中间结果与第二个源操作数中的低位双精度浮点值相加，执行舍入，并将结果的双精度浮点值存储到目的操作数（第一个源操作数）中。

VFMADD213SD：将来自第二个源操作数的低位双精度浮点值与第一个源操作数中的低位双精度浮点值相乘，将无限精度中间结果与第三个源操作数中的低位双精度浮点值相加，执行舍入，并将结果的双精度浮点值存储到目的操作数（第一个源操作数）中。

VFMADD231SD：将来自第二个源操作数的低位双精度浮点值与第三个源操作数中的低位双精度浮点值相乘，将无限精度中间结果与第一个源操作数中的低位双精度浮点值相加，执行舍入，并将结果的双精度浮点值存储到目的操作数（第一个源操作数）中。

VEX.128 和 EVEX 编码版本：目的操作数（也是第一个源操作数）在 reg_field 中编码。第二个源操作数在 VEX.vvvv/EVEX.vvvv 中编码。第三个源操作数在 rm_field 中编码。目的寄存器的 [127:64] 位保持不变。目的寄存器的 [MAXVL-1:128] 位被清零。

EVEX 编码版本：目的寄存器的低四字元素根据写掩码被更新。

操作¶

在以下操作中，“*”和“+”符号表示使用无限精度输入和输出进行乘法和加法（无舍入）。

VFMADD132SD DEST, SRC2, SRC3 (EVEX 编码版本)¶

IF (EVEX.b = 1) and SRC3 *is a register*
    THEN
        SET_ROUNDING_MODE_FOR_THIS_INSTRUCTION(EVEX.RC);
    ELSE
        SET_ROUNDING_MODE_FOR_THIS_INSTRUCTION(MXCSR.RC);
FI;
IF k1[0] or *no writemask*
    THEN DEST[63:0] := RoundFPControl(DEST[63:0]*SRC3[63:0] + SRC2[63:0])
    ELSE
        IF *merging-masking* ; merging-masking
            THEN *DEST[63:0] remains unchanged*
            ELSE ; zeroing-masking
                THEN DEST[63:0] := 0
        FI;
FI;
DEST[127:64] := DEST[127:64]
DEST[MAXVL-1:128] := 0
