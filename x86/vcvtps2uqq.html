VCVTPS2UQQ — 将打包的单精度浮点值转换为打包的无符号四字整数值

操作码/指令 Op/En 64/32位模式支持 CPUID 特征标志 描述
EVEX.128.66.0F.W0 79 /r VCVTPS2UQQ xmm1 {k1}{z}, xmm2/m64/m32bcst A V/V AVX512VL AVX512DQ 在写掩码 k1 的控制下，将 zmm2/m64/m32bcst 中的两个打包单精度浮点值转换为 zmm1 中的两个打包无符号四字整数值。
EVEX.256.66.0F.W0 79 /r VCVTPS2UQQ ymm1 {k1}{z}, xmm2/m128/m32bcst A V/V AVX512VL AVX512DQ 在写掩码 k1 的控制下，将 xmm2/m128/m32bcst 中的四个打包单精度浮点值转换为 ymm1 中的四个打包无符号四字整数值。
EVEX.512.66.0F.W0 79 /r VCVTPS2UQQ zmm1 {k1}{z}, ymm2/m256/m32bcst{er} A V/V AVX512DQ 在写掩码 k1 的控制下，将 ymm2/m256/m32bcst 中的八个打包单精度浮点值转换为 zmm1 中的八个打包无符号四字整数值。

指令操作数编码¶

Op/En 元组类型 操作数 1 操作数 2 操作数 3 操作数 4
A Half ModRM:reg (w) ModRM:r/m (r) N/A N/A

描述¶

将源操作数中的最多八个打包单精度浮点值转换为目的操作数中的无符号四字整数。

当转换不精确时，返回的值根据 MXCSR 寄存器中的舍入控制位或嵌入式舍入控制位进行舍入。如果转换后的结果无法在目的格式中表示，则会引发浮点无效异常；如果该异常被屏蔽，则返回整数值 2w – 1，其中 w 代表目的格式中的位数。

源操作数是 YMM/XMM/XMM（低 64 位）寄存器或 256/128/64 位内存位置。目的操作数是根据写掩码 k1 有条件更新的 ZMM/YMM/XMM 寄存器。

EVEX.vvvv 保留且必须为 1111b，否则指令会导致 #UD。

操作¶

VCVTPS2UQQ（EVEX 编码版本）当源操作数为寄存器时¶

(KL, VL) = (2, 128), (4, 256), (8, 512)
IF (VL == 512) AND (EVEX.b == 1)
    THEN
        SET_ROUNDING_MODE_FOR_THIS_INSTRUCTION(EVEX.RC);
    ELSE
        SET_ROUNDING_MODE_FOR_THIS_INSTRUCTION(MXCSR.RC);
FI;
FOR j := 0 TO KL-1
    i := j * 64
    k := j * 32
    IF k1[j] OR *no writemask*
        THEN DEST[i+63:i] :=
            Convert_Single_Precision_To_UQuadInteger(SRC[k+31:k])
        ELSE
            IF *merging-masking* ; merging-masking
                THEN *DEST[i+63:i] remains unchanged*
                ELSE ; zeroing-masking
                    DEST[i+63:i] := 0
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0

VCVTPS2UQQ（EVEX 编码版本）当源操作数为内存源时¶

(KL, VL) = (2, 128), (4, 256), (8, 512)
FOR j := 0 TO KL-1
    i := j * 64
    k := j * 32
    IF k1[j] OR *no writemask*
        THEN
            IF (EVEX.b == 1)
                THEN
                    DEST[i+63:i] :=
            Convert_Single_Precision_To_UQuadInteger(SRC[31:0])
                ELSE
                    DEST[i+63:i] :=
            Convert_Single_Precision_To_UQuadInteger(SRC[k+31:k])
            FI;
        ELSE
            IF *merging-masking* ; merging-masking
                THEN *DEST[i+63:i] remains unchanged*
                ELSE ; zeroing-masking
                    DEST[i+63:i] := 0
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0
