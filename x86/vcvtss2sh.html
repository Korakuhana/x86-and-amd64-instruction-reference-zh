VCVTSS2SH — 将低位 FP32 值转换为 FP16 值

操作码/指令 Op/En 64/32位模式支持 CPUID 特征标志 描述
EVEX.LLIG.NP.MAP5.W0 1D /r VCVTSS2SH xmm1{k1}{z}, xmm2, xmm3/m32 {er} A V/V AVX512-FP16 在写掩码 k1 的控制下，将 xmm3/m32 中的低位 FP32 值转换为 FP16 值，并将结果存储在 xmm1 的低位元素中。xmm2 的 127:16 位被复制到 xmm1[127:16]。

指令操作数编码¶

Op/En 元组 操作数 1 操作数 2 操作数 3 操作数 4
A Scalar ModRM:reg (w) VEX.vvvv (r) ModRM:r/m (r) N/A

描述¶

此指令将第二个源操作数中的低位 FP32 值转换为 FP16 值，并将结果存储在目的操作数的低位元素中。

当转换不精确时，返回的值根据 MXCSR 寄存器中的舍入控制位进行舍入。

目的操作数的 127:16 位从第一个源操作数的对应位复制。目的操作数的 MAXVL-1:128 位清零。目的寄存器的低位 FP16 元素根据写掩码进行更新。

操作¶

VCVTSS2SH dest, src1, src2¶

IF *SRC2 is a register* and (EVEX.b = 1):
    SET_RM(EVEX.RC)
ELSE:
    SET_RM(MXCSR.RC)
IF k1[0] OR *no writemask*:
    DEST.fp16[0] := Convert_fp32_to_fp16(SRC2.fp32[0])
ELSE IF *zeroing*:
    DEST.fp16[0] := 0
// else dest.fp16[0] remains unchanged
DEST[127:16] := SRC1[127:16]
DEST[MAXVL-1:128] := 0
