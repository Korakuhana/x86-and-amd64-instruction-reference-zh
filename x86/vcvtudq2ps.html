VCVTUDQ2PS — 将打包的无符号双字整数转换为打包的单精度浮点值

操作码/指令 Op/En 64/32位模式支持 CPUID 特征标志 描述
EVEX.128.F2.0F.W0 7A /r VCVTUDQ2PS xmm1 {k1}{z}, xmm2/m128/m32bcst A V/V AVX512VL AVX512F 在写掩码 k1 的控制下，将 xmm2/m128/m32bcst 中的四个打包无符号双字整数转换为 xmm1 中的打包单精度浮点值。
EVEX.256.F2.0F.W0 7A /r VCVTUDQ2PS ymm1 {k1}{z}, ymm2/m256/m32bcst A V/V AVX512VL AVX512F 在写掩码 k1 的控制下，将 ymm2/m256/m32bcst 中的八个打包无符号双字整数转换为 ymm1 中的打包单精度浮点值。
EVEX.512.F2.0F.W0 7A /r VCVTUDQ2PS zmm1 {k1}{z}, zmm2/m512/m32bcst{er} A V/V AVX512F 在写掩码 k1 的控制下，将 zmm2/m512/m32bcst 中的十六个打包无符号双字整数转换为 zmm1 中的十六个打包单精度浮点值。

指令操作数编码¶

Op/En 元组类型 操作数 1 操作数 2 操作数 3 操作数 4
A Full ModRM:reg (w) ModRM:r/m (r) N/A N/A

描述¶

将源操作数（第二个操作数）中的打包无符号双字整数转换为目的操作数（第一个操作数）中的单精度浮点值。

源操作数是 ZMM/YMM/XMM 寄存器、512/256/128 位内存位置，或从 32 位内存位置广播的 512/256/128 位向量。目的操作数是根据写掩码 k1 有条件更新的 ZMM/YMM/XMM 寄存器。

注意：EVEX.vvvv 保留且必须为 1111b，否则指令会导致 #UD。

操作¶

VCVTUDQ2PS（EVEX 编码版本）当源操作数为寄存器时¶

(KL, VL) = (4, 128), (8, 256), (16, 512)
IF (VL = 512) AND (EVEX.b = 1)
    THEN
        SET_ROUNDING_MODE_FOR_THIS_INSTRUCTION(EVEX.RC);
    ELSE
        SET_ROUNDING_MODE_FOR_THIS_INSTRUCTION(MXCSR.RC);
FI;
FOR j := 0 TO KL-1
    i := j * 32
    IF k1[j] OR *no writemask*
        THEN DEST[i+31:i] :=
            Convert_UInteger_To_Single_Precision_Floating_Point(SRC[i+31:i])
        ELSE
            IF *merging-masking* ; merging-masking
                THEN *DEST[i+31:i] remains unchanged*
                ELSE ; zeroing-masking
                    DEST[i+31:i] := 0
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0

VCVTUDQ2PS（EVEX 编码版本）当源操作数为内存源时¶

(KL, VL) = (4, 128), (8, 256), (16, 512)
FOR j := 0 TO KL-1
    i := j * 32
    IF k1[j] OR *no writemask*
        THEN
            IF (EVEX.b = 1)
                THEN
                    DEST[i+31:i] :=
            Convert_UInteger_To_Single_Precision_Floating_Point(SRC[31:0])
                ELSE
                    DEST[i+31:i] :=
            Convert_UInteger_To_Single_Precision_Floating_Point(SRC[i+31:i])
            FI;
        ELSE
            IF *merging-masking* ; merging-masking
                THEN *DEST[i+31:i] remains unchanged*
                ELSE ; zeroing-masking
                    DEST[i+31:i] := 0
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0
