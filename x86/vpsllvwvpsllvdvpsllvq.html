描述¶

将第一源操作数中各个数据元素（字、双字或四字）中的位，向左移位第二源操作数中各个相应元素指定的移位数。随着数据元素中的位向左移位，空出的低位被清零（设为 0）。

计数移位数分别在第二源操作数的每个数据元素中指定。如果第二源操作数对应数据元素中指定的无符号整数值大于 15（对于字）、31（对于双字）或 63（对于四字），则目的数据元素被写入 0。

VEX.128 编码版本：目的和第一源操作数是 XMM 寄存器。计数操作数可以是 XMM 寄存器或 128 位内存位置。对应目的寄存器的 bits (MAXVL-1:128) 清零。

VEX.256 编码版本：目的和第一源操作数是 YMM 寄存器。计数操作数可以是 YMM 寄存器或 256 位内存。对应 ZMM 寄存器的 bits (MAXVL-1:256) 清零。

EVEX 编码的 VPSLLVD/Q：目的和第一源操作数是 ZMM/YMM/XMM 寄存器。计数操作数可以是 ZMM/YMM/XMM 寄存器、512/256/128 位内存位置，或从 32/64 位内存位置广播的 512 位向量。目的操作数在写掩码 k1 的控制下有条件地更新。

EVEX 编码的 VPSLLVW：目的和第一源操作数是 ZMM/YMM/XMM 寄存器。计数操作数可以是 ZMM/YMM/XMM 寄存器或 512/256/128 位内存位置。目的操作数在写掩码 k1 的控制下有条件地更新。

操作¶

VPSLLVW (EVEX 编码版本)¶

(KL, VL) = (8, 128), (16, 256), (32, 512)
FOR j := 0 TO KL-1
    i := j * 16
    IF k1[j] OR *no writemask*
        THEN DEST[i+15:i] := ZeroExtend(SRC1[i+15:i] << SRC2[i+15:i])
        ELSE
            IF *merging-masking* ; merging-masking
                THEN *DEST[i+15:i] remains unchanged*
                ELSE
                        ; zeroing-masking
                    DEST[i+15:i] := 0
            FI
    FI;
ENDFOR;
DEST[MAXVL-1:VL] := 0;

VPSLLVD (VEX.128 版本)¶

COUNT_0 := SRC2[31 : 0]
    (* Repeat Each COUNT_i for the 2nd through 4th dwords of SRC2*)
COUNT_3 := SRC2[127 : 96];
IF COUNT_0 < 32 THEN
DEST[31:0] := ZeroExtend(SRC1[31:0] << COUNT_0);
ELSE
DEST[31:0] := 0;
    (* Repeat shift operation for 2nd through 4th dwords *)
IF COUNT_3 < 32 THEN
DEST[127:96] := ZeroExtend(SRC1[127:96] << COUNT_3);
ELSE
DEST[127:96] := 0;
DEST[MAXVL-1:128] := 0;

(KL, VL) = (4, 128), (8, 256), (16, 512)
FOR j := 0 TO KL-1
    i := j * 32
    IF k1[j] OR *no writemask* THEN
            IF (EVEX.b = 1) AND (SRC2 *is memory*)
                THEN DEST[i+31:i] := ZeroExtend(SRC1[i+31:i] << SRC2[31:0])
                ELSE DEST[i+31:i] := ZeroExtend(SRC1[i+31:i] << SRC2[i+31:i])

                ELSE ; zeroing-masking
                    DEST[i+31:i] := 0
            FI
    FI;
ENDFOR;
