AESKEYGENASSIST — AES 轮密钥生成辅助

操作码/指令 Op/En 64/32位模式 CPUID 特征标志 描述
66 0F 3A DF /r ib AESKEYGENASSIST xmm1, xmm2/m128, imm8 RMI V/V AES 利用立即数中指定的 8 位轮常数（RCON），对 xmm2/m128 中指定的 128 位数据进行操作，辅助生成 AES 轮密钥，并将结果存入 xmm1。
VEX.128.66.0F3A.WIG DF /r ib VAESKEYGENASSIST xmm1, xmm2/m128, imm8 RMI V/V AES 和 AVX 标志 利用立即数中指定的 8 位轮常数（RCON），对 xmm2/m128 中指定的 128 位数据进行操作，辅助生成 AES 轮密钥，并将结果存入 xmm1。

指令操作数编码¶

Op/En 操作数 1 操作数 2 操作数 3 操作数 4
RMI ModRM:reg (w) ModRM:r/m (r) imm8 N/A

描述¶

使用源操作数中指定的 128 位数据和作为立即数指定的 8 位轮常数，通过计算生成加密轮密钥的步骤，辅助扩展 AES 密码密钥，并将结果存入目标操作数。

目标操作数是一个 XMM 寄存器。源操作数可以是一个 XMM 寄存器或 128 位内存地址。

128 位传统 SSE 版本：相应 YMM 目标寄存器的高位（MAXVL-1:128）保持不变。

VEX.128 编码版本：目标 YMM 寄存器的高位（MAXVL-1:128）清零。

注意：在 VEX 编码版本中，VEX.vvvv 是保留的，必须为 1111b，否则指令将引发 #UD。

操作¶

AESKEYGENASSIST

X3[31:0] := SRC [127: 96];
X2[31:0] := SRC [95: 64];
X1[31:0] := SRC [63: 32];
X0[31:0] := SRC [31: 0];
RCON[31:0] := ZeroExtend(imm8[7:0]);
DEST[31:0] := SubWord(X1);
DEST[63:32 ] := RotWord( SubWord(X1) ) XOR RCON;
DEST[95:64] := SubWord(X3);
DEST[127:96] := RotWord( SubWord(X3) ) XOR RCON;
DEST[MAXVL-1:128] (Unmodified)

VAESKEYGENASSIST

X3[31:0] := SRC [127: 96];
X2[31:0] := SRC [95: 64];
X1[31:0] := SRC [63: 32];
X0[31:0] := SRC [31: 0];
RCON[31:0] := ZeroExtend(imm8[7:0]);
DEST[31:0] := SubWord(X1);
DEST[63:32 ] := RotWord( SubWord(X1) ) XOR RCON;
DEST[95:64] := SubWord(X3);
DEST[127:96] := RotWord( SubWord(X3) ) XOR RCON;
DEST[MAXVL-1:128] := 0;
