SUBPS — 减去打包单精度浮点值

操作码/指令 Op/En 64/32位模式支持 CPUID 特征标志 描述
NP 0F 5C /r SUBPS xmm1, xmm2/m128 A V/V SSE 从 xmm1 中减去 xmm2/mem 中的打包单精度浮点值，并将结果存储在 xmm1 中。
VEX.128.0F.WIG 5C /r VSUBPS xmm1,xmm2, xmm3/m128 B V/V AVX 从 xmm2 中减去 xmm3/mem 中的打包单精度浮点值，并将结果存储在 xmm1 中。

EVEX.128.0F.W0 5C /r VSUBPS xmm1 {k1}{z}, xmm2, xmm3/m128/m32bcst C V/V AVX512VL AVX512F 从 xmm2 中减去 xmm3/m128/m32bcst 中的打包单精度浮点值，并在写掩码 k1 的控制下将结果存储在 xmm1 中。
EVEX.256.0F.W0 5C /r VSUBPS ymm1 {k1}{z}, ymm2, ymm3/m256/m32bcst C V/V AVX512VL AVX512F 从 ymm2 中减去 ymm3/m256/m32bcst 中的打包单精度浮点值，并在写掩码 k1 的控制下将结果存储在 ymm1 中。
EVEX.512.0F.W0 5C /r VSUBPS zmm1 {k1}{z}, zmm2, zmm3/m512/m32bcst{er} C V/V AVX512F 从 zmm2 中减去 zmm3/m512/m32bcst 中的打包单精度浮点值，并在写掩码 k1 的控制下将结果存储在 zmm1 中。

描述¶

从第一源操作数中对第二源操作数中的打包单精度浮点值进行 SIMD 减法，并将打包单精度浮点结果存储在目的操作数中。

VEX.128 和 EVEX.128 编码版本：第二源操作数是 XMM 寄存器或 128 位内存位置。第一源操作数和目的操作数是 XMM 寄存器。对应目的寄存器的高位 (MAXVL-1:128) 被清零。

VEX.256 和 EVEX.256 编码版本：第二源操作数是 YMM 寄存器或 256 位内存位置。第一源操作数和目的操作数是 YMM 寄存器。对应目的寄存器的高位 (MAXVL-1:256) 被清零。

EVEX.512 编码版本：第二源操作数是 ZMM 寄存器、512 位内存位置，或从 32 位内存位置广播的 512 位向量。第一源操作数和目的操作数是 ZMM 寄存器。目的操作数根据写掩码进行条件更新。

128 位传统 SSE 版本：第二源操作数可以是 XMM 寄存器或 128 位内存位置。目的操作数与第一源 XMM 寄存器相同，且对应寄存器目的操作数的高位 (MAXVL-1:128) 保持不变。

操作¶

VSUBPS（当 SRC2 操作数为向量寄存器时的 EVEX 编码版本）¶

(KL, VL) = (4, 128), (8, 256), (16, 512)
IF (VL = 512) AND (EVEX.b = 1)
    THEN
        SET_ROUNDING_MODE_FOR_THIS_INSTRUCTION(EVEX.RC);
    ELSE
        SET_ROUNDING_MODE_FOR_THIS_INSTRUCTION(MXCSR.RC);
FI;
FOR j := 0 TO KL-1
    i := j * 32
    IF k1[j] OR *no writemask*
        THEN DEST[i+31:i] := SRC1[i+31:i] - SRC2[i+31:i]
    ELSE
        IF *merging-masking* ; merging-masking
            THEN *DEST[31:0] remains unchanged*
            ELSE ; zeroing-masking
                DEST[31:0] := 0
        FI;
    FI;
ENDFOR;
DEST[MAXVL-1:VL] := 0

VSUBPS（当 SRC2 操作数为内存源时的 EVEX 编码版本）¶

(KL, VL) = (4, 128), (8, 256),(16, 512)
FOR j := 0 TO KL-1
    i := j * 32
    IF k1[j] OR *no writemask* THEN
            IF (EVEX.b = 1)
                THEN DEST[i+31:i] := SRC1[i+31:i] - SRC2[31:0];
                ELSE DEST[i+31:i] := SRC1[i+31:i] - SRC2[i+31:i];
            FI;
    ELSE
        IF *merging-masking* ; merging-masking
            THEN *DEST[31:0] remains unchanged*
            ELSE ; zeroing-masking
                DEST[31:0] := 0
        FI;
    FI;
ENDFOR;
DEST[MAXVL-1:VL] := 0
