VSUBSH — 减法标量 FP16 值

操作码/指令 Op/En 64/32位模式支持 CPUID 功能标志 描述
EVEX.LLIG.F3.MAP5.W0 5C /r VSUBSH xmm1{k1}{z}, xmm2, xmm3/m16 {er} A V/V AVX512-FP16 在写掩码 k1 控制下，从 xmm2 中减去 xmm3/m16 中的低 FP16 值，并将结果存储在 xmm1 中。将 xmm2 中的 bits 127:16 复制到 xmm1[127:16]。

指令操作数编码¶

Op/En 元组 操作数 1 操作数 2 操作数 3 操作数 4
A Scalar ModRM:reg (w) VEX.vvvv (r) ModRM:r/m (r) N/A

描述¶

此指令从第一源操作数中的对应值中减去第二源操作数中的低 FP16 值，并将 FP16 结果存储在目的操作数中。目的操作数的 bits 127:16 从第一源操作数的对应位复制。目的操作数的 bits MAXVL-1:128 清零。目的操作数的低 FP16 元素在写掩码控制下更新。

操作¶

VSUBSH (EVEX 编码版本)¶

IF EVEX.b = 1 and SRC2 is a register:
    SET_ROUNDING_MODE_FOR_THIS_INSTRUCTION(EVEX.RC)
ELSE
    SET_ROUNDING_MODE_FOR_THIS_INSTRUCTION(MXCSR.RC)
IF k1[0] OR *no writemask*:
    DEST.fp16[0] := SRC1.fp16[0] - SRC2.fp16[0]
ELSE IF *zeroing*:
    DEST.fp16[0] := 0
// else dest.fp16[0] remains unchanged
DEST[127:16] := SRC1[127:16]
DEST[MAXVL-1:128] := 0
