SAHF — 将 AH 写入标志寄存器

操作码 指令 Op/En 64位模式 兼容/传统模式 描述
9E SAHF ZO Invalid1 Valid 将 AH 中的值加载到 EFLAGS 寄存器的 SF、ZF、AF、PF 和 CF 中。

1. 在特定步进中有效；参见描述部分。

指令操作数编码¶

Op/En 操作数 1 操作数 2 操作数 3 操作数 4
ZO N/A N/A N/A N/A

描述¶

将 EFLAGS 寄存器的 SF、ZF、AF、PF 和 CF 标志加载为 AH 寄存器中对应位的值（分别为第 7、6、4、2 和 0 位）。AH 寄存器的第 1、3 和 5 位被忽略；EFLAGS 寄存器中对应的保留位（1、3 和 5）保持不变，如下方的“操作”部分所示。

此指令在兼容模式和传统模式下如上所述执行。仅当 CPUID.80000001H:ECX.LAHF-SAHF[bit 0] = 1 时，它在 64 位模式下才有效。

操作¶

IF IA-64 Mode
    THEN
        IF CPUID.80000001H.ECX[0] = 1;
            THEN
                RFLAGS(SF:ZF:0:AF:0:PF:1:CF) := AH;
            ELSE
                #UD;
        FI
    ELSE
        EFLAGS(SF:ZF:0:AF:0:PF:1:CF) := AH;
FI;

影响的标志位¶

SF、ZF、AF、PF 和 CF 标志加载自 AH 寄存器的值。EFLAGS 寄存器的第 1、3 和 5 位不受影响，其值分别保持为 1、0 和 0。

保护模式异常¶

无。

实地址模式异常¶

无。

虚拟 8086 模式异常¶

无。

兼容模式异常¶

无。

64位模式异常¶

#UD 如果 CPUID.80000001H.ECX[0] = 0。
如果使用了 LOCK 前缀。
