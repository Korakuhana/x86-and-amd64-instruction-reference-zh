XABORT — 事务异常终止

操作码/指令 Op/En 64/32位模式支持 CPUID 功能标志 描述
C6 F8 ib XABORT imm8 A V/V RTM 如果处于 RTM 执行中，则引发 RTM 终止。

指令操作数编码¶

Op/En 操作数 1 操作数 2 操作数 3 操作数 4
A imm8 N/A N/A N/A

描述¶

XABORT 强制发生 RTM 终止。在 RTM 终止之后，逻辑处理器在通过最外层 XBEGIN 指令计算的后备地址处恢复执行。EAX 寄存器会被更新以反映是 XABORT 指令引发了该终止，并且 imm8 参数将提供在 EAX 的 bits 31:24 中。

操作¶

XABORT¶

IF RTM_ACTIVE = 0
    THEN
        Treat as NOP;
    ELSE
        GOTO RTM_ABORT_PROCESSING;
FI;
(* 对于在 RTM 执行期间遇到的任何 RTM 终止条件 *)
RTM_ABORT_PROCESSING:
    Restore architectural register state;
    Discard memory updates performed in transaction;
    Update EAX with status and XABORT argument;
    RTM_NEST_COUNT:= 0;
    RTM_ACTIVE:= 0;
    SUSLDTRK_ACTIVE := 0;
    IF 64-bit Mode
        THEN
            RIP:= fallbackRIP;
        ELSE
            EIP := fallbackEIP;
    FI;
END

无。
