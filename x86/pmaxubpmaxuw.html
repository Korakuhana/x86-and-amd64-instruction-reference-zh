描述¶

对第二个源操作数和第一个源操作数中的打包无符号字节、字整数执行 SIMD 比较，并向目的操作数返回每对整数的最大值。

传统 SSE 版本 PMAXUB：源操作数可以是 MMX 技术寄存器或 64 位内存位置。目的操作数可以是 MMX 技术寄存器。

128位传统 SSE 版本：第一个源操作数和目的操作数是 XMM 寄存器。第二个源操作数是 XMM 寄存器或 128 位内存位置。对应目的寄存器的位（MAXVL-1:128）保持不变。

VEX.128 编码版本：第一个源操作数和目的操作数是 XMM 寄存器。第二个源操作数是 XMM 寄存器或 128 位内存位置。对应目的寄存器的位（MAXVL-1:128）清零。

VEX.256 编码版本：第二个源操作数可以是 YMM 寄存器或 256 位内存位置。第一个源操作数和目的操作数是 YMM 寄存器。

EVEX 编码版本：第一个源操作数是 ZMM/YMM/XMM 寄存器；第二个源操作数是 ZMM/YMM/XMM 寄存器或 512/256/128 位内存位置。根据写掩码 k1 条件性地更新目的操作数。

VPMAXUB（EVEX 编码版本）¶

(KL, VL) = (16, 128), (32, 256), (64, 512)
FOR j := 0 TO KL-1
    i := j * 8
    IF k1[j] OR *no writemask* THEN
        IF SRC1[i+7:i] > SRC2[i+7:i]
            THEN DEST[i+7:i] := SRC1[i+7:i];
            ELSE DEST[i+7:i] := SRC2[i+7:i];
        FI;
        ELSE
            IF *merging-masking*
                THEN *DEST[i+7:i] remains unchanged*
                ELSE ; zeroing-masking
                    DEST[i+7:i] := 0
            FI
    FI;
ENDFOR;
DEST[MAXVL-1:VL] := 0
