描述¶

AFFINEB 指令在伽罗瓦域 28 中计算仿射变换。对于此指令，仿射变换由 A * x + b 定义，其中“A”是一个 8 x 8 位矩阵，“x”和“b”是 8 位向量。一个 SIMD 寄存器（操作数 1）保存表示为 16、32 或 64 个 8 位向量的“x”。第二个 SIMD 寄存器（操作数 2）或内存操作数包含 2、4 或 8 个“A”值，由第一个寄存器中相应对齐的 8 个“x”值对其进行运算。“b”向量对于所有计算都是常数，并且包含在立即数字节中。

此指令的 EVEX 编码形式不支持内存故障抑制。该指令的 SSE 编码形式要求其内存操作进行 16 字节对齐。

操作¶

define parity(x):
    t := 0 // single bit
    FOR i := 0 to 7:
        t = t xor x.bit[i]
    return t
define affine_byte(tsrc2qw, src1byte, imm):
    FOR i := 0 to 7:
        * parity(x) = 1 if x has an odd number of 1s in it, and 0 otherwise.*
        retbyte.bit[i] := parity(tsrc2qw.byte[7-i] AND src1byte) XOR imm8.bit[i]
    return retbyte

VGF2P8AFFINEQB dest, src1, src2, imm8 (EVEX Encoded Version)¶

(KL, VL) = (2, 128), (4, 256), (8, 512)
FOR j := 0 TO KL-1:
    IF SRC2 is memory and EVEX.b==1:
        tsrc2 := SRC2.qword[0]
    ELSE:
        tsrc2 := SRC2.qword[j]
    FOR b := 0 to 7:
        IF k1[j*8+b] OR *no writemask*:
            DEST.qword[j].byte[b] := affine_byte(tsrc2, SRC1.qword[j].byte[b], imm8)
        ELSE IF *zeroing*:
            DEST.qword[j].byte[b] := 0
        *ELSE DEST.qword[j].byte[b] remains unchanged*
DEST[MAX_VL-1:VL] := 0
