VCMPSH — 比较标量 FP16 值

操作码/指令 Op/En 64/32位模式支持 CPUID 特征标志 描述
EVEX.LLIG.F3.0F3A.W0 C2 /r /ib VCMPSH k1{k2}, xmm2, xmm3/m16 {sae}, imm8 A V/V AVX512-FP16 在写掩码 k2 的控制下，使用 imm8 的第 4:0 位作为比较条件，比较 xmm3/m16 和 xmm2 中的低位 FP16 值，并将结果存储在掩码寄存器 k1 中。

指令操作数编码¶

Op/En 元组 操作数 1 操作数 2 操作数 3 操作数 4
A Scalar ModRM:reg (w) VEX.vvvv (r) ModRM:r/m (r) imm8 (r)

描述¶

此指令比较来自源操作数最低元素的 FP16 值，并将结果存储在目的掩码操作数中。比较条件操作数（立即数字节的第 4:0 位）指定对该对打包 FP16 值执行的比较类型。低位目的位根据写掩码进行更新。目的操作数的 MAXKL-1:1 位清零。

VCMPSH（EVEX 编码版本）¶

IF k2[0] OR *no writemask*:
    DEST.bit[0] := SRC1.fp16[0] CMP_OPERATOR SRC2.fp16[0]
ELSE
    DEST.bit[0] := 0
DEST[MAXKL-1:1] := 0
