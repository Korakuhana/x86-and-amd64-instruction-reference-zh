MOVQ — 移动四字

操作码/指令 Op/ En 64/32位模式 CPUID 特征标志 描述
NP 0F 6F /r MOVQ mm, mm/m64 A 有效/有效 MMX 将四字从 mm/m64 移动到 mm。
NP 0F 7F /r MOVQ mm/m64, mm B 有效/有效 MMX 将四字从 mm 移动到 mm/m64。
F3 0F 7E /r MOVQ xmm1, xmm2/m64 A 有效/有效 SSE2 将四字从 xmm2/mem64 移动到 xmm1。
VEX.128.F3.0F.WIG 7E /r VMOVQ xmm1, xmm2/m64 A 有效/有效 AVX 将四字从 xmm2 移动到 xmm1。
EVEX.128.F3.0F.W1 7E /r VMOVQ xmm1, xmm2/m64 C 有效/有效 AVX512F 将四字从 xmm2/m64 移动到 xmm1。
66 0F D6 /r MOVQ xmm2/m64, xmm1 B 有效/有效 SSE2 将四字从 xmm1 移动到 xmm2/mem64。
VEX.128.66.0F.WIG D6 /r VMOVQ xmm1/m64, xmm2 B 有效/有效 AVX 将四字从 xmm2 寄存器移动到 xmm1/m64。
EVEX.128.66.0F.W1 D6 /r VMOVQ xmm1/m64, xmm2 D 有效/有效 AVX512F 将四字从 xmm2 寄存器移动到 xmm1/m64。

描述¶

将四字从源操作数（第二个操作数）复制到目的操作数（第一个操作数）。源操作数和目的操作数可以是 MMX 技术寄存器、XMM 寄存器或 64 位内存位置。此指令可用于在两个 MMX 技术寄存器之间，或在 MMX 技术寄存器与 64 位内存位置之间移动四字；或者在两个 XMM 寄存器之间，或在 XMM 寄存器与 64 位内存位置之间移动数据。该指令不能用于在内存位置之间传送数据。

当源操作数是 XMM 寄存器时，移动低四字；当目的操作数是 XMM 寄存器时，该四字存储到寄存器的低四字中，并且高四字被清零。

在 64 位模式下且如果不使用 VEX/EVEX 编码，以 REX.R 形式使用 REX 前缀允许此指令访问其他寄存器 (XMM8-XMM15)。

注意：VEX.vvvv 和 EVEX.vvvv 是保留的，必须为 1111b，否则指令将产生 #UD 异常。

如果 VMOVQ 用 VEX.L= 1 编码，尝试执行用 VEX.L= 1 编码的指令将导致 #UD 异常。

操作¶

MOVQ Instruction When Operating on MMX Technology Registers and Memory Locations¶

DEST := SRC;

MOVQ Instruction When Source and Destination Operands are XMM Registers¶

DEST[63:0] := SRC[63:0];
DEST[127:64] := 0000000000000000H;

MOVQ Instruction When Source Operand is XMM Register and Destination¶

operand is memory location:
    DEST := SRC[63:0];

MOVQ Instruction When Source Operand is Memory Location and Destination¶

operand is XMM register:
    DEST[63:0] := SRC;
    DEST[127:64] := 0000000000000000H;

VMOVQ (VEX.128.F3.0F 7E) With XMM Register Source and Destination¶

DEST[63:0] := SRC[63:0]
DEST[MAXVL-1:64] := 0

VMOVQ (VEX.128.66.0F D6) With XMM Register Source and Destination¶

DEST[63:0] := SRC[63:0]
DEST[MAXVL-1:64] := 0

VMOVQ (7E - EVEX Encoded Version) With XMM Register Source and Destination¶

DEST[63:0] := SRC[63:0]
DEST[MAXVL-1:64] := 0

VMOVQ (D6 - EVEX Encoded Version) With XMM Register Source and Destination¶

DEST[63:0] := SRC[63:0]
DEST[MAXVL-1:64] := 0

VMOVQ (7E) With Memory Source¶

DEST[63:0] := SRC[63:0]
DEST[MAXVL-1:64] := 0

VMOVQ (7E - EVEX Encoded Version) With Memory Source¶

DEST[63:0] := SRC[63:0]
DEST[:MAXVL-1:64] := 0

VMOVQ (D6) With Memory DEST¶

DEST[63:0] := SRC2[63:0]

无。
