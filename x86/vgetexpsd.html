VGETEXPSD — 将标量双精度浮点值的指数转换为双精度浮点值

操作码/指令 Op/En 64/32位模式支持 CPUID 功能标志 描述
EVEX.LLIG.66.0F38.W1 43 /r VGETEXPSD xmm1 {k1}{z}, xmm2, xmm3/m64{sae} A V/V AVX512F 将 xmm3/m64 中低位双精度浮点值的有偏指数（第 62:52 位）转换为表示无偏整数指数的双精度浮点值。在写掩码 k1 下将结果存储到 xmm1 的低 64 位，并与 xmm2 的其他元素合并。

指令操作数编码¶

Op/En 元组类型 操作数 1 操作数 2 操作数 3 操作数 4
A Tuple1 Scalar ModRM:reg (w) EVEX.vvvv (r) ModRM:r/m (r) N/A

描述¶

从源操作数（第三操作数）低位四字数据元素的规范化双精度浮点表示中提取有偏指数作为无偏有符号整数值，或者将输入数据的非规范表示转换为无偏负整数值。无偏指数的整数值被转换为双精度浮点值，并作为双精度浮点数写入目的操作数（第一操作数）。目的 XMM 寄存器的第 (127:64) 位复制自第一源操作数中的对应位。

目的操作数必须是 XMM 寄存器，源操作数可以是 XMM 寄存器或 float64 内存位置。

如果使用写掩码，目的操作数低位的四字元素将根据写掩码寄存器 k1 的值有条件地更新。如果不使用写掩码，目的操作数的低位四字元素将被无条件更新。

每个 GETEXP 操作都会将指数值转换为浮点数（允许非规范表示的输入值）。输入值的特殊情况在 表 5-15 中列出。

公式为：

GETEXP(x) = floor(log2(|x|))

记号 floor(x) 表示不超过实数 x 的最大整数。

VGETEXPSD (EVEX 编码版本)¶

IF k1[0] OR *no writemask*
    THEN DEST[63:0] :=
            ConvertExpDPFP(SRC2[63:0])
    ELSE
        IF *merging-masking*
            THEN *DEST[63:0] remains unchanged*
            ELSE ; zeroing-masking
                DEST[63:0] := 0
        FI
FI;
DEST[127:64] := SRC1[127:64]
DEST[MAXVL-1:128] := 0
