指令操作数编码¶

Op/En 元组 操作数 1 操作数 2 操作数 3 操作数 4
A Scalar ModRM:reg (r, w) VEX.vvvv (r) ModRM:r/m (r) N/A

描述¶

此指令使用三个源操作数对低位 FP16 值执行标量乘减或取反乘减计算，并将结果写入目的操作数。目的操作数也是第一个源操作数。此指令的“N”（取反）形式从取反的无限精度中间积中减去剩余的操作数。记号“132”、“213”和“231”表示操作数在 ±A * B − C 中的使用方式，其中每个数字对应操作数编号，目的操作数为操作数 1；请参阅 Table 5-7。

目的操作数的 [127:16] 位保持不变。目的操作数的 [MAXVL-1:128] 位被清零。目的操作数的低位 FP16 元素根据写掩码进行更新。

操作¶

VF[,N]MSUB132SH DEST, SRC2, SRC3 (EVEX 编码版本)¶

IF EVEX.b = 1 and SRC3 is a register:
    SET_RM(EVEX.RC)
ELSE
    SET_RM(MXCSR.RC)
IF k1[0] OR *no writemask*:
    IF *negative form*:
        DEST.fp16[0] := RoundFPControl(-DEST.fp16[0]*SRC3.fp16[0] - SRC2.fp16[0])
    ELSE:
        DEST.fp16[0] := RoundFPControl(DEST.fp16[0]*SRC3.fp16[0] - SRC2.fp16[0])
ELSE IF *zeroing*:
    DEST.fp16[0] := 0
// else DEST.fp16[0] remains unchanged
//DEST[127:16] remains unchanged
DEST[MAXVL-1:128] := 0
