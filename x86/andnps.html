描述¶

对来自第一源操作数和第二源操作数的四个、八个或十六个打包单精度浮点值执行按位逻辑非与（AND NOT）操作，并将结果存储在目的操作数中。

EVEX 编码版本：第一源操作数是 ZMM/YMM/XMM 寄存器。第二源操作数可以是 ZMM/YMM/XMM 寄存器、512/256/128 位内存位置，或者是从 32 位内存位置广播的 512/256/128 位向量。目的操作数是根据写掩码 k1 条件更新的 ZMM/YMM/XMM 寄存器。

VEX.256 编码版本：第一源操作数是 YMM 寄存器。第二源操作数是 YMM 寄存器或 256 位内存位置。目的操作数是 YMM 寄存器。对应 ZMM 寄存器目标的高位（MAXVL-1:256）清零。

VEX.128 编码版本：第一源操作数是 XMM 寄存器。第二源操作数是 XMM 寄存器或 128 位内存位置。目的操作数是 XMM 寄存器。对应 ZMM 寄存器目标的高位（MAXVL-1:128）清零。

128 位传统 SSE 版本：第二源操作数可以是 XMM 寄存器或 128 位内存位置。目标操作数与第一源操作数 XMM 寄存器相同，并且对应 ZMM 寄存器目标的高位（MAXVL-1:128）保持不变。

操作¶

VANDNPS (EVEX Encoded Versions)¶

(KL, VL) = (4, 128), (8, 256), (16, 512)
FOR j := 0 TO KL-1
    i := j * 32
    IF k1[j] OR *no writemask*
            IF (EVEX.b == 1) AND (SRC2 *is memory*)
                THEN
                    DEST[i+31:i] := (NOT(SRC1[i+31:i])) BITWISE AND SRC2[31:0]
                ELSE
                    DEST[i+31:i] := (NOT(SRC1[i+31:i])) BITWISE AND SRC2[i+31:i]
            FI;
        ELSE
            IF *merging-masking* ; merging-masking
                THEN *DEST[i+31:i] remains unchanged*
                ELSE ; zeroing-masking
                    DEST[i+31:i] = 0
            FI;
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0
