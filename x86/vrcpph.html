VRCPPH — 计算打包 FP16 值的近似倒数

操作码/指令 Op/En 64/32位模式支持 CPUID 功能标志 描述
EVEX.128.66.MAP6.W0 4C /r VRCPPH xmm1{k1}{z}, xmm2/m128/m16bcst A V/V AVX512-FP16 AVX512VL 在写掩码 k1 控制下，计算 xmm2/m128/m16bcst 中打包 FP16 值的近似倒数并将结果存储在 xmm1 中。
EVEX.256.66.MAP6.W0 4C /r VRCPPH ymm1{k1}{z}, ymm2/m256/m16bcst A V/V AVX512-FP16 AVX512VL 在写掩码 k1 控制下，计算 ymm2/m256/m16bcst 中打包 FP16 值的近似倒数并将结果存储在 ymm1 中。
EVEX.512.66.MAP6.W0 4C /r VRCPPH zmm1{k1}{z}, zmm2/m512/m16bcst A V/V AVX512-FP16 在写掩码 k1 控制下，计算 zmm2/m512/m16bcst 中打包 FP16 值的近似倒数并将结果存储在 zmm1 中。

指令操作数编码¶

Op/En 元组 操作数 1 操作数 2 操作数 3 操作数 4
A Full ModRM:reg (w) ModRM:r/m (r) N/A N/A

描述¶

此指令对源操作数（第二操作数）中的 8/16/32 个打包 FP16 值执行 SIMD 近似倒数计算，并将打包 FP16 结果存储在目的操作数中。该近似的最大相对误差小于 2−11 + 2−14。

特殊情况请参阅 表 5-28。

操作¶

VRCPPH dest{k1}, src¶

VL = 128, 256 or 512
KL := VL/16
FOR i := 0 to KL-1:
    IF k1[i] or *no writemask*:
        IF SRC is memory and (EVEX.b = 1):
            tsrc := src.fp16[0]
        ELSE:
            tsrc := src.fp16[i]
        DEST.fp16[i] := APPROXIMATE(1.0 / tsrc)
    ELSE IF *zeroing*:
        DEST.fp16[i] := 0
    //else DEST.fp16[i] remains unchanged
DEST[MAXVL-1:VL] := 0
