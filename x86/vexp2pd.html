VEXP2PD — 打包双精度浮点值指数 2^x 的近似值，相对误差小于 2^-23

操作码/指令 Op/En 64/32位模式支持 CPUID 特征标志 描述
EVEX.512.66.0F38.W1 C8 /r VEXP2PD zmm1 {k1}{z}, zmm2/m512/m64bcst {sae} A V/V AVX512ER 计算 zmm2/m512/m64bcst 中打包双精度浮点值的指数 2^x 近似值（最大相对误差小于 2^-23），并在写掩码 k1 的控制下将浮点结果存储在 zmm1 中。

指令操作数编码¶

Op/En 元组类型 操作数 1 操作数 2 操作数 3 操作数 4

描述¶

计算源操作数（第二个操作数）中双精度浮点值的近似以 2 为底的指数估算值，并在写掩码 k1 控制下将结果存储到目的操作数（第一个操作数）中。近似的以 2 为底的指数估算的相对误差小于 2^-23。

非正规输入值被视为零，且不触发 #DE 信号，无论 MXCSR.DAZ 如何。非正规结果被清除为零（FTZ），且不触发 #UE 信号，无论 MXCSR.FTZ 如何。

源操作数是一个 ZMM 寄存器、一个 512 位内存位置或一个从 64 位内存位置广播的 512 位向量。目的操作数是一个 ZMM 寄存器，使用写掩码 k1 进行条件更新。

EVEX.vvvv 保留且必须为 1111b，否则指令将产生 #UD。

VEXP2xx 的数值精确实现可在 IA 近似指令 vrcp14、vrsqrt14、vrcp28、vrsqrt28、vexp2 的参考实现 找到。

操作¶

VEXP2PD¶

(KL, VL) = (8, 512)
FOR j := 0 TO KL-1
    i := j * 64
    IF k1[j] OR *no writemask* THEN
            IF (EVEX.b = 1) AND (SRC *is memory*)
                THEN DEST[i+63:i] := EXP2_23_DP(SRC[63:0])
                ELSE DEST[i+63:i] := EXP2_23_DP(SRC[i+63:i])
            FI;
    ELSE
        IF *merging-masking* ; merging-masking
            THEN *DEST[i+63:i] remains unchanged*
            ELSE ; zeroing-masking
                DEST[i+63:i] := 0
        FI;
    FI;
ENDFOR;
