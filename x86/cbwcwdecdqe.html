CBW/CWDE/CDQE — 字节转换为字/字转换为双字/双字转换为四字

操作码 指令 Op/En 64位模式 兼容/传统模式 描述
98 CBW ZO Valid Valid AX := AL 的符号扩展。
98 CWDE ZO Valid Valid EAX := AX 的符号扩展。
REX.W + 98 CDQE ZO Valid N.E. RAX := EAX 的符号扩展。

指令操作数编码 ¶

Op/En 操作数 1 操作数 2 操作数 3 操作数 4
ZO N/A N/A N/A N/A

描述 ¶

通过符号扩展将源操作数的大小翻倍。CBW（字节转换为字）指令将源操作数中的符号位（第 7 位）复制到 AH 寄存器的每一个位中。CWDE（字转换为双字）指令将 AX 寄存器中字的符号位（第 15 位）复制到 EAX 寄存器的高 16 位中。

CBW 和 CWDE 使用相同的操作码。CBW 指令设计在操作数大小属性为 16 时使用；CWDE 设计在操作数大小属性为 32 时使用。某些汇编器可能会强制操作数大小，而另外一些汇编器可能会将这两个助记符视为同义词（CBW/CWDE），并使用操作数大小属性的设置来确定要转换的值的大小。

在 64 位模式下，默认操作大小是目的寄存器的大小。使用 REX.W 前缀可将此指令提升为对 64 位操作数进行操作（提升后为 CDQE）。在这种情况下，CDQE 将 EAX 寄存器中双字的符号位（第 31 位）复制到 RAX 的高 32 位中。

操作 ¶ 

IF OperandSize = 16 (* Instruction = CBW *)
    THEN
        AX := SignExtend(AL);
    ELSE IF (OperandSize = 32, Instruction = CWDE)
        EAX := SignExtend(AX); FI;
    ELSE (* 64-Bit Mode, OperandSize = 64, Instruction = CDQE*)
        RAX := SignExtend(EAX);
FI;

受影响的标志位 ¶

无。
