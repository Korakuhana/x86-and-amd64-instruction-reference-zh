KNOTW/KNOTB/KNOTQ/KNOTD — 取反掩码寄存器

操作码/指令 Op/En 64/32位模式支持 CPUID 特征标志 描述
VEX.L0.0F.W0 44 /r KNOTW k1, k2 RR V/V AVX512F 对 16 位掩码 k2 进行按位取反。
VEX.L0.66.0F.W0 44 /r KNOTB k1, k2 RR V/V AVX512DQ 对 8 位掩码 k2 进行按位取反。
VEX.L0.0F.W1 44 /r KNOTQ k1, k2 RR V/V AVX512BW 对 64 位掩码 k2 进行按位取反。
VEX.L0.66.0F.W1 44 /r KNOTD k1, k2 RR V/V AVX512BW 对 32 位掩码 k2 进行按位取反。

指令操作数编码¶

Op/En 操作数 1 操作数 2
RR ModRM:reg (w) ModRM:r/m (r, ModRM:[7:6] 必须是 11b)

描述¶

对向量掩码 k2 进行按位取反操作，并将结果写入向量掩码 k1。

操作¶

KNOTW¶

DEST[15:0] := BITWISE NOT SRC[15:0]
DEST[MAX_KL-1:16] := 0

KNOTB¶

DEST[7:0] := BITWISE NOT SRC[7:0]
DEST[MAX_KL-1:8] := 0

KNOTQ¶

DEST[63:0] := BITWISE NOT SRC[63:0]
DEST[MAX_KL-1:64] := 0

KNOTD¶

DEST[31:0] := BITWISE NOT SRC[31:0]
DEST[MAX_KL-1:32] := 0

Intel C/C++ 编译器等价内部函数¶

KNOTW __mmask16 _mm512_knot(__mmask16 a);
