描述¶

对源操作数（第二个操作数）中的四个、八个或十六个打包单精度浮点值进行 SIMD 平方根计算，并将打包单精度浮点结果存储在目的操作数中。

EVEX 编码版本：源操作数是 ZMM/YMM/XMM 寄存器、512/256/128 位内存位置，或从 32 位内存位置广播的 512/256/128 位向量。目的操作数是根据写掩码更新的 ZMM/YMM/XMM 寄存器。

VEX.256 编码版本：源操作数是 YMM 寄存器或 256 位内存位置。目的操作数是 YMM 寄存器。对应 ZMM 寄存器目的操作数的高位 (MAXVL-1:256) 被清零。

VEX.128 编码版本：源操作数是 XMM 寄存器或 128 位内存位置。目的操作数是 XMM 寄存器。对应 ZMM 寄存器目的操作数的高位 (MAXVL-1:128) 被清零。

128 位传统 SSE 版本：第二源操作数可以是 XMM 寄存器或 128 位内存位置。目的操作数与第一源 XMM 寄存器相同，且对应 ZMM 寄存器目的操作数的高位 (MAXVL-1:128) 保持不变。

注意：VEX.vvvv 和 EVEX.vvvv 保留且必须为 1111b，否则指令将产生 #UD 异常。

操作¶

VSQRTPS（EVEX 编码版本）¶

(KL, VL) = (4, 128), (8, 256), (16, 512)
IF (VL = 512) AND (EVEX.b = 1) AND (SRC *is register*)
    THEN
        SET_ROUNDING_MODE_FOR_THIS_INSTRUCTION(EVEX.RC);
    ELSE
        SET_ROUNDING_MODE_FOR_THIS_INSTRUCTION(MXCSR.RC);
FI;
FOR j := 0 TO KL-1
    i := j * 32
    IF k1[j] OR *no writemask* THEN
            IF (EVEX.b = 1) AND (SRC *is memory*)
                THEN DEST[i+31:i] := SQRT(SRC[31:0])
                ELSE DEST[i+31:i] := SQRT(SRC[i+31:i])
            FI;
        ELSE
            IF *merging-masking* ; merging-masking
                THEN *DEST[i+31:i] remains unchanged*
                ELSE ; zeroing-masking
                    DEST[i+31:i] := 0
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0

#UD 如果 VEX.vvvv != 1111B。

#UD 如果 EVEX.vvvv != 1111B。
