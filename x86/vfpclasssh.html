VFPCLASSSH — 测试标量 FP16 值的类型

操作码/指令 Op/En 64/32位模式支持 CPUID 特征标志 描述
EVEX.LLIG.NP.0F3A.W0 67 /r /ib VFPCLASSSH k1{k2}, xmm1/m16, imm8 A V/V AVX512-FP16 测试输入的以下类别：NaN、+0、-0、+Infinity、-Infinity、非正规数、有限负数。立即数字段为每个类别测试提供了一个掩码位。掩码测试结果进行逻辑或（OR）组合以形成掩码结果。

指令操作数编码¶

Op/En 元组类型 操作数 1 操作数 2 操作数 3 操作数 4
A Scalar ModRM:reg (w) ModRM:r/m (r) imm8 (r) N/A

描述¶

该指令检查源操作数中的低位 FP16 值是否属于由 imm8 字节中设置的位指定的特殊类别；类别请参见表 5-9。输入值的所有指定类别的分类结果进行逻辑或（OR）运算，以形成输入元素的最终布尔结果。结果根据写掩码写入目的掩码寄存器中的低位。目的掩码寄存器中的其他位被清零。

操作¶

VFPCLASSSH dest{k2}, src, imm8¶

IF k2[0] or *no writemask*:
    DEST.bit[0] := check_fp_class_fp16(src.fp16[0], imm8)
        // see VFPCLASSPH
ELSE:
    DEST.bit[0] := 0
DEST[MAXKL-1:1] := 0
