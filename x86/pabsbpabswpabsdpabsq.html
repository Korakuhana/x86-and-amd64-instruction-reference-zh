描述¶

PABSB/W/D 计算源操作数（第二个操作数）中每个数据元素的绝对值，并将无符号结果存储在目的操作数（第一个操作数）中。PABSB 对有符号字节进行操作，PABSW 对有符号 16 位字进行操作，PABSD 对有符号 32 位整数进行操作。

EVEX 编码的 VPABSD/Q：源操作数是 ZMM/YMM/XMM 寄存器、512/256/128 位内存位置，或者从 32/64 位内存位置广播的 512/256/128 位向量。目的操作数是根据写掩码更新的 ZMM/YMM/XMM 寄存器。

EVEX 编码的 VPABSB/W：源操作数是 ZMM/YMM/XMM 寄存器或 512/256/128 位内存位置。目的操作数是根据写掩码更新的 ZMM/YMM/XMM 寄存器。

VEX.256 编码版本：源操作数是 YMM 寄存器或 256 位内存位置。目的操作数是 YMM 寄存器。对应寄存器目的操作数的高位（MAXVL-1:256）清零。

VEX.128 编码版本：源操作数是 XMM 寄存器或 128 位内存位置。目的操作数是 XMM 寄存器。对应寄存器目的操作数的高位（MAXVL-1:128）清零。

128位传统 SSE 版本：源操作数可以是 XMM 寄存器或 128 位内存位置。目的操作数是 XMM 寄存器。对应寄存器目的操作数的高位（VL_MAX-1:128）保持不变。

VEX.vvvv 和 EVEX.vvvv 是保留的，必须为 1111b，否则指令将触发 #UD。

VPABSB（EVEX 编码版本）¶

(KL, VL) = (16, 128), (32, 256), (64, 512)
FOR j := 0 TO KL-1
    i := j * 8
    IF k1[j] OR *no writemask*
        THEN
            Unsigned DEST[i+7:i] := ABS(SRC[i+7:i])
        ELSE
            IF *merging-masking* ; merging-masking
                THEN *DEST[i+7:i] remains unchanged*
                ELSE *zeroing-masking*
                        ; zeroing-masking
                    DEST[i+7:i] := 0
            FI
    FI;
ENDFOR;
DEST[MAXVL-1:VL] := 0

FOR j := 0 TO KL-1
    i := j * 32
