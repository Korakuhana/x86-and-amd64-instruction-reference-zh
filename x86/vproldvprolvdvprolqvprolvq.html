描述¶

将第一源操作数中各个数据元素（双字或四字）中的位，按计数操作数中指定的位数向左循环移位。如果计数操作数指定的值大于 31（对于双字）或 63（对于四字），则使用计数操作数模数据大小（32 或 64）的结果。

EVEX.128 编码版本：目的操作数是 XMM 寄存器。源操作数是 XMM 寄存器或内存位置（用于立即数形式）。计数操作数可以来自 XMM 寄存器、内存位置或 8 位立即数。对应 ZMM 寄存器的 bits (MAXVL-1:128) 清零。

EVEX.256 编码版本：目的操作数是 YMM 寄存器。源操作数是 YMM 寄存器或内存位置（用于立即数形式）。计数操作数可以来自 XMM 寄存器、内存位置或 8 位立即数。对应 ZMM 寄存器的 bits (MAXVL-1:256) 清零。

EVEX.512 编码版本：目的操作数是根据写掩码更新的 ZMM 寄存器。对于立即数形式的计数操作数，源操作数可以是 ZMM 寄存器、512 位内存位置或从 32/64 位内存位置广播的 512 位向量，计数操作数是 8 位立即数。对于可变形式的计数操作数，第一源操作数（第二操作数）是 ZMM 寄存器，计数器操作数（第三操作数）是 ZMM 寄存器、512 位内存位置，或从 32/64 位内存位置广播的 512 位向量。

操作¶

LEFT_ROTATE_DWORDS(SRC, COUNT_SRC)
COUNT := COUNT_SRC modulo 32;
DEST[31:0] := (SRC << COUNT) | (SRC >> (32 - COUNT));
LEFT_ROTATE_QWORDS(SRC, COUNT_SRC)
COUNT := COUNT_SRC modulo 64;
DEST[63:0] := (SRC << COUNT) | (SRC >> (64 - COUNT));

VPROLD (EVEX 编码版本)¶

(KL, VL) = (4, 128), (8, 256), (16, 512)
FOR j := 0 TO KL-1
    i := j * 32
    IF k1[j] OR *no writemask* THEN
            IF (EVEX.b = 1) AND (SRC1 *is memory*)
                THEN DEST[i+31:i] := LEFT_ROTATE_DWORDS(SRC1[31:0], imm8)
                ELSE DEST[i+31:i] := LEFT_ROTATE_DWORDS(SRC1[i+31:i], imm8)
            FI;
        ELSE
            IF *merging-masking* ; merging-masking
                THEN *DEST[i+31:i] remains unchanged*
                ELSE *zeroing-masking*
                        ; zeroing-masking
                    DEST[i+31:i] := 0
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0
