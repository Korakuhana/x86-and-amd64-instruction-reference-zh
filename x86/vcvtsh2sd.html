VCVTSH2SD — 将低位 FP16 值转换为 FP64 值

操作码/指令 Op/En 64/32位模式支持 CPUID 特征标志 描述
EVEX.LLIG.F3.MAP5.W0 5A /r VCVTSH2SD xmm1{k1}{z}, xmm2, xmm3/m16 {sae} A V/V AVX512-FP16 在写掩码 k1 的控制下，将 xmm3/m16 中的低位 FP16 值转换为 FP64 值，并将结果存储在 xmm1 的低位元素中。xmm2 的 127:64 位被复制到 xmm1[127:64]。

指令操作数编码¶

Op/En 元组 操作数 1 操作数 2 操作数 3 操作数 4
A Scalar ModRM:reg (w) VEX.vvvv (r) ModRM:r/m (r) N/A

描述¶

此指令将第二个源操作数中的低位 FP16 元素转换为目的操作数低位元素中的 FP64 元素。

目的操作数的 127:64 位从第一个源操作数的对应位复制。目的操作数的 MAXVL-1:128 位清零。目的寄存器的低位 FP64 元素根据写掩码进行更新。

操作¶

VCVTSH2SD dest, src1, src2¶

IF k1[0] OR *no writemask*:
    DEST.fp64[0] := Convert_fp16_to_fp64(SRC2.fp16[0])
ELSE IF *zeroing*:
    DEST.fp64[0] := 0
// else dest.fp64[0] remains unchanged
DEST[127:64] := SRC1[127:64]
DEST[MAXVL-1:128] := 0
