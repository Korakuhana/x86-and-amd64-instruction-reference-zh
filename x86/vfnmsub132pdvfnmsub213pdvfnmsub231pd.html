描述¶

VFNMSUB132PD：将来自第一个源操作数的两个、四个或八个打包双精度浮点值与第三个源操作数中的两个、四个或八个打包双精度浮点值相乘。从取反的无限精度中间结果中，减去第二个源操作数中的两个、四个或八个打包双精度浮点值，执行舍入并将结果的两个、四个或八个打包双精度浮点值存储到目的操作数（第一个源操作数）中。

VFNMSUB213PD：将来自第二个源操作数的两个、四个或八个打包双精度浮点值与第一个源操作数中的两个、四个或八个打包双精度浮点值相乘。从取反的无限精度中间结果中，减去第三个源操作数中的两个、四个或八个打包双精度浮点值，执行舍入并将结果的两个、四个或八个打包双精度浮点值存储到目的操作数（第一个源操作数）中。

VFNMSUB231PD：将来自第二个源操作数的两个、四个或八个打包双精度浮点值与第三个源操作数中的两个、四个或八个打包双精度浮点值相乘。从取反的无限精度中间结果中，减去第一个源操作数中的两个、四个或八个打包双精度浮点值，执行舍入并将结果的两个、四个或八个打包双精度浮点值存储到目的操作数（第一个源操作数）中。

EVEX 编码版本：目的操作数（也是第一个源操作数）和第二个源操作数是 ZMM/YMM/XMM 寄存器。第三个源操作数是 ZMM/YMM/XMM 寄存器、512/256/128 位内存位置或从 64 位内存位置广播的 512/256/128 位向量。目的操作数根据写掩码 k1 进行有条件地更新。

VEX.256 编码版本：目的操作数（也是第一个源操作数）是 YMM 寄存器，并在 reg_field 中编码。第二个源操作数是 YMM 寄存器，并在 VEX.vvvv 中编码。第三个源操作数是 YMM 寄存器或 256 位内存位置，并在 rm_field 中编码。

VEX.128 编码版本：目的操作数（也是第一个源操作数）是 XMM 寄存器，并在 reg_field 中编码。第二个源操作数是 XMM 寄存器，并在 VEX.vvvv 中编码。第三个源操作数是 XMM 寄存器或 128 位内存位置，并在 rm_field 中编码。YMM 目的寄存器的高 128 位被清零。

操作¶

在以下操作中，“*”和“-”符号表示使用无限精度输入和输出进行乘法和减法（无舍入）。

VFNMSUB132PD DEST, SRC2, SRC3 (VEX 编码版本)¶

IF (VEX.128) THEN
    MAXNUM := 2
ELSEIF (VEX.256)
    MAXNUM := 4
FI
For i = 0 to MAXNUM-1 {
    n := 64*i;
    DEST[n+63:n] := RoundFPControl_MXCSR( - (DEST[n+63:n]*SRC3[n+63:n]) - SRC2[n+63:n])
}
IF (VEX.128) THEN
    DEST[MAXVL-1:128] := 0
ELSEIF (VEX.256)
    DEST[MAXVL-1:256] := 0
FI

VFNMSUB132PD DEST, SRC2, SRC3 (EVEX 编码版本，当 src3 操作数为寄存器时)¶

(KL, VL) = (2, 128), (4, 256), (8, 512)
IF (VL = 512) AND (EVEX.b = 1)
    THEN
        SET_ROUNDING_MODE_FOR_THIS_INSTRUCTION(EVEX.RC);
    ELSE
        SET_ROUNDING_MODE_FOR_THIS_INSTRUCTION(MXCSR.RC);
FI;
FOR j := 0 TO KL-1
    i := j * 64
    IF k1[j] OR *no writemask*
        THEN DEST[i+63:i] :=
            RoundFPControl(-(DEST[i+63:i]*SRC3[i+63:i]) - SRC2[i+63:i])
        ELSE
            IF *merging-masking* ; merging-masking
                THEN *DEST[i+63:i] remains unchanged*
                ELSE ; zeroing-masking
                    DEST[i+63:i] := 0
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0
