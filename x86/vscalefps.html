描述¶

对第一源操作数中的打包单精度浮点值进行浮点缩放：将其乘以 2 的“第二源操作数中 float32 值”次幂。

此运算的方程式为：

zmm1 := zmm2 * 2floor(zmm3)

Floor(zmm3) 表示 ≤ zmm3 的最大整数值。

如果结果无法用单精度表示，则发出适当的溢出响应（对于正缩放操作数）或适当的下溢响应（对于负缩放操作数）。溢出和下溢响应取决于舍入模式（适用于符合 IEEE 标准的舍入），以及 MXCSR 中的其他设置（异常掩码位、FTZ 位）和 SAE 位。

EVEX.512 编码版本：第一源操作数是 ZMM 寄存器。第二源操作数是 ZMM 寄存器、512 位内存位置，或从 32 位内存位置广播的 512 位向量。目的操作数是根据写掩码 k1 有条件更新的 ZMM 寄存器。

EVEX.256 编码版本：第一源操作数是 YMM 寄存器。第二源操作数是 YMM 寄存器、256 位内存位置，或从 32 位内存位置广播的 256 位向量。目的操作数是 YMM 寄存器，在写掩码 k1 控制下有条件地更新。

EVEX.128 编码版本：第一源操作数是 XMM 寄存器。第二源操作数是 XMM 寄存器、128 位内存位置，或从 32 位内存位置广播的 128 位向量。目的操作数是 XMM 寄存器，在写掩码 k1 控制下有条件地更新。

特殊情况输入值的处理列在 表 5-39 和 表 5-43 中。

操作¶

SCALE(SRC1, SRC2)
{ ; Check for denormal operands
TMP_SRC2 := SRC2
TMP_SRC1 := SRC1
IF (SRC2 is denormal AND MXCSR.DAZ) THEN TMP_SRC2=0
IF (SRC1 is denormal AND MXCSR.DAZ) THEN TMP_SRC1=0
/* SRC2 is a 32 bits floating-point value */
DEST[31:0] := TMP_SRC1[31:0] * POW(2, Floor(TMP_SRC2[31:0]))
}

VSCALEFPS (EVEX 编码版本)¶

(KL, VL) = (4, 128), (8, 256), (16, 512)
IF (VL = 512) AND (EVEX.b = 1) AND (SRC2 *is register*)
    THEN
        SET_ROUNDING_MODE_FOR_THIS_INSTRUCTION(EVEX.RC);
    ELSE
        SET_ROUNDING_MODE_FOR_THIS_INSTRUCTION(MXCSR.RC);
FI;
FOR j := 0 TO KL-1
    i := j * 32
    IF k1[j] OR *no writemask* THEN
            IF (EVEX.b = 1) AND (SRC2 *is memory*)
                THEN DEST[i+31:i] := SCALE(SRC1[i+31:i], SRC2[31:0]);
                ELSE DEST[i+31:i] := SCALE(SRC1[i+31:i], SRC2[i+31:i]);
            FI;
        ELSE
            IF *merging-masking* ; merging-masking
                THEN *DEST[i+31:i] remains unchanged*
                ELSE ; zeroing-masking
                    DEST[i+31:i] := 0
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0;

SIMD 浮点异常¶

溢出, 下溢, 无效, 精度, 渐近下溢（针对 Src1）。

Src2 不报告渐近下溢异常。
