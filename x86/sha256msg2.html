SHA256MSG2 — 为下四个 SHA256 消息双字执行最终计算

操作码/指令 Op/En 64/32位模式支持 CPUID 特征标志 描述
NP 0F 38 CD /r SHA256MSG2 xmm1, xmm2/m128 RM V/V SHA 使用来自 xmm1 和 xmm2/m128 的前几个消息双字，为接下来的四个 SHA256 消息双字执行最终计算，并将结果存储在 xmm1 中。

指令操作数编码¶

Op/En 操作数 1 操作数 2 操作数 3
RM ModRM:reg (r, w) ModRM:r/m (r) N/A

描述¶

SHA256MSG2 指令是两条 SHA2 消息调度指令之一。该指令执行最终计算以得到接下来的四个 SHA256 消息双字。

操作¶

SHA256MSG2¶

W14 := SRC2[95:64] ;
W15 := SRC2[127:96] ;
W16 := SRC1[31: 0] + σ1( W14) ;
W17 := SRC1[63: 32] + σ1( W15) ;
W18 := SRC1[95: 64] + σ1( W16) ;
W19 := SRC1[127: 96] + σ1( W17) ;
DEST[127:96] := W19 ;
DEST[95:64] := W18 ;
DEST[63:32] := W17 ;
DEST[31:0] := W16;
