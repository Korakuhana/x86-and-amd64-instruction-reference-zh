描述¶

ENQCMD 指令允许软件向入队寄存器 (enqueue registers)写入命令，入队寄存器是通过内存映射 I/O (MMIO) 访问的特殊设备寄存器。

入队寄存器期望的写入格式如下：

位 19:0 传送进程地址空间标识符 (PASID)，这是系统软件可分配给单个软件线程的值。位 31 包含特权标识（0 = 用户；1 = 管理员）。实现入队寄存器的设备可以使用这两个值以及高 60 字节中特定于设备的命令。

ENQCMD 指令首先从源内存操作数中读取 64 字节的命令数据。这是一种普通加载，其可缓存性和内存顺序通常由内存类型隐式定义。源操作数不需要对齐，并且不保证所有 64 字节都是原子加载的。源操作数的位 31:0 必须为零。

然后该指令将这 64 字节格式化为与图 3-16所给格式一致的命令数据：

  • Command[19:0] 获取 IA32_PASID[19:0]。1
  • Command[30:20] 为零。
  • Command[31] 为 0（表示用户级；无论 CPL 为多少，都使用此值）。
  • Command[511:32] 获取从内存读取的源操作数的位 511:32。

ENQCMD 指令使用入队存储 (enqueue store)（定义如下）将此命令 data 写入目标操作数。目标操作数的地址在通用寄存器中指定为 ES 段的偏移量（段不能被覆盖）。2 目标线性地址必须对齐到 64 字节边界。入队存储的操作忽略目标内存地址的内存类型。

1. 系统软件应该加载 IA32_PASID MSR，以便位 19:0 包含当前软件线程的 PASID。MSR 的有效位 IA32_PASID[31] 必须为 1。有关 IA32_PASID MSR 的其他详细信息，请参见 Intel® 64 和 IA-32 架构软件开发人员手册卷 4。

2. 在 64 位模式下，寄存器操作数的宽度为 64 位（带有 67H 前缀时为 32 位）。在非 64 位模式下，当 CS.D = 1时，宽度为 32 位（带有 67H 前缀时为 16 位）。在非 64 位模式下，当 CS.D = 0 时，宽度为 16 位（带有 67H 前缀时为 32 位）。

入队存储的排序与之前对 WB 或 WC 内存的存储（包括 non-temporal 存储）或 CLFLUSHOPT 或 CLWB 的执行（当应用于入队存储地址以外的地址时）无关。软件可以通过在入队存储之前执行内存屏障指令（如 SFENCE 或 MFENCE）来强制执行此类排序。

入队存储不会将数据写入缓存层次结构，也不会将任何数据提取到缓存层次结构中。入队存储的命令数据绝不会与写入同一地址的任何其他存储的数据合并。

与其他存储不同，入队存储会返回状态，ENQCMD 指令将该状态加载到 RFLAGS 寄存器中的 ZF 标志中：

  • ZF = 0（成功）表示 64 字节的命令数据已原子方式写入到设备的入队寄存器并已被设备接受。（这并不保证设备已经执行了命令；它可能只是排队等候稍后执行。）
  • ZF = 1（重试）表示命令数据未被接受。如果目标地址是入队寄存器，但由于容量或其他临时原因导致命令未被接受，则返回此状态。如果目标地址不是入队寄存器（包括内存地址的情况），也会返回此状态；在这些情况下，存储会被丢弃，并且既不写入 MMIO 也不写入内存。

ENQCMD 指令的可用性由 CPUID 功能标志 ENQCMD（CPUID.(EAX=07H, ECX=0H):ECX[位 29]）指示。

操作¶

IF IA32_PASID[31] = 0
    THEN #GP;
ELSE
    COMMAND := (SRC & ~FFFFFFFFH) | (IA32_PASID & FFFFFH);
    DEST := COMMAND;
FI;

Intel C/C++ 编译器固有等效项¶

ENQCMD int_enqcmd(void *dst, const void *src)

影响的标志寄存器¶

如果入队存储完成返回重试状态，则设置 ZF 标志；否则清除。清除所有其他标志。

SIMD 浮点异常¶

无。

保护模式异常¶

#GP(0) 对于 CS、DS、ES、FS 或 GS 段中非法的内存操作数有效地址。
如果目标线性地址未对齐到 64 字节边界。
如果 IA32_PASID MSR 中的 PASID 有效字段（位 31）为 0。
如果源操作数的位 31:0 不全为零。
#SS(0) 对于 SS 段中的非法地址。
#PF(错误代码) 如果发生页面错误。
#UD 如果 CPUID.07H.0H:ECX.ENQCMD[位 29] = 0。
如果使用了 LOCK 前缀。

#GP 如果操作数的任何部分位于有效地址空间 0 到 FFFFH 之外。
如果目标线性地址未对齐到 64 字节边界。
如果 IA32_PASID MSR 中的 PASID 有效字段（位 31）为 0。
如果源操作数的位 31:0 不全为零。
#UD 如果 CPUID.07H.0H:ECX.ENQCMD[bit 29] = 0。
如果使用了 LOCK 前缀。
