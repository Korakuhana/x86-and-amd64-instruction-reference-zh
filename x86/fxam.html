FXAM — 检查浮点数

操作码 模式 传统模式 描述
D9 E5 对 ST(0) 中的值或数字进行分类。

描述¶

检查 ST(0) 寄存器的内容，并设置 FPU 状态字中的条件码标志 C0、C2 和 C3，以指示该寄存器中值或数字的类别（见下表）。

C1 标志设置为 ST(0) 中值的符号，无论该寄存器是空还是满。

此指令的操作在非 64 位模式和 64 位模式下相同。

操作¶

C1 := sign bit of ST; (* 0 for positive, 1 for negative *)
CASE (class of value or number in ST(0)) OF
    Unsupported:C3, C2, C0 := 000;
    NaN:
        C3, C2, C0 := 001;
    Normal:
        C3, C2, C0 := 010;
    Infinity:
        C3, C2, C0 := 011;
    Zero:
        C3, C2, C0 := 100;
    Empty:
        C3, C2, C0 := 101;
    Denormal:
        C3, C2, C0 := 110;
ESAC;

受影响的 FPU 标志位¶

C1 ST(0) 中值的符号。
C0, C2, C3 参见表 3-42。

浮点异常¶

无。

保护模式异常¶

#NM CR0.EM[bit 2] 或 CR0.TS[bit 3] = 1。
#MF 如果存在待处理 of x87 FPU 异常。
#UD 如果使用了 LOCK 前缀。

实地址模式异常¶

与保护模式下的异常相同。

虚拟-8086 模式异常¶

与保护模式下的异常相同。

兼容模式异常¶

与保护模式下的异常相同。

64位模式异常¶

与保护模式下的异常相同。
