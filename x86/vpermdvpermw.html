VPERMD/VPERMW — 置换打包双字/字元素

操作码/指令 Op / En 64/32位模式支持 CPUID 功能标志 描述
VEX.256.66.0F38.W0 36 /r VPERMD ymm1, ymm2, ymm3/m256 A V/V AVX2 使用 ymm2 中的索引置换 ymm3/m256 中的双字，并将结果存储在 ymm1 中。
EVEX.256.66.0F38.W0 36 /r VPERMD ymm1 {k1}{z}, ymm2, ymm3/m256/m32bcst B V/V AVX512VL AVX512F 使用 ymm2 中的索引置换 ymm3/m256/m32bcst 中的双字，在写掩码 k1 下将结果存储在 ymm1 中。
EVEX.512.66.0F38.W0 36 /r VPERMD zmm1 {k1}{z}, zmm2, zmm3/m512/m32bcst B V/V AVX512F 使用 zmm2 中的索引置换 zmm3/m512/m32bcst 中的双字，在写掩码 k1 下将结果存储在 zmm1 中。
EVEX.128.66.0F38.W1 8D /r VPERMW xmm1 {k1}{z}, xmm2, xmm3/m128 C V/V AVX512VL AVX512BW 使用 xmm2 中的索引置换 xmm3/m128 中的字整数，在写掩码 k1 下将结果存储在 xmm1 中。
EVEX.256.66.0F38.W1 8D /r VPERMW ymm1 {k1}{z}, ymm2, ymm3/m256 C V/V AVX512VL AVX512BW 使用 ymm2 中的索引置换 ymm3/m256 中的字整数，在写掩码 k1 下将结果存储在 ymm1 中。

描述¶

根据第一源操作数（第二操作数）中的索引，将双字（或字）从第二源操作数（第三操作数）复制到目的操作数（第一操作数）。注意，此指令允许将源操作数中的某个双字（或字）复制到目的操作数中的多个位置。

VEX.256 编码的 VPERMD：第一和第二操作数是 YMM 寄存器，第三操作数可以是 YMM 寄存器或内存位置。对应目的寄存器的 bits (MAXVL-1:256) 清零。

EVEX 编码的 VPERMD：第一和第二操作数是 ZMM/YMM 寄存器，第三操作数可以是 ZMM/YMM 寄存器、512/256 位内存位置，或者是从 32 位内存位置广播的 512/256 位向量。目的操作数中的元素在写掩码 k1 下更新。

VPERMW：第一和第二操作数是 ZMM/YMM/XMM 寄存器，第三操作数可以是 ZMM/YMM/XMM 寄存器或 512/256/128 位内存位置。目的操作数在写掩码 k1 下更新。

EVEX.128 编码版本：对应 ZMM 寄存器的 bits (MAXVL-1:128) 清零。

操作¶

VPERMD (EVEX 编码版本)¶

(KL, VL) = (8, 256), (16, 512)
IF VL = 256 THEN n := 2; FI;
IF VL = 512 THEN n := 3; FI;
FOR j := 0 TO KL-1
    i := j * 32
    id := 32*SRC1[i+n:i]
    IF k1[j] OR *no writemask*
        THEN
            IF (EVEX.b = 1) AND (SRC2 *is memory*)
                THEN DEST[i+31:i] := SRC2[31:0];
                ELSE DEST[i+31:i] := SRC2[id+31:id];
            FI;
        ELSE
            IF *merging-masking* ; merging-masking
                THEN *DEST[i+31:i] remains unchanged*
                ELSE ; zeroing-masking
                    DEST[i+31:i] := 0
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0

VPERMD (VEX.256 编码版本)¶

DEST[31:0] := (SRC2[255:0] >> (SRC1[2:0] * 32))[31:0];
DEST[63:32] := (SRC2[255:0] >> (SRC1[34:32] * 32))[31:0];
DEST[95:64] := (SRC2[255:0] >> (SRC1[66:64] * 32))[31:0];
DEST[127:96] := (SRC2[255:0] >> (SRC1[98:96] * 32))[31:0];
DEST[159:128] := (SRC2[255:0] >> (SRC1[130:128] * 32))[31:0];
DEST[191:160] := (SRC2[255:0] >> (SRC1[162:160] * 32))[31:0];
DEST[223:192] := (SRC2[255:0] >> (SRC1[194:192] * 32))[31:0];
DEST[255:224] := (SRC2[255:0] >> (SRC1[226:224] * 32))[31:0];
DEST[MAXVL-1:256] := 0

VPERMW (EVEX 编码版本)¶

(KL, VL) = (8, 128), (16, 256), (32, 512)
IF VL = 128 THEN n := 2; FI;
IF VL = 256 THEN n := 3; FI;
IF VL = 512 THEN n := 4; FI;
FOR j := 0 TO KL-1
    i := j * 16
    id := 16*SRC1[i+n:i]
    IF k1[j] OR *no writemask*
        THEN DEST[i+15:i] := SRC2[id+15:id]
        ELSE
            IF *merging-masking*
                        ; merging-masking
                THEN *DEST[i+15:i] remains unchanged*
                ELSE
                        ; zeroing-masking
                    DEST[i+15:i] := 0
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0
