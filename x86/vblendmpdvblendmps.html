指令操作数编码¶

描述¶

使用操作掩码（opmask）寄存器作为选择控制，在第一源操作数（第二个操作数）中的 float64/float32 元素与第二源操作数（第三个操作数）中的元素之间进行逐元素混合。混合结果写入目的寄存器。

目的操作数和第一源操作数是 ZMM/YMM/XMM 寄存器。第二源操作数可以是 ZMM/YMM/XMM 寄存器、512/256/128 位内存位置，或从 64 位内存位置广播的 512/256/128 位向量。

操作掩码寄存器在此指令中不作为写掩码使用。相反，掩码用作元素选择器：根据相关掩码位的值（对于第一源操作数为 0，对于第二源操作数为 1），有条件地在第一源操作数或第二源操作数之间选择目的操作数的每个元素。

如果设置了 EVEX.z，则目的操作数中对应掩码位值为 0 的元素将被清零。

操作¶

VBLENDMPD（EVEX 编码版本）¶

(KL, VL) = (2, 128), (4, 256), (8, 512)
FOR j := 0 TO KL-1
    i := j * 64
    IF k1[j] OR *no controlmask*
        THEN
            IF (EVEX.b = 1) AND (SRC2 *is memory*)
                THEN
                    DEST[i+63:i] := SRC2[63:0]
                ELSE
                    DEST[i+63:i] := SRC2[i+63:i]
            FI;
        ELSE
            IF *merging-masking* ; merging-masking
                THEN DEST[i+63:i] := SRC1[i+63:i]
                ELSE ; zeroing-masking
                    DEST[i+63:i] := 0
            FI;
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0

VBLENDMPS（EVEX 编码版本）¶

(KL, VL) = (4, 128), (8, 256), (16, 512)
FOR j := 0 TO KL-1
    i := j * 32
    IF k1[j] OR *no controlmask*
        THEN
            IF (EVEX.b = 1) AND (SRC2 *is memory*)
                THEN
                    DEST[i+31:i] := SRC2[31:0]
                ELSE
                    DEST[i+31:i] := SRC2[i+31:i]
            FI;
        ELSE
            IF *merging-masking*
                THEN DEST[i+31:i] := SRC1[i+31:i]
                ELSE ; zeroing-masking
                    DEST[i+31:i] := 0
            FI;
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0
