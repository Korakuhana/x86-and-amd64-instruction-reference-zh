描述¶

此指令对第一源操作数中的打包 FP16 值进行浮点缩放：将其乘以 2 的“第二源操作数中 FP16 值”次幂。在写掩码控制下更新目的操作数中的元素。

此运算的方程式为：

zmm1 := zmm2 * 2floor(zmm3)

Floor(zmm3) 表示 ≤ zmm3 的最大整数值。

如果结果无法用 FP16 表示，则发出适当的溢出响应（对于正缩放操作数）或适当的下溢响应（对于负缩放操作数）。溢出和下溢响应取决于舍入模式（适用于符合 IEEE 标准的舍入），以及 MXCSR 中的其他设置（异常掩码位）和 SAE 位。

特殊情况输入值的处理列在 表 5-41 和 表 5-42 中。

操作¶

def scale_fp16(src1,src2):
    tmp1 := src1
    tmp2 := src2
    return tmp1 * POW(2, FLOOR(tmp2))

VSCALEFPH dest{k1}, src1, src2¶

VL = 128, 256, or 512
KL := VL / 16
IF (VL = 512) AND (EVEX.b = 1) and no memory operand:
    SET_RM(EVEX.RC)
ELSE
    SET_RM(MXCSR.RC)
FOR i := 0 to KL-1:
    IF k1[i] or *no writemask*:
        IF SRC2 is memory and (EVEX.b = 1):
            tsrc := src2.fp16[0]
        ELSE:
            tsrc := src2.fp16[i]
        dest.fp16[i] := scale_fp16(src1.fp16[i],tsrc)
    ELSE IF *zeroing*:
        dest.fp16[i] := 0
    //else dest.fp16[i] remains unchanged
DEST[MAXVL-1:VL] := 0

SIMD 浮点异常¶

无效, 下溢, 溢出, 精度, 渐近下溢。

其他异常¶

EVEX 编码指令，请参阅表 2-46，“第 E2 类异常条件”。

针对 src1 操作数检查并触发渐近下溢操作数异常 (#D)，但不针对 src2 操作数检查。仅在 src2 操作数不是 NaN 的情况下才对 src1 操作数检查渐近下溢操作数异常。如果 src2 操作数是 NaN，则处理器产生 NaN 并且不触发渐近下溢操作数异常，即使 src1 操作数是渐近下溢数。
