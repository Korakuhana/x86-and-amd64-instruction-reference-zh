描述¶

该指令将源操作数（第二操作数）中的 FP16 值转换为具有由 imm8 字节指定的尾数规范化和符号控制的 FP16 值，参见表 5-19。转换后的结果使用写掩码 k1 写入目的操作数（第一操作数）。规范化尾数由 interv（imm8[1:0]）指定，符号控制（SC）由立即数字节的第 3:2 位指定。

目的元素根据写掩码进行更新。

对于每个输入的 FP16 值 x，转换操作为：

GetMant(x) = ±2k|x.significand|

其中：

1 ≤ |x.significand| < 2

无偏指数 k 取决于由 interv 定义的区间范围，以及源的指数是偶数还是奇数。最终结果的符号由符号控制、源符号和前导小数位决定。

imm8[1:0] 的编码值和符号控制如表 5-19 所示。

每个转换后的 FP16 结果都根据符号控制、无偏指数 k（加上偏置）和规范化到由 interv 指定范围的尾数进行编码。

在处理浮点特殊数值时，GetMant() 函数遵循表 5-20。

1. 在 SC[1]==0 的情况下，Getmant(SRC) 的符号根据 SC[0] 声明。

操作¶

def getmant_fp16(src, sign_control, normalization_interval):
    bias := 15
    dst.sign := sign_control[0] ? 0 : src.sign
    signed_one := sign_control[0] ? +1.0 : -1.0
    dst.exp := src.exp
    dst.fraction := src.fraction
    zero := (dst.exp = 0) and (dst.fraction = 0)
    denormal := (dst.exp = 0) and (dst.fraction != 0)
    infinity := (dst.exp = 0x1F) and (dst.fraction = 0)
    nan := (dst.exp = 0x1F) and (dst.fraction != 0)
    src_signaling := src.fraction[9]
    snan := nan and (src_signaling = 0)
    positive := (src.sign = 0)
    negative := (src.sign = 1)
    if nan:
        if snan:
            MXCSR.IE := 1
        return qnan(src)
    if positive and (zero or infinity):
        return 1.0
    if negative:
        if zero:
            return signed_one
        if infinity:
            if sign_control[1]:
                MXCSR.IE := 1
                return QNaN_Indefinite
            return signed_one
        if sign_control[1]:
            MXCSR.IE := 1
            return QNaN_Indefinite
    if denormal:
        jbit := 0
        dst.exp := bias // set exponent to bias value
        while jbit = 0:
            jbit := dst.fraction[9]
            dst.fraction := dst.fraction << 1
            dst.exp : = dst.exp - 1
        MXCSR.DE := 1
    unbaiased_exp := dst.exp - bias
    odd_exp := unbaiased_exp[0]
    signaling_bit := dst.fraction[9]
    if normalization_interval = 0b00:
        dst.exp := bias
    else if normalization_interval = 0b01:
        dst.exp := odd_exp ? bias-1 : bias
    else if normalization_interval = 0b10:
        dst.exp := bias-1
    else if normalization_interval = 0b11:
        dst.exp := signaling_bit ? bias-1 : bias
    return dst

VGETMANTPH dest{k1}, src, imm8¶

VL = 128, 256 or 512
KL := VL/16
sign_control := imm8[3:2]
normalization_interval := imm8[1:0]
FOR i := 0 to KL-1:
    IF k1[i] or *no writemask*:
        IF SRC is memory and (EVEX.b = 1):
            tsrc := src.fp16[0]
        ELSE:
            tsrc := src.fp16[i]
        DEST.fp16[i] := getmant_fp16(tsrc, sign_control, normalization_interval)
    ELSE IF *zeroing*:
        DEST.fp16[i] := 0
    //else DEST.fp16[i] remains unchanged
DEST[MAXVL-1:VL] := 0
