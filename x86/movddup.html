MOVDDUP — 复制双精度浮点值

操作码/指令 Op / En 64/32位模式支持 CPUID 特征标志 描述
F2 0F 12 /r MOVDDUP xmm1, xmm2/m64 A 有效/有效 SSE3 从 xmm2/m64 移动双精度浮点值并复制到 xmm1 中。
VEX.128.F2.0F.WIG 12 /r VMOVDDUP xmm1, xmm2/m64 A 有效/有效 AVX 从 xmm2/m64 移动双精度浮点值并复制到 xmm1 中。
VEX.256.F2.0F.WIG 12 /r VMOVDDUP ymm1, ymm2/m256 A 有效/有效 AVX 从 ymm2/mem 移动偶数索引的双精度浮点值，并将每个元素复制到 ymm1 中。
EVEX.128.F2.0F.W1 12 /r VMOVDDUP xmm1 {k1}{z}, xmm2/m64 B 有效/有效 AVX512VL AVX512F 从 xmm2/m64 移动双精度浮点值并根据写掩码 k1 将每个元素复制到 xmm1 中。
EVEX.256.F2.0F.W1 12 /r VMOVDDUP ymm1 {k1}{z}, ymm2/m256 B 有效/有效 AVX512VL AVX512F 从 ymm2/m256 移动偶数索引的双精度浮点值，并根据写掩码 k1 将每个元素复制到 ymm1 中。
EVEX.512.F2.0F.W1 12 /r VMOVDDUP zmm1 {k1}{z}, zmm2/m512 B 有效/有效 AVX512F 从 zmm2/m512 移动偶数索引的双精度浮点值，并根据写掩码 k1 将每个元素复制到 zmm1 中。

指令操作数编码¶

Op/En 元组类型 操作数 1 操作数 2 操作数 3 操作数 4
A N/A ModRM:reg (w) ModRM:r/m (r) N/A N/A

描述¶

对于 256 位及更高版本：从源操作数（第二个操作数）中复制偶数索引的双精度浮点值到相邻对中，并存储到目的操作数（第一个操作数）中。

对于 128 位版本：从源操作数（第二个操作数）中复制低位双精度浮点值，并存储到目的操作数（第一个操作数）中。

128 位传统 SSE 版本：对应目的寄存器的位 (MAXVL-1:128) 保持不变。源操作数是 XMM 寄存器或 64 位内存位置。

VEX.128 和 EVEX.128 编码版本：目的寄存器的位 (MAXVL-1:128) 被清零。源操作数是 XMM 寄存器或 64 位内存位置。对于 EVEX 版本，目的操作数根据写掩码进行条件更新。

VEX.256 和 EVEX.256 编码版本：目的寄存器的位 (MAXVL-1:256) 被清零。源操作数是 YMM 寄存器或 256 位内存位置。对于 EVEX 版本，目的操作数根据写掩码进行条件更新。

EVEX.512 编码版本：目的操作数根据写掩码进行更新。源操作数是 ZMM 寄存器或 512 位内存位置。

注意：VEX.vvvv 和 EVEX.vvvv 是保留的，必须为 1111b，否则指令将产生 #UD 异常。

操作¶

(KL, VL) = (2, 128), (4, 256), (8, 512)
TMP_SRC[63:0] := SRC[63:0]
TMP_SRC[127:64] := SRC[63:0]
IF VL >= 256
    THEN
        TMP_SRC[191:128] := SRC[191:128]
        TMP_SRC[255:192] := SRC[191:128]
FI;
IF VL >= 512
    THEN
        TMP_SRC[319:256] := SRC[319:256]
        TMP_SRC[383:320] := SRC[319:256]
        TMP_SRC[477:384] := SRC[477:384]
        TMP_SRC[511:484] := SRC[477:384]
FI;
FOR j := 0 TO KL-1
    i := j * 64
    IF k1[j] OR *no writemask*
        THEN DEST[i+63:i] := TMP_SRC[i+63:i]
        ELSE
            IF *merging-masking*
                THEN *DEST[i+63:i] remains unchanged*
                ELSE
                        ; zeroing-masking
                    DEST[i+63:i] := 0
                        ; zeroing-masking
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0
