描述¶

此指令将源操作数（第二个操作数）低阶位中的打包半精度（16 位）浮点值转换为打包单精度浮点值，并将转换后的值写入目的操作数（第一个操作数）。

如果是渐近下溢（反常数）操作数，则返回正确的正常结果。MXCSR.DAZ 被忽略并被视为 0。MXCSR 上不报告渐近下溢异常。

VEX.128 版本：源操作数是 XMM 寄存器或 64 位内存位置。目的操作数是 XMM 寄存器。对应目的寄存器的高位 (MAXVL-1:128) 清零。

VEX.256 版本：源操作数是 XMM 寄存器或 128 位内存位置。目的操作数是 YMM 寄存器。对应目的寄存器的高位 (MAXVL-1:256) 清零。

EVEX 编码版本：源操作数是 YMM/XMM/XMM（低 64 位）寄存器或 256/128/64 位内存位置。目的操作数是根据写掩码 k1 有条件更新的 ZMM/YMM/XMM 寄存器。

下图展示了如何将四个打包半精度（占 64 位）转换为四个单精度（占 128 位）浮点值。

注意：VEX.vvvv 和 EVEX.vvvv 保留（必须为 1111b）。

VCVTPH2PSX 指令是 PH 到 PS 转换指令的新形式，编码在 map 6 中。AVX512F 中存在的该指令的先前版本 VCVTPH2PS（编码在 map 2，0F38 中）不支持嵌入式广播。VCVTPH2PSX 指令提供了嵌入式广播选项。

与 AVX512_FP16 关联的指令始终处理 FP16 反常数输入；反常数输入不被视为零。

VCVTPH2PS（EVEX 编码版本）¶

(KL, VL) = (4, 128), (8, 256), (16, 512)
FOR j := 0 TO KL-1
    i := j * 32
    k := j * 16
    IF k1[j] OR *no writemask*
        THEN DEST[i+31:i] :=
            vCvt_h2s(SRC[k+15:k])
        ELSE
            IF *merging-masking*
                        ; merging-masking
                THEN *DEST[i+31:i] remains unchanged*
                ELSE
                        ; zeroing-masking
                    DEST[i+31:i] := 0
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0

VCVTPH2PSX DEST, SRC¶

VL = 128, 256, or 512
KL := VL/32
FOR j := 0 TO KL-1:
    IF k1[j] OR *no writemask*:
        IF *SRC is memory* and EVEX.b = 1:
            tsrc := SRC.fp16[0]
        ELSE
            tsrc := SRC.fp16[j]
        DEST.fp32[j] := Convert_fp16_to_fp32(tsrc)
    ELSE IF *zeroing*:
        DEST.fp32[j] := 0
    // else dest.fp32[j] remains unchanged
DEST[MAXVL-1:VL] := 0

SIMD 浮点异常¶

VEX 编码指令：无效。

EVEX 编码指令：无效。

带有广播的 EVEX 编码指令（VCVTPH2PSX）：无效、反常数。
