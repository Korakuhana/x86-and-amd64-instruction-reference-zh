VSUBPH — 减法打包 FP16 值

操作码/指令 Op/En 64/32位模式支持 CPUID 功能标志 描述
EVEX.128.NP.MAP5.W0 5C /r VSUBPH xmm1{k1}{z}, xmm2, xmm3/m128/m16bcst A V/V AVX512-FP16 AVX512VL 在写掩码 k1 控制下，从 xmm2 中减去 xmm3/m128/m16bcst 中的打包 FP16 值，并将结果存储在 xmm1 中。
EVEX.256.NP.MAP5.W0 5C /r VSUBPH ymm1{k1}{z}, ymm2, ymm3/m256/m16bcst A V/V AVX512-FP16 AVX512VL 在写掩码 k1 控制下，从 ymm2 中减去 ymm3/m256/m16bcst 中的打包 FP16 值，并将结果存储在 ymm1 中。
EVEX.512.NP.MAP5.W0 5C /r VSUBPH zmm1{k1}{z}, zmm2, zmm3/m512/m16bcst {er} A V/V AVX512-FP16 在写掩码 k1 控制下，从 zmm2 中减去 zmm3/m512/m16bcst 中的打包 FP16 值，并将结果存储在 zmm1 中。

指令操作数编码¶

Op/En 元组 操作数 1 操作数 2 操作数 3 操作数 4
A Full ModRM:reg (w) VEX.vvvv (r) ModRM:r/m (r) N/A

描述¶

此指令从第一源操作数中的对应元素中减去第二源操作数中的打包 FP16 值，并将打包 FP16 结果存储在目的操作数中。在写掩码控制下更新目的元素。

操作¶

VSUBPH (EVEX 编码版本) 当 src2 操作数为寄存器时¶

VL = 128, 256 or 512
KL := VL/16
IF (VL = 512) AND (EVEX.b = 1):
    SET_ROUNDING_MODE_FOR_THIS_INSTRUCTION(EVEX.RC)
ELSE
    SET_ROUNDING_MODE_FOR_THIS_INSTRUCTION(MXCSR.RC)
FOR j := 0 TO KL-1:
    IF k1[j] OR *no writemask*:
        DEST.fp16[j] := SRC1.fp16[j] - SRC2.fp16[j]
    ELSE IF *zeroing*:
        DEST.fp16[j] := 0
    // else dest.fp16[j] remains unchanged
DEST[MAXVL-1:VL] := 0

VSUBPH (EVEX 编码版本) 当 src2 操作数为内存源时¶

VL = 128, 256 or 512
KL := VL/16
FOR j := 0 TO KL-1:
    IF k1[j] OR *no writemask*:
        IF EVEX.b = 1:
            DEST.fp16[j] := SRC1.fp16[j] - SRC2.fp16[0]
        ELSE:
            DEST.fp16[j] := SRC1.fp16[j] - SRC2.fp16[j]
    ELSE IF *zeroing*:
        DEST.fp16[j] := 0
    // else dest.fp16[j] remains unchanged
DEST[MAXVL-1:VL] := 0
