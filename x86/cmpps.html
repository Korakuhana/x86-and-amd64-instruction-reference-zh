描述¶

对第二源操作数和第一源操作数中的打包单精度浮点值进行 SIMD 比较，并将比较结果返回给目的操作数。比较关系谓词操作数（即立即数字节）指定在每对打包值上执行的比较类型。

EVEX 编码版本：第一源操作数（第二操作数）是 ZMM/YMM/XMM 寄存器。第二源操作数可以是 ZMM/YMM/XMM 寄存器、512/256/128 位内存位置，或者从 32 位内存位置广播的 512/256/128 位向量。目的操作数（第一操作数）是 opmask（操作掩码）寄存器。在写掩码 k2 的作用下，比较结果被写入目的操作数。每个比较结果都是单个掩码位，1 表示比较为真，0 表示比较为假。

VEX.256 编码版本：第一源操作数（第二操作数）是 YMM 寄存器。第二源操作数（第三操作数）可以是 YMM 寄存器或 256 位内存位置。目的操作数（第一操作数）是 YMM 寄存器。执行八次比较，并将结果写入目的操作数。每次比较的结果是全 1 的双字掩码（比较为真）或全 0（比较为假）。

128 位传统 SSE 版本：第一源与目的操作数（第一操作数）是 XMM 寄存器。第二源操作数（第二操作数）可以是 XMM 寄存器或 128 位内存位置。对应的 ZMM 目的寄存器的 bits (MAXVL-1:128) 保持不变。执行四次比较，结果写入目的操作数的 bits 127:0。每次比较的结果是全 1 的双字掩码（比较为真）或全 0（比较为假）。

VEX.128 编码版本：第一源操作数（第二操作数）是 XMM 寄存器。第二源操作数（第三操作数）可以是 XMM 寄存器或 128 位内存位置。目的 ZMM 寄存器的 bits (MAXVL-1:128) 清零。执行四次比较，结果写入目的操作数的 bits 127:0。

比较关系谓词操作数是 8 位立即数：

  • 对于使用 VEX 前缀和 EVEX 前缀编码的指令，bits 4:0 定义了要执行的比较类型（参见表 3-1）。立即数的 bits 5 到 7 保留。
  • 对于未使用 VEX 前缀的指令编码，bits 2:0 定义了要进行的比较类型（参见表 3-1 的前 8 行）。立即数的 bits 3 到 7 保留。

当被比较的两个源操作数中至少有一个是 NaN 时，无序（unordered）关系为真；当两个源操作数均不是 NaN 时，有序（ordered）关系为真。

后续使用目的操作数中的掩码结果作为输入操作数的计算指令不会产生异常，因为全 0 掩码对应浮点值 +0.0，而全 1 掩码对应 QNaN。

请注意，具有 “CPUID.1H:ECX.AVX = 0” 的处理器未实现 “大于”、“大于或等于”、“不大于” 和 “不大于或等于” 关系谓词。这些比较可以通过使用相反关系（即使用 “不小于或等于” 来进行 “大于” 比较）或通过软件模拟来完成。使用软件模拟时，程序必须交换操作数（必要时复制寄存器以保护现在将存于目的寄存器中的数据），然后使用不同的谓词执行比较。要用于这些模拟的谓词列在《Intel® 64 和 IA-32 架构软件开发人员手册》第 2A 卷中表 3-7（位于 cli.html 对应章节）的前 8 行中的 Emulation 标题下。

编译器和汇编器除了三操作数 CMPPS 指令之外，对于具有 “CPUID.1H:ECX.AVX = 0” 的处理器，还可以实现以下双操作数伪操作。参见表 3-4。编译器应将保留的 imm8 值视为非法语法。

处理器未实现的大于关系在软件中需要多于一条指令来模拟，因此不应被实现为伪操作。（对于这些关系，程序员应当反转相应小于关系的操作数，并使用移动指令确保掩码被移动到正确的目的寄存器，且源操作数保持完整。）

具有 “CPUID.1H:ECX.AVX = 1” 的处理器实现了表 3-5 中所示的全部 32 个谓词，不再需要软件模拟。编译器和汇编器除了四操作数 VCMPPS 指令之外，还可以实现以下三操作数伪操作。参见表 3-5，其中 reg1 和 reg2 记号表示 XMM 寄存器或 YMM 寄存器。编译器应将保留的 imm8 值视为非法语法。或者，内部函数可以将伪操作映射到预定义常量，以支持更简单的内部函数接口。编译器和汇编器可以类似地通过扩展表 3-5 中列出的语法，为 EVEX 编码的 VCMPPS 指令实现三操作数伪操作。

VCMPPS (EVEX Encoded Versions)¶

(KL, VL) = (4, 128), (8, 256), (16, 512)
FOR j := 0 TO KL-1
    i := j * 32
    IF k2[j] OR *no writemask*
        THEN
            IF (EVEX.b = 1) AND (SRC2 *is memory*)
                THEN
                    CMP := SRC1[i+31:i] OP5 SRC2[31:0]
                ELSE
                    CMP := SRC1[i+31:i] OP5 SRC2[i+31:i]
            FI;
            IF CMP = TRUE
                THEN DEST[j] := 1;
                ELSE DEST[j] := 0; FI;
        ELSE DEST[j] := 0
                        ; zeroing-masking onlyFI;
    FI;
ENDFOR
DEST[MAX_KL-1:KL] := 0

SIMD 浮点异常¶
