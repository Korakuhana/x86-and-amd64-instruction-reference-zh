SHA256MSG1 — 为下四个 SHA256 消息双字执行中间计算

操作码/指令 Op/En 64/32位模式支持 CPUID 特征标志 描述
NP 0F 38 CC /r SHA256MSG1 xmm1, xmm2/m128 RM V/V SHA 使用来自 xmm1 和 xmm2/m128 的前四个 SHA256 消息双字，为接下来的四个 SHA256 消息双字执行中间计算，并将结果存储在 xmm1 中。

指令操作数编码¶

Op/En 操作数 1 操作数 2 操作数 3
RM ModRM:reg (r, w) ModRM:r/m (r) N/A

描述¶

SHA256MSG1 指令是两条 SHA256 消息调度指令之一。该指令为接下来的四个 SHA256 消息双字执行中间计算。

操作¶

SHA256MSG1¶

W4 := SRC2[31: 0] ;
W3 := SRC1[127:96] ;
W2 := SRC1[95:64] ;
W1 := SRC1[63: 32] ;
W0 := SRC1[31: 0] ;
DEST[127:96] := W3 + σ0( W4);
DEST[95:64] := W2 + σ0( W3);
DEST[63:32] := W1 + σ0( W2);
DEST[31:0] := W0 + σ0( W1);
