EMODPR — 限制 EPC 页面的权限

操作码/指令 操作/编码 64/32位模式支持 CPUID功能标志 描述
EAX = 0EH ENCLS[EMODPR] IR V/V SGX2 此叶函数限制已初始化 enclave 中 EPC 页面的关联访问权限。

指令操作数编码¶

操作/编码 EAX RBX RCX
IR EMODPR (输入) 返回错误代码 (输出) SECINFO 的地址 (输入) 目标 EPC 页面的地址 (输入)

描述¶

此叶函数限制已初始化 enclave 中 EPC 页面的关联访问权限。SECINFO 参数的 RWX 位被视为权限掩码；提供未限制页面权限的值将不起作用。此指令只能在当前特权级别为 0 时执行。

RBX 包含 SECINFO 结构的有效地址，而 RCX 包含 EPC 页面的有效地址。下表提供了有关 EMODPR 叶函数内存参数的附加信息。

EMODPR 内存参数语义¶

SECINFO EPCPAGE
允许非 Enclave 进行读访问 允许 Enclave 进行读/写访问

如果发生以下任何情况，指令将产生异常：

EMODPR 异常条件¶

操作数未正确对齐。 如果设置了不受支持的安全属性。
Enclave 未初始化。 SECS 被另一个线程锁定。
EPC 页面被另一个线程锁定。 RCX 未包含正在运行的 enclave 中 EPC 页面的有效地址。
EPC 页面无效。

EMODPR 操作流程中的临时变量¶

名称 类型 大小 (位) 描述
TMP_SECS 有效地址 32/64 EPC 操作数所属 SECS 的物理地址。
SCRATCH_SECINFO SECINFO 512 用于保存 DS:RBX 内容的暂存存储。

IF (DS:RBX is not 64Byte Aligned)

THEN #GP(0); FI;

IF (DS:RCX is not 4KByte Aligned)

THEN #GP(0); FI;

IF (DS:RCX does not resolve within an EPC)

THEN #PF(DS:RCX); FI;

SCRATCH_SECINFO := DS:RBX;

(* Check for misconfigured SECINFO flags*)

IF ( (SCRATCH_SECINFO reserved fields are not zero ) or

(SCRATCH_SECINFO.FLAGS.R is 0 and SCRATCH_SECINFO.FLAGS.W is not 0) )

THEN #GP(0); FI;

(* Check concurrency with SGX1 or SGX2 instructions on the EPC page *)

IF (SGX1 or other SGX2 instructions accessing EPC page)

THEN #GP(0); FI;

IF (EPCM(DS:RCX).VALID is 0 )

THEN #PF(DS:RCX); FI;

(* Check the EPC page for concurrency *)

IF (EPC page in use by another SGX2 instruction)

THEN

RFLAGS.ZF := 1;

RAX := SGX_EPC_PAGE_CONFLICT;

GOTO DONE;

FI;

IF (EPCM(DS:RCX).PENDING is not 0 or (EPCM(DS:RCX).MODIFIED is not 0) )

THEN

RFLAGS.ZF := 1;

RAX := SGX_PAGE_NOT_MODIFIABLE;

GOTO DONE;

FI;

IF (EPCM(DS:RCX).PT is not PT_REG)

THEN #PF(DS:RCX); FI;

TMP_SECS := GET_SECS_ADDRESS

IF (TMP_SECS.ATTRIBUTES.INIT = 0)

THEN #GP(0); FI;

(* Set the PR bit to indicate that permission restriction is in progress *)

EPCM(DS:RCX).PR := 1;

(* Update EPCM permissions *)

EPCM(DS:RCX).R := EPCM(DS:RCX).R & SCRATCH_SECINFO.FLAGS.R;

EPCM(DS:RCX).W := EPCM(DS:RCX).W & SCRATCH_SECINFO.FLAGS.W;

EPCM(DS:RCX).X := EPCM(DS:RCX).X & SCRATCH_SECINFO.FLAGS.X;

RFLAGS.ZF := 0;

RAX := 0;

DONE:

RFLAGS.CF,PF,AF,OF,SF := 0;

影响的标志寄存器¶

如果页面不可修改或有其他 SGX2 指令并发执行，则设置 ZF，否则清除。清除 CF、PF、AF、OF、SF。

保护模式异常¶

#GP(0) 如果内存操作数有效地址超出 DS 段限制。
如果内存操作数未正确对齐。
如果内存操作数被锁定。
#PF(错误代码) 如果访问内存操作数时发生页面错误。
如果内存操作数不是 EPC 页面。

64位模式异常¶

#GP(0) 如果内存操作数为非规范形式。
如果内存操作数未正确对齐。
如果内存操作数被锁定。
#PF(错误代码) 如果访问内存操作数时发生页面错误。
如果内存操作数不是 EPC 页面。
