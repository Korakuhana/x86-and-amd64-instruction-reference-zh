VCVTTPH2W — 使用截断将打包的 FP16 值转换为有符号字整数

描述¶

此指令将源操作数中的打包 FP16 值转换为目的操作数中的有符号字整数。

当转换不精确时，返回截断的值（向零舍入）。如果转换后的结果无法在目的格式中表示，则会引发浮点无效异常；如果该异常被屏蔽，则返回不确定整数值。

目的元素根据写掩码进行更新。

操作¶

VCVTTPH2W dest, src¶

VL = 128, 256 or 512
KL := VL / 16
FOR j := 0 TO KL-1:
    IF k1[j] OR *no writemask*:
        IF *SRC is memory* and EVEX.b = 1:
            tsrc := SRC.fp16[0]
        ELSE
            tsrc := SRC.fp16[j]
        DEST.word[j] := Convert_fp16_to_integer16_truncate(tsrc)
    ELSE IF *zeroing*:
        DEST.word[j] := 0
    // else dest.word[j] remains unchanged
DEST[MAXVL-1:VL] := 0
