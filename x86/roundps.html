ROUNDPS — 对打包单精度浮点值进行舍入

操作码/指令 Op/En 64/32位模式支持 CPUID 特征标志 描述
66 0F 3A 08 /r ib ROUNDPS xmm1, xmm2/m128, imm8 RMI V/V SSE4_1 对 xmm2/m128 中的打包单精度浮点值进行舍入，并将结果保存在 xmm1 中。舍入模式由 imm8 决定。
VEX.128.66.0F3A.WIG 08 /r ib VROUNDPS xmm1, xmm2/m128, imm8 RMI V/V AVX 对 xmm2/m128 中的打包单精度浮点值进行舍入，并将结果保存在 xmm1 中。舍入模式由 imm8 决定。
VEX.256.66.0F3A.WIG 08 /r ib VROUNDPS ymm1, ymm2/m256, imm8 RMI V/V AVX 对 ymm2/m256 中的打包单精度浮点值进行舍入，并将结果保存在 ymm1 中。舍入模式由 imm8 决定。

指令操作数编码¶

Op/En 操作数 1 操作数 2 操作数 3 操作数 4
RMI ModRM:reg (w) ModRM:r/m (r) imm8 N/A

描述¶

对源操作数（第二个操作数）中的 4 个单精度浮点值使用立即数操作数（第三个操作数）中指定的舍入模式进行舍入，并将结果保存在目的操作数（第一个操作数）中。舍入过程将每个输入的浮点值舍入为整数值，并将该整数结果作为单精度浮点值返回。

立即数操作数指定舍入操作的控制字段，定义了三个位字段，如图 4-24 所示（参见 ROUNDPD 页面）。立即数字节的第 3 位控制处理器对精度异常的行为，第 2 位选择舍入模式控制的来源。位 1:0 指定非粘性舍入模式值（表 4-18 列出了舍入模式字段的编码值）。

根据立即数操作数发出精度浮点异常信号。如果任何源操作数是 SNaN，则它将被转换为 QNaN。如果 DAZ 设置为 '1'，则在舍入前反常数将被转换为零。

128位传统 SSE 版本：第二个源可以是 XMM 寄存器或 128 位内存位置。目的寄存器与第一个源 XMM 寄存器不分离，对应 YMM 目的寄存器的位（MAXVL-1:128）保持不变。

VEX.128 编码版本：源操作数为第二个源操作数或 128 位内存位置。目的操作数是 XMM 寄存器。对应 YMM 目的寄存器的位（MAXVL-1:128）清零。

VEX.256 编码版本：源操作数是 YMM 寄存器或 256 位内存位置。目的操作数是 YMM 寄存器。

注意：在 VEX 编码版本中，VEX.vvvv 是保留的，必须为 1111b，否则指令将触发 #UD。

操作¶

IF (imm[2] = ‘1)
    THEN // 舍入模式由 MXCSR.RC 决定
        DEST[31:0] := ConvertSPFPToInteger_M(SRC[31:0]);
        DEST[63:32] := ConvertSPFPToInteger_M(SRC[63:32]);
        DEST[95:64] := ConvertSPFPToInteger_M(SRC[95:64]);
        DEST[127:96] := ConvertSPFPToInteger_M(SRC[127:96]);
    ELSE // 舍入模式由 IMM8.RC 决定
        DEST[31:0] := ConvertSPFPToInteger_Imm(SRC[31:0]);
        DEST[63:32] := ConvertSPFPToInteger_Imm(SRC[63:32]);
        DEST[95:64] := ConvertSPFPToInteger_Imm(SRC[95:64]);
        DEST[127:96] := ConvertSPFPToInteger_Imm(SRC[127:96]);
FI;

DEST[31:0] := RoundToInteger(SRC[31:0], ROUND_CONTROL)

DEST[MAXVL-1:128] (Unmodified)

SIMD 浮点异常¶

无效（仅当 SRC = SNaN 时发出信号）。

精度（仅当 imm[3] = '0 时发出信号；如果 imm[3] = '1，则忽略 MXCSR 中的精度掩码，且不发出精度异常信号。）

请注意，ROUNDPS 不会发出反常数（Denormal）信号。

其他异常¶

请参阅表 2-19，“第 2 类异常条件”，此外：
