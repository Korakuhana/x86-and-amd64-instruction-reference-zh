PSHUFHW — 置换打包高位字

描述¶

从源操作数 128 位通道的高位四字中复制字，并根据立即数操作数选定的位置，将其插入目的操作数对应通道的高位四字字位置。此 256 位操作类似于 256 位 VPSHUFD 指令所使用的通道内操作（如图 4-16 所示）。对于 128 位操作，仅低位 128 位通道起作用。立即数操作数中的每个 2 位字段都选择目的操作数高位四字中的一个字位置的内容。立即数操作数字段的二进制编码选择源操作数高位四字中的字（0、1、2 或 3、4），以便复制到目的操作数。源操作数的低位四字被复制到目的操作数的低位四字中（对于每个 128 位通道）。

注意，此指令允许将源操作数高位四字中的一个字复制到目的操作数高位四字中的多个字位置。

在 64 位模式下且未在 VEX/EVEX 中编码时，使用 REX.R 形式的 REX 前缀允许此指令访问其他寄存器（XMM8-XMM15）。

128位传统 SSE 版本：目的操作数是 XMM 寄存器。源操作数可以是 XMM 寄存器或 128 位内存位置。对应 YMM 目的寄存器的位（MAXVL-1:128）保持不变。

VEX.128 编码版本：目的操作数是 XMM 寄存器。源操作数可以是 XMM 寄存器或 128 位内存位置。目的 YMM 寄存器的位（MAXVL-1:128）清零。VEX.vvvv 是保留的且必须为 1111b，VEX.L 必须为 0，否则指令将产生 #UD。

VEX.256 编码版本：目的操作数是 YMM 寄存器。源操作数可以是 YMM 寄存器或 256 位内存位置。

EVEX 编码版本：目的操作数是 ZMM/YMM/XMM 寄存器。源操作数可以是 ZMM/YMM/XMM 寄存器或 512/256/128 位内存位置。目的操作数根据写掩码更新。

注意：在 VEX 编码版本中，VEX.vvvv 是保留的，必须为 1111b，否则指令将产生 #UD。

VPSHUFHW（EVEX 编码版本）¶

(KL, VL) = (8, 128), (16, 256), (32, 512)
IF VL >= 128
    TMP_DEST[63:0] := SRC1[63:0]
    TMP_DEST[79:64] := (SRC1 >> (imm[1:0] *16))[79:64]
    TMP_DEST[95:80] := (SRC1 >> (imm[3:2] * 16))[79:64]
    TMP_DEST[111:96] := (SRC1 >> (imm[5:4] * 16))[79:64]
    TMP_DEST[127:112] := (SRC1 >> (imm[7:6] * 16))[79:64]
FI;
IF VL >= 256
    TMP_DEST[191:128] := SRC1[191:128]
    TMP_DEST[207:192] := (SRC1 >> (imm[1:0] *16))[207:192]
    TMP_DEST[223:208] := (SRC1 >> (imm[3:2] * 16))[207:192]
    TMP_DEST[239:224] := (SRC1 >> (imm[5:4] * 16))[207:192]
    TMP_DEST[255:240] := (SRC1 >> (imm[7:6] * 16))[207:192]
FI;
IF VL >= 512
    TMP_DEST[319:256] := SRC1[319:256]
    TMP_DEST[335:320] := (SRC1 >> (imm[1:0] *16))[335:320]
    TMP_DEST[351:336] := (SRC1 >> (imm[3:2] * 16))[335:320]
    TMP_DEST[367:352] := (SRC1 >> (imm[5:4] * 16))[335:320]
    TMP_DEST[383:368] := (SRC1 >> (imm[7:6] * 16))[335:320]
    TMP_DEST[447:384] := SRC1[447:384]
    TMP_DEST[463:448] := (SRC1 >> (imm[1:0] *16))[463:448]
    TMP_DEST[479:464] := (SRC1 >> (imm[3:2] * 16))[463:448]
    TMP_DEST[495:480] := (SRC1 >> (imm[5:4] * 16))[463:448]
    TMP_DEST[511:496] := (SRC1 >> (imm[7:6] * 16))[463:448]
FI;
FOR j := 0 TO KL-1
    i := j * 16
    IF k1[j] OR *no writemask*
        THEN DEST[i+15:i] := TMP_DEST[i+15:i];
        ELSE
            IF *merging-masking*
                         ; merging-masking
                THEN *DEST[i+15:i] remains unchanged*
                ELSE *zeroing-masking*
                             ; zeroing-masking
                    DEST[i+15:i] := 0
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0
