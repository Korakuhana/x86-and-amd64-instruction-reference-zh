SHA1RNDS4 — 执行四轮 SHA1 操作

操作码/指令 Op/En 64/32位模式支持 CPUID 特征标志 描述
NP 0F 3A CC /r ib SHA1RNDS4 xmm1, xmm2/m128, imm8 RMI V/V SHA 对来自 xmm1 的 SHA1 状态 (A,B,C,D) 执行四轮 SHA1 操作，使用来自 xmm2/m128 的下 4 轮消息双字和状态变量 E 的预计算和。立即数字节控制逻辑函数和轮常数。

指令操作数编码¶

Op/En 操作数 1 操作数 2 操作数 3
RMI ModRM:reg (r, w) ModRM:r/m (r) imm8

描述¶

SHA1RNDS4 指令执行四轮 SHA1 操作，使用来自第一个操作数（既是源操作数又是目的操作数）的初始 SHA1 状态 (A,B,C,D)，以及来自第二个操作数（源操作数）的下 4 轮消息双字的预计算和与状态变量 E。经过四轮处理后更新的 SHA1 状态 (A,B,C,D) 存储在目的操作数中。

SHA1RNDS4¶

函数 f() 和常数 K 取决于立即数的值。
IF ( imm8[1:0] = 0 )
    THEN f() := f0(),
        K := K0;
ELSE IF ( imm8[1:0]
        = 1 )
    THEN f() := f1(),
        K := K1;
ELSE IF ( imm8[1:0]
        = 2 )
    THEN f() := f2(),
        K := K2;
ELSE IF ( imm8[1:0]
        = 3 )
    THEN f() := f3(),
        K := K3;
FI;

B := SRC1[95:64];

D := SRC1[31:0];
W0E := SRC2[127:96];
W1 := SRC2[95:64];
W2 := SRC2[63:32];

第 i = 0 轮操作:
A_1 := f (B, C, D) + (A ROL 5) +W0E +K;
B_1 := A;
C_1 := B ROL 30;
D_1 := C;
E_1 := D;
FOR i = 1 to 3
    A_(i +1) := f (B_i, C_i, D_i) + (A_i ROL 5) +Wi+ E_i +K;
    B_(i +1) := A_i;
    C_(i +1) := B_i ROL 30;
    D_(i +1) := C_i;
    E_(i +1) := D_i;
ENDFOR
DEST[127:96] := A_4;
DEST[95:64] := B_4;
DEST[63:32] := C_4;
DEST[31:0] := D_4;
