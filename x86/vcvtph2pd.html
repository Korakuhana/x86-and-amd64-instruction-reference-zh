描述¶

此指令将目的寄存器中的打包 FP16 值转换为 FP64 值。目的元素根据写掩码进行更新。

此指令可处理正常及渐近下溢（反常数）的 FP16 输入。

操作¶

VCVTPH2PD DEST, SRC¶

VL = 128, 256, or 512
KL := VL/64
FOR j := 0 TO KL-1:
    IF k1[j] OR *no writemask*:
        IF *SRC is memory* and EVEX.b = 1:
            tsrc := SRC.fp16[0]
        ELSE
            tsrc := SRC.fp16[j]
        DEST.fp64[j] := Convert_fp16_to_fp64(tsrc)
    ELSE IF *zeroing*:
        DEST.fp64[j] := 0
    // else dest.fp64[j] remains unchanged
DEST[MAXVL-1:VL] := 0
