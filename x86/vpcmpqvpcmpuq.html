描述¶

对第二源操作数 and 第一源操作数中的打包整数值进行 SIMD 比较，并将比较结果返回给掩码目的操作数。比较条件操作数（立即数字节）指定对两个源操作数中的每对打包值执行的比较类型。每次比较的结果为单个掩码位，1 表示比较为真，0 表示比较为假。

VPCMPQ/VPCMPUQ 在有符号/无符号四字整数值对之间执行比较。

第一源操作数（第二操作数）是 ZMM/YMM/XMM 寄存器。第二源操作数可以是 ZMM/YMM/XMM 寄存器或 512/256/128 位内存位置，或者是从 64 位内存位置广播的 512 位向量。目的操作数（第一操作数）是掩码寄存器 k1。最多执行 8/4/2 次比较，结果在写掩码 k2 下被写入目的操作数。

比较条件操作数是一个 8 位立即数：第 2:0 位定义要执行的比较类型。立即数的第 3 到 7 位保留。编译器可以实现表 5-21 中列出的伪操作助记符。

操作¶

CASE (COMPARISON PREDICATE) OF
    0: OP := EQ;
    1: OP := LT;
    2: OP := LE;
    3: OP := FALSE;
    4: OP := NEQ;
    5: OP := NLT;
    6: OP := NLE;
    7: OP := TRUE;
ESAC;

VPCMPQ (EVEX 编码版本)¶

(KL, VL) = (2, 128), (4, 256), (8, 512)
FOR j := 0 TO KL-1
    i := j * 64
    IF k2[j] OR *no writemask*
        THEN
            IF (EVEX.b = 1) AND (SRC2 *is memory*)
                THEN CMP := SRC1[i+63:i] OP SRC2[63:0];
                ELSE CMP := SRC1[i+63:i] OP SRC2[i+63:i];
            FI;
            IF CMP = TRUE
                THEN DEST[j] := 1;
                ELSE DEST[j] := 0; FI;
        ELSE DEST[j] := 0
                    ; zeroing-masking only
    FI;
ENDFOR
DEST[MAX_KL-1:KL] := 0
