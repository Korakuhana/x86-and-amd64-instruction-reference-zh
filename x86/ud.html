UD — 未定义指令

操作码 指令 Op/En 64位模式 兼容/传统模式 描述
0F FF /r UD01 r32, r/m32 RM Valid Valid 产生无效操作码异常。
0F B9 /r UD1 r32, r/m32 RM Valid Valid 产生无效操作码异常。
0F 0B UD2 ZO Valid Valid 产生无效操作码异常。

1. 某些处理器解码 UD0 指令时没有 ModR/M 字节。结果是，当带有 ModR/M 字节（及任何隐含字节）的指令越过页面或段边界时，这些处理器将引发无效操作码异常，而不是指令提取故障。

描述¶

生成无效操作码异常。提供此指令用于软件测试以显式生成无效操作码异常。此指令的操作码专门为此目的而保留。

除了引发无效操作码异常之外，此指令对处理器状态或内存没有任何影响。

尽管是执行 UD 指令导致了无效操作码异常，但由于交付该异常而保存的指令指针引用的是 UD 指令（而不是其后续指令）。

此指令在非 64 位模式和 64 位模式下的操作相同。

操作¶

#UD (* 生成无效操作码异常 *);

受影响的标志位¶

无。

异常（所有运行模式）¶

#UD 在所有运行模式下引发无效操作码异常。
