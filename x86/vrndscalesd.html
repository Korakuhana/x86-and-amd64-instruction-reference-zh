VRNDSCALESD — 对标量双精度浮点值进行舍入以保留指定尾数位数

操作码/指令 Op/En 64/32位模式支持 CPUID 功能标志 描述
EVEX.LLIG.66.0F3A.W1 0B /r ib VRNDSCALESD xmm1 {k1}{z}, xmm2, xmm3/m64{sae}, imm8 A V/V AVX512F 在写掩码控制下，将 xmm3/m64 中的标量双精度浮点值舍入到由 imm8 字段指定的尾数位数。将结果存储在 xmm1 寄存器中。

指令操作数编码¶

Op/En 元组类型 操作数 1 操作数 2 操作数 3 操作数 4
A Tuple1 Scalar ModRM:reg (w) EVEX.vvvv (r) ModRM:r/m (r) imm8

描述¶

根据立即数操作数中指定的舍入模式，对第二源操作数（第三操作数）低四字元素中的双精度浮点值进行舍入（参见 图 5-29），并根据写掩码将结果存放在目的操作数（第一操作数）的对应元素中。目的操作数 bits 127:64 处的四字元素从第一源操作数（第二操作数）复制。

目的操作数和第一源操作数是 XMM 寄存器，第二源操作数可以是 XMM 寄存器或内存位置。目的寄存器的 bits MAXVL-1:128 清零。

舍入过程将输入舍入为整数值，加上 imm8[7:4] 指定的保留在结果中的尾数位数，并将结果作为双精度浮点值返回。

需要注意的是，执行此指令时不会引起溢出（尽管源值会被 imm8[7:4] 的值缩放）。

立即数操作数还指定了舍入操作的控制字段，定义了三个位字段，如底部的“图 5-29”所示。立即数字节的第 3 位控制处理器对精度异常的行为，第 2 位选择舍入模式控制的来源。第 1:0 位指定一个非粘性（non-sticky）舍入模式值（下方的立即数控制图表列出了舍入模式字段的编码值）。

根据立即数操作数发出精度浮点异常信号。如果任何源操作数是 SNaN，则它将被转换为 QNaN。如果 DAZ 设置为 1，则在舍入之前，渐近下溢数将被转换为零。

此指令结果的符号会被保留，包括零的符号。

VRNDSCALESD 的运算公式为：

ROUND(x) = 2-M*Round_to_INT(x*2M, round_ctrl),

round_ctrl = imm[3:0];

M=imm[7:4];

x*2M 的运算是在假设指数范围无限的情况下计算的（即永远不会发生溢出）。

VRNDSCALESD 是 VEX 编码的 VROUNDSD 指令的更通用形式。在 VROUNDSD 中，运算公式为：

ROUND(x) = Round_to_INT(x, round_ctrl),

round_ctrl = imm[3:0];

EVEX 编码版本：源操作数是 XMM 寄存器或 64 位内存位置。目的操作数是 XMM 寄存器。

输入值的特殊情况处理列在 表 5-31 中。

操作¶

RoundToIntegerDP(SRC[63:0], imm8[7:0]) {
    if (imm8[2] = 1)
        rounding_direction := MXCSR:RC
                        ; get round control from MXCSR
    else
        rounding_direction := imm8[1:0]
                        ; get round control from imm8[1:0]
    FI
    M := imm8[7:4] ; get the scaling factor
    case (rounding_direction)
    00: TMP[63:0] := round_to_nearest_even_integer(2M*SRC[63:0])
    01: TMP[63:0] := round_to_equal_or_smaller_integer(2M*SRC[63:0])
    10: TMP[63:0] := round_to_equal_or_larger_integer(2M*SRC[63:0])
    11: TMP[63:0] := round_to_nearest_smallest_magnitude_integer(2M*SRC[63:0])
    ESAC
    Dest[63:0] := 2-M* TMP[63:0]
                    ; scale down back to 2-M
    if (imm8[3] = 0) Then ; check SPE
        if (SRC[63:0] != Dest[63:0]) Then
                        ; check precision lost
            set_precision()
                    ; set #PE
        FI;
    FI;
    return(Dest[63:0])
}

SIMD 浮点异常¶

无效, 精度。

如果启用了 SPE，则不报告精度异常（无论 MXCSR 异常掩码如何）。
