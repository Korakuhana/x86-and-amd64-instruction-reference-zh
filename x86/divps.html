DIVPS — 除以打包单精度浮点值

指令/指令格式 操作/编码 64/32位模式支持 CPUID功能标志 描述
NP 0F 5E /r DIVPS xmm1, xmm2/m128 A V/V SSE 将 xmm1 中的打包单精度浮点值除以 xmm2/mem 中的打包单精度浮点值。
VEX.128.0F.WIG 5E /r VDIVPS xmm1, xmm2, xmm3/m128 B V/V AVX 将 xmm2 中的打包单精度浮点值除以 xmm3/mem 中的打包单精度浮点值。
VEX.256.0F.WIG 5E /r VDIVPS ymm1, ymm2, ymm3/m256 B V/V AVX 将 ymm2 中的打包单精度浮点值除以 ymm3/mem 中的打包单精度浮点值。
EVEX.128.0F.W0 5E /r VDIVPS xmm1 {k1}{z}, xmm2, xmm3/m128/m32bcst C V/V AVX512VL AVX512F 根据写掩码 k1，将 xmm2 中的打包单精度浮点值除以 xmm3/m128/m32bcst 中的打包单精度浮点值，并将结果写入 xmm1。
EVEX.256.0F.W0 5E /r VDIVPS ymm1 {k1}{z}, ymm2, ymm3/m256/m32bcst C V/V AVX512VL AVX512F 根据写掩码 k1，将 ymm2 中的打包单精度浮点值除以 ymm3/m256/m32bcst 中的打包单精度浮点值，并将结果写入 ymm1。
EVEX.512.0F.W0 5E /r VDIVPS zmm1 {k1}{z}, zmm2, zmm3/m512/m32bcst{er} C V/V AVX512F 根据写掩码 k1，将 zmm2 中的打包单精度浮点值除以 zmm3/m512/m32bcst 中的打包单精度浮点值，并将结果写入 zmm1。

描述¶

对第一个源操作数（第二个操作数）中的四个、八个或十六个打包单精度浮点值除以第二个源操作数（第三个操作数）中的四个、八个或十六个打包单精度浮点值进行 SIMD 除法。结果写入目标操作数（第一个操作数）。

EVEX 编码版本：第一个源操作数（第二个操作数）是 ZMM/YMM/XMM 寄存器。第二个源操作数可以是 ZMM/YMM/XMM 寄存器、512/256/128 位内存位置或从 32 位内存位置广播的 512/256/128 位向量。目标操作数是 ZMM/YMM/XMM 寄存器，并根据写掩码 k1 条件式更新。

VEX.256 编码版本：第一个源操作数是 YMM 寄存器。第二个源操作数可以是 YMM 寄存器或 256 位内存位置。目标操作数是 YMM 寄存器。

VEX.128 编码版本：第一个源操作数是 XMM 寄存器。第二个源操作数可以是 XMM 寄存器或 128 位内存位置。目标操作数是 XMM 寄存器。对应 ZMM 寄存器目标操作数的高位（MAXVL-1:128）清零。

128 位传统 SSE 版本：第二个源操作数可以是 XMM 寄存器或 128 位内存位置。目标操作数与第一个源操作数 XMM 寄存器相同，对应 ZMM 寄存器目标操作数的高位（MAXVL-1:128）保持不变。

操作¶

VDIVPS (EVEX 编码版本)¶

(KL, VL) = (4, 128), (8, 256), (16, 512)
IF (VL = 512) AND (EVEX.b = 1) AND SRC2 *is a register*
    THEN
        SET_ROUNDING_MODE_FOR_THIS_INSTRUCTION(EVEX.RC);
    ELSE
        SET_ROUNDING_MODE_FOR_THIS_INSTRUCTION(MXCSR.RC);
FI;
FOR j := 0 TO KL-1
    i := j * 32
    IF k1[j] OR *no writemask*
        THEN
            IF (EVEX.b = 1) AND (SRC2 *is memory*)
                THEN
                    DEST[i+31:i] := SRC1[i+31:i] / SRC2[31:0]
                ELSE
                    DEST[i+31:i] := SRC1[i+31:i] / SRC2[i+31:i]
            FI;
        ELSE
            IF *merging-masking* ; merging-masking
                THEN *DEST[i+31:i] remains unchanged*
                ELSE ; zeroing-masking
                    DEST[i+31:i] := 0
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0
