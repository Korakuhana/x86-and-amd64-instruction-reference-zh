VRSQRT28PS — 相对误差小于 2^-28 的打包单精度浮点值平方根倒数近似

操作码/指令 Op/En 64/32位模式支持 CPUID 功能标志 描述
EVEX.512.66.0F38.W0 CC /r VRSQRT28PS zmm1 {k1}{z}, zmm2/m512/m32bcst {sae} A V/V AVX512ER 在写掩码 k1 控制下，计算 zmm2/m512/m32bcst 中打包单精度浮点值平方根的近似倒数（相对误差 < 2-28），并将结果存储在 zmm1 中。

指令操作数编码¶

Op/En 元组类型 操作数 1 操作数 2 操作数 3 操作数 4
A Full ModRM:reg (w) ModRM:r/m (r) N/A N/A

描述¶

计算源操作数（第二操作数）中 float32 值的平方根倒数近似值，并将结果存储在目的操作数（第一操作数）中。在最终舍入之前，近似倒数估算的最大相对误差小于 2-28。最终结果在写入目的操作数之前被舍入至相对误差 < 2-23。

如果任何源元素是 NaN，则为该元素返回静默化 (quietized) 的 NaN 源值。负（非零）源数以及 -∞，都会返回规范 (canonical) NaN 并设置无效标志 (#I)。

-0 的值必须返回 -∞ 并置位被零除标志 (#Z)。负数应返回 NaN 并置位无效标志 (#I)。但请注意，该指令会将输入渐近下溢数刷新为具有相同符号的零，因此负渐近下溢数返回 -∞ 并置位被零除标志。

源操作数是 ZMM 寄存器、512 位内存位置，或从 32 位内存位置广播的 512 位向量。目的操作数是 ZMM 寄存器，在写掩码 k1 控制下有条件地更新。

EVEX.vvvv 是保留的，必须为 1111b，否则指令将引发 #UD。

操作¶

VRSQRT28PS (EVEX 编码版本)¶

(KL, VL) = (16, 512)
FOR j := 0 TO KL-1
    i := j * 32
    IF k1[j] OR *no writemask* THEN
            IF (EVEX.b = 1) AND (SRC *is memory*)
                THEN DEST[i+31:i] := (1.0/ SQRT(SRC[31:0]));
                ELSE DEST[i+31:i] := (1.0/ SQRT(SRC[i+31:i]));
            FI;
    ELSE
        IF *merging-masking* ; merging-masking
            THEN *DEST[i+31:i] remains unchanged*
            ELSE ; zeroing-masking
                DEST[i+31:i] := 0
        FI;
    FI;
ENDFOR;

SIMD 浮点异常¶

无效（如果输入为 SNaN），被零除。
