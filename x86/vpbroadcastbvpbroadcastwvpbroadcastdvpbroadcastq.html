描述¶

使用写掩码 k1 将通用寄存器（第二操作数）中的 8 位、16 位、32 位或 64 位值广播到目的向量寄存器（第一操作数）的所有位置。

EVEX.vvvv 是保留的，必须为 1111b，否则指令会导致 #UD。

操作¶

VPBROADCASTB (EVEX 编码版本)¶

(KL, VL) = (16, 128), (32, 256), (64, 512)
FOR j := 0 TO KL-1
    i := j * 8
    IF k1[j] OR *no writemask*
        THEN DEST[i+7:i] := SRC[7:0]
        ELSE
            IF *merging-masking*
                        ; merging-masking
                THEN *DEST[i+7:i] remains unchanged*
                ELSE
                        ; zeroing-masking
                    DEST[i+7:i] := 0
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0
