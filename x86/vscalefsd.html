VSCALEFSD — 用双精度浮点值缩放标量双精度浮点值

操作码/指令 Op/En 64/32位模式支持 CPUID 功能标志 描述
EVEX.LLIG.66.0F38.W1 2D /r VSCALEFSD xmm1 {k1}{z}, xmm2, xmm3/m64{er} A V/V AVX512F 在写掩码 k1 控制下，使用来自 xmm3/m64 的值缩放 xmm2 中的标量双精度浮点值。

指令操作数编码¶

Op/En 元组类型 操作数 1 操作数 2 操作数 3 操作数 4
A Tuple1 Scalar ModRM:reg (w) EVEX.vvvv (r) ModRM:r/m (r) N/A

描述¶

对第一源操作数中的标量双精度浮点值进行浮点缩放：将其乘以 2 的“第二源操作数中双精度浮点值”次幂。

此运算的方程式为：

xmm1 := xmm2 * 2floor(xmm3)

Floor(xmm3) 表示 ≤ xmm3 的最大整数值。

如果结果无法用双精度表示，则发出适当的溢出响应（对于正缩放操作数）或适当的下溢响应（对于负缩放操作数）。溢出和下溢响应取决于舍入模式（适用于符合 IEEE 标准的舍入），以及 MXCSR 中的其他设置（异常掩码位、FTZ 位）和 SAE 位。

EVEX 编码版本：第一源操作数是 XMM 寄存器。第二源操作数是 XMM 寄存器或内存位置。目的操作数是根据写掩码 k1 有条件更新的 XMM 寄存器。

特殊情况输入值的处理列在 表 5-39 和 表 5-40 中。

操作¶

SCALE(SRC1, SRC2)
{
    ; Check for denormal operands
TMP_SRC2 := SRC2
TMP_SRC1 := SRC1
IF (SRC2 is denormal AND MXCSR.DAZ) THEN TMP_SRC2=0
IF (SRC1 is denormal AND MXCSR.DAZ) THEN TMP_SRC1=0
/* SRC2 is a 64 bits floating-point value */
DEST[63:0] := TMP_SRC1[63:0] * POW(2, Floor(TMP_SRC2[63:0]))
}

VSCALEFSD (EVEX 编码版本)¶

IF (EVEX.b= 1) and SRC2 *is a register*
    THEN
        SET_ROUNDING_MODE_FOR_THIS_INSTRUCTION(EVEX.RC);
    ELSE
        SET_ROUNDING_MODE_FOR_THIS_INSTRUCTION(MXCSR.RC);
FI;
IF k1[0] OR *no writemask*
    THEN DEST[63:0] := SCALE(SRC1[63:0], SRC2[63:0])
    ELSE
        IF *merging-masking* ; merging-masking
            THEN *DEST[63:0] remains unchanged*
            ELSE ; zeroing-masking
                DEST[63:0] := 0
        FI
FI;
DEST[127:64] := SRC1[127:64]
DEST[MAXVL-1:128] := 0

SIMD 浮点异常¶

溢出, 下溢, 无效, 精度, 渐近下溢（针对 Src1）。

Src2 不报告渐近下溢异常。
