BLENDVPD — 可变混合打包双精度浮点值

操作码/指令 Op/En 64/32位模式 CPUID 特征标志 描述
66 0F 38 15 /r BLENDVPD xmm1, xmm2/m128 , <XMM0> RM0 V/V SSE4_1 根据 XMM0 中指定的掩码，从 xmm1 和 xmm2 中选择打包双精度浮点值，并将结果存入 xmm1。
VEX.128.66.0F3A.W0 4B /r /is4 VBLENDVPD xmm1, xmm2, xmm3/m128, xmm4 RVMR V/V AVX 根据掩码操作数 xmm4 中的掩码位，条件性地将双精度浮点值从 xmm2 或 xmm3/m128 复制到 xmm1。
VEX.256.66.0F3A.W0 4B /r /is4 VBLENDVPD ymm1, ymm2, ymm3/m256, ymm4 RVMR V/V AVX 根据掩码操作数 ymm4 中的掩码位，条件性地将双精度浮点值从 ymm2 或 ymm3/m256 复制到 ymm1。

描述¶

根据掩码寄存器操作数中定义的掩码位，条件性地从第二源操作数和第一源操作数复制双精度浮点值的每个四字（quadword）数据元素。掩码位是掩码寄存器中每个四字元素的最高有效位。

目的操作数的每个四字元素复制自：

  • 如果掩码位为 “1”，则为第二源操作数中对应的四字元素；或者
  • 如果掩码位为 “0”，则为第一源操作数中对应的四字元素。

针对 BLENDVPD 的隐式掩码操作数的寄存器分配被定义为架构寄存器 XMM0。

128 位传统 SSE 版本：第一源操作数与目的操作数相同。对应 YMM 目的寄存器的位（MAXVL-1:128）保持不变。掩码寄存器操作数隐式定义为架构寄存器 XMM0。尝试使用 VEX 前缀执行 BLENDVPD 将导致 #UD。

VEX.128 编码版本：第一源操作数和目的操作数是 XMM 寄存器。第二源操作数是 XMM 寄存器或 128 位内存位置。掩码操作数是第三源寄存器，并编码在立即数字节（imm8）的 [7:4] 位中。imm8 的 [3:0] 位被忽略。在 32 位模式下，imm8[7] 被忽略。对应 YMM 寄存器（目的寄存器）的高位（MAXVL-1:128）清零。VEX.W 必须为 0，否则指令将导致 #UD。

VEX.256 编码版本：第一源操作数和目的操作数是 YMM 寄存器。第二源操作数可以是 YMM 寄存器或 256 位内存位置。掩码操作数是第三源寄存器，并编码在立即数字节（imm8）的 [7:4] 位中。imm8 的 [3:0] 位被忽略。在 32 位模式下，imm8[7] 被忽略。VEX.W 必须为 0，否则指令将导致 #UD。

VBLENDVPD 允许掩码使用任意的 XMM 或 YMM 寄存器。相比之下，BLENDVPD 隐式将 XMM0 作为掩码，且不支持非破坏性的目标操作数操作。

操作¶

BLENDVPD (128-bit Legacy SSE Version)¶

MASK := XMM0
IF (MASK[63] = 0) THEN DEST[63:0] := DEST[63:0]
    ELSE DEST [63:0] := SRC[63:0] FI
IF (MASK[127] = 0) THEN DEST[127:64] := DEST[127:64]
    ELSE DEST [127:64] := SRC[127:64] FI
DEST[MAXVL-1:128] (Unmodified)

VBLENDVPD (VEX.128 Encoded Version)¶

MASK := SRC3
IF (MASK[63] = 0) THEN DEST[63:0] := SRC1[63:0]
    ELSE DEST [63:0] := SRC2[63:0] FI
IF (MASK[127] = 0) THEN DEST[127:64] := SRC1[127:64]
    ELSE DEST [127:64] := SRC2[127:64] FI
DEST[MAXVL-1:128] := 0

VBLENDVPD (VEX.256 Encoded Version)¶

MASK := SRC3
IF (MASK[63] = 0) THEN DEST[63:0] := SRC1[63:0]
    ELSE DEST [63:0] := SRC2[63:0] FI
IF (MASK[127] = 0) THEN DEST[127:64] := SRC1[127:64]
    ELSE DEST [127:64] := SRC2[127:64] FI
IF (MASK[191] = 0) THEN DEST[191:128] := SRC1[191:128]
    ELSE DEST [191:128] := SRC2[191:128] FI
IF (MASK[255] = 0) THEN DEST[255:192] := SRC1[255:192]
    ELSE DEST [255:192] := SRC2[255:192] FI
