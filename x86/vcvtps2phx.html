VCVTPS2PHX — 将打包的单精度浮点值转换为打包的 FP16 值

操作码/指令 Op/En 64/32位模式支持 CPUID 特征标志 描述

描述¶

此指令将源操作数中的打包单精度浮点值转换为 FP16 值，并存储到目的操作数中。

VCVTPS2PHX 指令支持广播。

此指令使用 MXCSR.DAZ 处理 FP32 输入。FP16 输出可以是正常数或反常数，并且不会根据 MXCSR 设置有条件地刷新。

操作¶

VCVTPS2PHX DEST, SRC（具有广播支持的 AVX512_FP16 加载版本）¶

VL = 128, 256, or 512
KL := VL / 32
IF *SRC is a register* and (VL == 512) and (EVEX.b = 1):
    SET_RM(EVEX.RC)
ELSE:
    SET_RM(MXCSR.RC)
FOR j := 0 TO KL-1:
    IF k1[j] OR *no writemask*:
        IF *SRC is memory* and EVEX.b = 1:
            tsrc := SRC.fp32[0]
        ELSE
            tsrc := SRC.fp32[j]
        DEST.fp16[j] := Convert_fp32_to_fp16(tsrc)
    ELSE IF *zeroing*:
        DEST.fp16[j] := 0
    // else dest.fp16[j] remains unchanged
DEST[MAXVL-1:VL/2] := 0

无。

SIMD 浮点异常¶

无效、欠溢、溢出、精度、反常数（如果 MXCSR.DAZ=0）。
