FDECSTP — 递减栈顶指针

操作码 模式 传统模式 描述
D9 F6 递减 FPU 状态字中的 TOP 字段。

描述¶

从 FPU 状态字的 TOP 字段中减去 1（递减栈顶指针）。如果 TOP 字段包含 0，则将其设置为 7。该指令的作用是将栈循环旋转一个位置。FPU 数据寄存器和标记（tag）寄存器的内容不受影响。

此指令的操作在非 64 位模式和 64 位模式下相同。

操作¶

IF TOP = 0
	THEN TOP := 7;
	ELSE TOP := TOP – 1;
FI;

影响的 FPU 标志寄存器¶

C1 标志设置为 0。C0、C2 和 C3 标志未定义。

浮点异常¶

无。

保护模式异常¶

#NM CR0.EM[bit 2] 或 CR0.TS[bit 3] = 1。
#MF 如果有挂起的 x87 FPU 异常。
#UD 如果使用了 LOCK 前缀。

实地址模式异常¶

与保护模式下的异常相同。

虚拟-8086 模式异常¶

与保护模式下的异常相同。

兼容模式异常¶

与保护模式下的异常相同。

64位模式异常¶

与保护模式下的异常相同。
