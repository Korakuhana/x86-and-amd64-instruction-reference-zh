FXCH — 交换寄存器内容

操作码 指令 64位模式 兼容/传统模式 描述
D9 C8+i FXCH ST(i) Valid Valid 交换 ST(0) 和 ST(i) 的内容。
D9 C9 FXCH Valid Valid 交换 ST(0) 和 ST(1) 的内容。

描述¶

交换寄存器 ST(0) 和 ST(i) 的内容。如果未指定源操作数，则交换 ST(0) 和 ST(1) 的内容。

该指令提供了一种将 FPU 寄存器栈中的值移动到栈顶 [ST(0)] 的简单方法，以便那些只能对 ST(0) 中的值进行操作的浮点指令可以对它们进行操作。例如，以下指令序列对寄存器栈顶部的第三个寄存器进行平方根操作：

FXCH ST(3);

FSQRT;

FXCH ST(3);

此指令的操作在非 64 位模式和 64 位模式下相同。

操作¶

IF (Number-of-operands) is 1
    THEN
        temp := ST(0);
        ST(0) := SRC;
        SRC := temp;
    ELSE
        temp := ST(0);
        ST(0) := ST(1);
        ST(1) := temp;
FI;

受影响的 FPU 标志位¶

C1 设置为 0。
C0, C2, C3 未定义。

浮点异常¶

#IS 发生栈下溢。

保护模式异常¶

#NM CR0.EM[bit 2] 或 CR0.TS[bit 3] = 1。
#MF 如果存在待处理的 x87 FPU 异常。
#UD 如果使用了 LOCK 前缀。

实地址模式异常¶

与保护模式下的异常相同。

虚拟-8086 模式异常¶

与保护模式下的异常相同。

兼容模式异常¶

与保护模式下的异常相同。

64位模式异常¶

与保护模式下的异常相同。
