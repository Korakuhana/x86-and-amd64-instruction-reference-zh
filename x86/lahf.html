LAHF — 将状态标志位加载到 AH 寄存器

操作码 指令 Op/En 64位模式 兼容/传统模式 描述
9F LAHF ZO Valid1 Valid 加载：AH := EFLAGS(SF:ZF:0:AF:0:PF:1:CF)。

1. 在特定步进中有效；参见描述部分。

指令操作数编码¶

Op/En 操作数 1 操作数 2 操作数 3 操作数 4
ZO N/A N/A N/A N/A

描述¶

此指令在兼容模式和传统模式下如上所述执行。仅当 CPUID.80000001H:ECX.LAHF-SAHF[bit 0] = 1 时，它在 64 位模式下才有效。

操作¶

IF 64-Bit Mode
    THEN
        IF CPUID.80000001H:ECX.LAHF-SAHF[bit 0] = 1;
            THEN AH := RFLAGS(SF:ZF:0:AF:0:PF:1:CF);
            ELSE #UD;
        FI;
    ELSE
        AH := EFLAGS(SF:ZF:0:AF:0:PF:1:CF);
FI;

影响的标志位¶

无。EFLAGS 寄存器中标志位的状态不受影响。

保护模式异常¶

#UD 如果使用了 LOCK 前缀。

实地址模式异常¶

与保护模式下的异常相同。

虚拟 8086 模式异常¶

与保护模式下的异常相同。

兼容模式异常¶

与保护模式下的异常相同。

64位模式异常¶

#UD 如果 CPUID.80000001H:ECX.LAHF-SAHF[bit 0] = 0。
如果使用了 LOCK 前缀。
