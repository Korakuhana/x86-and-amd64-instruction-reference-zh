VCVTUDQ2PH — 将打包的无符号双字整数转换为打包的 FP16 值

操作码/指令 Op/En 64/32位模式支持 CPUID 特征标志 描述
EVEX.128.F2.MAP5.W0 7A /r VCVTUDQ2PH xmm1{k1}{z}, xmm2/m128/m32bcst A V/V AVX512-FP16 AVX512VL 在写掩码 k1 的控制下，将 xmm2/m128/m32bcst 中的四个打包无符号双字整数转换为打包 FP16 值，并将结果存储在 xmm1 中。
EVEX.256.F2.MAP5.W0 7A /r VCVTUDQ2PH xmm1{k1}{z}, ymm2/m256/m32bcst A V/V AVX512-FP16 AVX512VL 在写掩码 k1 的控制下，将 ymm2/m256/m32bcst 中的八个打包无符号双字整数转换为打包 FP16 值，并将结果存储在 xmm1 中。
EVEX.512.F2.MAP5.W0 7A /r VCVTUDQ2PH ymm1{k1}{z}, zmm2/m512/m32bcst {er} A V/V AVX512-FP16 在写掩码 k1 的控制下，将 zmm2/m512/m32bcst 中的十六个打包无符号双字整数转换为打包 FP16 值，并将结果存储在 ymm1 中。

指令操作数编码¶

A Full ModRM:reg (w) ModRM:r/m (r) N/A N/A

描述¶

此指令将源操作数中的打包无符号双字整数转换为目的操作数中的打包 FP16 值。目的元素根据写掩码进行更新。

EVEX.vvvv 保留且必须为 1111b，否则指令会导致 #UD。

如果转换操作的结果溢出且 MXCSR.OM=0，则将引发 OE=1, PE=1 的 SIMD 异常。

操作¶

VCVTUDQ2PH dest, src¶

VL = 128, 256 or 512
KL := VL / 32
IF *SRC is a register* and (VL = 512) AND (EVEX.b = 1):
    SET_RM(EVEX.RC)
ELSE:
    SET_RM(MXCSR.RC)
FOR j := 0 TO KL-1:
    IF k1[j] OR *no writemask*:
        IF *SRC is memory* and EVEX.b = 1:
            tsrc := SRC.dword[0]
        ELSE
            tsrc := SRC.dword[j]
        DEST.fp16[j] := Convert_unsigned_integer32_to_fp16(tsrc)
    ELSE IF *zeroing*:
        DEST.fp16[j] := 0
    // else dest.fp16[j] remains unchanged
DEST[MAXVL-1:VL/2] := 0
