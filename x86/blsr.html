BLSR — 复位最低设置位

操作码/指令 Op/En 64/32位模式 CPUID 功能标志 描述
VEX.LZ.0F38.W0 F3 /1 BLSR r32, r/m32 VM V/V BMI1 复位 r/m32 的最低设置位，保留 r/m32 的所有其他位并将结果写入 r32。
VEX.LZ.0F38.W1 F3 /1 BLSR r64, r/m64 VM V/N.E. BMI1 复位 r/m64 的最低设置位，保留 r/m64 的所有其他位并将结果写入 r64。

指令操作数编码¶

Op/En 操作数 1 操作数 2 操作数 3 操作数 4
VM VEX.vvvv (w) ModRM:r/m (r) N/A N/A

描述¶

将所有位从源操作数复制到目的操作数，并将目的操作数中对应于源操作数最低设置位的位置复位（=0）。如果源操作数为零，则 BLSR 设置 CF 标志。

在实模式和虚拟 8086 模式下不支持此指令。如果不在 64 位模式下，操作数大小始终为 32 位。在 64 位模式下，操作数大小为 64 位时需要 VEX.W1。在非 64 位模式下，VEX.W1 被忽略。尝试在 VEX.L 不等于 0 的情况下执行此指令将导致 #UD。

操作¶

temp := (SRC-1) bitwiseAND ( SRC );
SF := temp[OperandSize -1];
ZF := (temp = 0);
IF SRC = 0
    CF := 1;
ELSE
    CF := 0;
FI
DEST := temp;

受影响的标志位¶

ZF 和 SF 标志根据结果更新。如果源操作数为零，则设置 CF。OF 标志位被清除。AF 和 PF 标志位是未定义的。

BLSR unsigned __int64 _blsr_u64(unsigned __int64 src);

无。
