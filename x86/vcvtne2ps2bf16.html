VCVTNE2PS2BF16 — 将两组打包单精度数据转换为一组打包 BF16 数据

操作码/指令 Op/En 64/32位模式支持 CPUID 特征标志 描述

描述¶

将两个包含打包单精度数据的 SIMD 寄存器转换为一个包含打包 BF16 数据的寄存器。

此指令不支持内存故障抑制。

此指令使用“舍入到最接近的偶数 (Round to nearest (even))”舍入模式。输出的渐近下溢（反常数）总是被清零（flushed to zero），输入的渐近下溢总是被视为零。不参考也不更新 MXCSR。不产生浮点异常。

操作¶

VCVTNE2PS2BF16 dest, src1, src2¶

VL = (128, 256, 512)
KL = VL/16
origdest := dest
FOR i := 0 to KL-1:
    IF k1[ i ] or *no writemask*:
        IF i < KL/2:
            IF src2 is memory and evex.b == 1:
                t := src2.fp32[0]
            ELSE:
                t := src2.fp32[ i ]
        ELSE:
            t := src1.fp32[ i-KL/2]
        // 有关转换辅助函数的定义，请参阅 VCVTNEPS2BF16
        dest.word[i] := convert_fp32_to_bfloat16(t)
    ELSE IF *zeroing*:
        dest.word[ i ] := 0
    ELSE: // 合并掩码，目的元素保持不变
        dest.word[ i ] := origdest.word[ i ]
DEST[MAXVL-1:VL] := 0
