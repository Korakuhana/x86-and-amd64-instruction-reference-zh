RSQRTPS — 计算打包单精度浮点值平方根的近似倒数

操作码/指令 Op/En 64/32位模式支持 CPUID 特征标志 描述
NP 0F 52 /r RSQRTPS xmm1, xmm2/m128 RM V/V SSE 计算 xmm2/m128 中打包单精度浮点值平方根的近似倒数，并将结果存储在 xmm1 中。
VEX.128.0F.WIG 52 /r VRSQRTPS xmm1, xmm2/m128 RM V/V AVX 计算 xmm2/m128 中打包单精度浮点值平方根的近似倒数，并将结果存储在 xmm1 中。
VEX.256.0F.WIG 52 /r VRSQRTPS ymm1, ymm2/m256 RM V/V AVX 计算 ymm2/m256 中打包单精度浮点值平方根的近似倒数，并将结果存储在 ymm1 中。

指令操作数编码¶

Op/En 操作数 1 操作数 2 操作数 3 操作数 4
RM ModRM:reg (w) ModRM:r/m (r) N/A N/A

描述¶

对源操作数（第二个操作数）中的四个打包单精度浮点值进行平方根近似倒数的 SIMD 计算，并将打包单精度浮点结果存储在目的操作数中。源操作数可以是 XMM 寄存器或 128 位内存位置。目的操作数是 XMM 寄存器。有关 SIMD 单精度浮点操作 of 图示，请参阅《Intel® 64 和 IA-32 架构软件开发人员手册》第 1 卷中的图 10-5。

此近似值的相对误差为：

|相对误差| ≤ 1.5 ∗ 2−12

RSQRTPS 指令不受 MXCSR 寄存器中舍入控制位的影响。当源值为 0.0 时，返回与源值符号相同的 ∞。反常源值被视为 0.0（具有相同的符号）。当源值为负值（除 −0.0 之外）时，返回浮点不确定值。当源值是 SNaN 或 QNaN 时，SNaN 被转换为 QNaN，或者返回源 QNaN。

在 64 位模式下，使用 REX.R 形式的 REX 前缀允许此指令访问其他寄存器（XMM8-XMM15）。

128位传统 SSE 版本：第二个源可以是 XMM 寄存器或 128 位内存位置。目的寄存器与第一个源 XMM 寄存器不分离，对应 YMM 目的寄存器的位（MAXVL-1:128）保持不变。

VEX.128 编码版本：第一个源操作数是 XMM 寄存器或 128 位内存位置。目的操作数是 XMM 寄存器。对应 YMM 目的寄存器的位（MAXVL-1:128）清零。

VEX.256 编码版本：第一个源操作数是 YMM 寄存器。第二个源操作数可以是 YMM 寄存器或 256 位内存位置。目的操作数是 YMM 寄存器。

注意：在 VEX 编码版本中，VEX.vvvv 是保留的，必须为 1111b，否则指令将触发 #UD。

操作¶

RSQRTPS（128位传统 SSE 版本）¶

DEST[31:0] := APPROXIMATE(1/SQRT(SRC[31:0]))
DEST[63:32] := APPROXIMATE(1/SQRT(SRC1[63:32]))
DEST[95:64] := APPROXIMATE(1/SQRT(SRC1[95:64]))
DEST[127:96] := APPROXIMATE(1/SQRT(SRC2[127:96]))
DEST[MAXVL-1:128] (Unmodified)

VRSQRTPS（VEX.128 编码版本）¶

DEST[31:0] := APPROXIMATE(1/SQRT(SRC[31:0]))
DEST[63:32] := APPROXIMATE(1/SQRT(SRC1[63:32]))
DEST[95:64] := APPROXIMATE(1/SQRT(SRC1[95:64]))
DEST[127:96] := APPROXIMATE(1/SQRT(SRC2[127:96]))
DEST[MAXVL-1:128] := 0

VRSQRTPS（VEX.256 编码版本）¶

DEST[31:0] := APPROXIMATE(1/SQRT(SRC[31:0]))
DEST[63:32] := APPROXIMATE(1/SQRT(SRC1[63:32]))
DEST[95:64] := APPROXIMATE(1/SQRT(SRC1[95:64]))
DEST[127:96] := APPROXIMATE(1/SQRT(SRC2[127:96]))
DEST[159:128] := APPROXIMATE(1/SQRT(SRC2[159:128]))
DEST[191:160] := APPROXIMATE(1/SQRT(SRC2[191:160]))
DEST[223:192] := APPROXIMATE(1/SQRT(SRC2[223:192]))
DEST[255:224] := APPROXIMATE(1/SQRT(SRC2[255:224]))

SIMD 浮点异常¶

无。

其他异常¶

请参阅表 2-21，“第 4 类异常条件”，此外：
