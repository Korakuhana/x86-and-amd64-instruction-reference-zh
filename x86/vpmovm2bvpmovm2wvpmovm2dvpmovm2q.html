指令操作数编码¶

Op/En 操作数 1 操作数 2 操作数 3 操作数 4
RM ModRM:reg (w) ModRM:r/m (r) N/A N/A

描述¶

将掩码寄存器转换为向量寄存器。根据源掩码寄存器中对应位的值，目的寄存器中的每个元素被设置为全 1 或全 0。

源操作数是掩码寄存器。目的操作数是 ZMM/YMM/XMM 寄存器。

EVEX.vvvv 保留且必须为 1111b，否则指令将导致 #UD。

操作¶

VPMOVM2B (EVEX 编码版本)¶

(KL, VL) = (16, 128), (32, 256), (64, 512)
FOR j := 0 TO KL-1
    i := j * 8
    IF SRC[j]
        THEN DEST[i+7:i] := -1
        ELSE DEST[i+7:i] := 0
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0
