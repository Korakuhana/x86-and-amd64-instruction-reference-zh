描述¶

将源操作数（第二操作数）中的双精度浮点值转换为具有由 imm8 字节指定的尾数规范化和符号控制的双精度浮点值，参见图 5-15。转换后的结果使用写掩码 k1 写入目的操作数（第一操作数）。规范化尾数由 interv（imm8[1:0]）指定，符号控制（sc）由立即数字节的第 3:2 位指定。

目的操作数是根据写掩码更新的 ZMM/YMM/XMM 寄存器。源操作数可以是 ZMM/YMM/XMM 寄存器、512/256/128 位内存位置，或从 64 位内存位置广播的 512/256/128 位向量。

对于每个输入的双精度浮点值 x，转换操作为：

GetMant(x) = ±2k|x.significand|

其中：

1 <= |x.significand| < 2

无偏指数 k 可以是 0 或 -1，具体取决于由 interv 定义的区间范围、尾数（有效数字）的范围以及源的指数是偶数还是奇数。最终结果的符号由 sc 和源符号决定。imm8[1:0] 的编码值和符号控制如图 5-15 所示。

每个转换后的双精度浮点结果都根据符号控制、无偏指数 k（加上偏置）和规范化到由 interv 指定范围的尾数进行编码。

在处理浮点特殊数值时，GetMant() 函数遵循表 5-18。

此指令是写掩码的，因此只有在向量掩码寄存器 k1 中设置了相应位的元素才会被计算并存储到目的操作数中。zmm1 中在 k1 中清零的对应位元素保留其先前值。

注意：EVEX.vvvv 是保留的，必须为 1111b；否则指令将产生 #UD。

1. 在 SC[1]==0 的情况下，Getmant(SRC) 的符号根据 SC[0] 声明。

操作¶

def getmant_fp64(src, sign_control, normalization_interval):
    bias := 1023
    dst.sign := sign_control[0] ? 0 : src.sign
    signed_one := sign_control[0] ? +1.0 : -1.0
    dst.exp := src.exp
    dst.fraction := src.fraction
    zero := (dst.exp = 0) and ((dst.fraction = 0) or (MXCSR.DAZ=1))
    denormal := (dst.exp = 0) and (dst.fraction != 0) and (MXCSR.DAZ=0)
    infinity := (dst.exp = 0x7FF) and (dst.fraction = 0)
    nan := (dst.exp = 0x7FF) and (dst.fraction != 0)
    src_signaling := src.fraction[51]
    snan := nan and (src_signaling = 0)
    positive := (src.sign = 0)
    negative := (src.sign = 1)
    if nan:
        if snan:
            MXCSR.IE := 1
        return qnan(src)
    if positive and (zero or infinity):
        return 1.0
    if negative:
        if zero:
            return signed_one
        if infinity:
            if sign_control[1]:
                MXCSR.IE := 1
                return QNaN_Indefinite
            return signed_one
        if sign_control[1]:
            MXCSR.IE := 1
            return QNaN_Indefinite
    if denormal:
        jbit := 0
        dst.exp := bias
        while jbit = 0:
            jbit := dst.fraction[51]
            dst.fraction := dst.fraction << 1
            dst.exp : = dst.exp - 1
        MXCSR.DE := 1
    unbiased_exp := dst.exp - bias
    odd_exp := unbiased_exp[0]
    signaling_bit := dst.fraction[51]
    if normalization_interval = 0b00:
        dst.exp := bias
    else if normalization_interval = 0b01:
        dst.exp := odd_exp ? bias-1 : bias
    else if normalization_interval = 0b10:
        dst.exp := bias-1
    else if normalization_interval = 0b11:
        dst.exp := signaling_bit ? bias-1 : bias
    return dst

VGETMANTPD (EVEX 编码版本)¶

VGETMANTPD dest{k1}, src, imm8
VL = 128, 256, or 512
KL := VL / 64
sign_control := imm8[3:2]
normalization_interval := imm8[1:0]
FOR i := 0 to KL-1:
    IF k1[i] or *no writemask*:
        IF SRC is memory and (EVEX.b = 1):
            tsrc := src.double[0]
        ELSE:
            tsrc := src.double[i]
        DEST.double[i] := getmant_fp64(tsrc, sign_control, normalization_interval)
    ELSE IF *zeroing*:
        DEST.double[i] := 0
    //else DEST.double[i] remains unchanged
DEST[MAX_VL-1:VL] := 0
