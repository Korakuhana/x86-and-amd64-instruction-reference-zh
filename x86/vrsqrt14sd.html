VRSQRT14SD — 计算标量双精度浮点值平方根的近似倒数

操作码/指令 Op/En 64/32位模式支持 CPUID 功能标志 描述
EVEX.LLIG.66.0F38.W1 4F /r VRSQRT14SD xmm1 {k1}{z}, xmm2, xmm3/m64 A V/V AVX512F 在写掩码 k1 控制下，计算 xmm3/m64 中标量双精度浮点值平方根的近似倒数，并将结果存储在 xmm1 的低四字元素中。将 xmm2 的 bits[127:64] 复制到 xmm1[127:64]。

指令操作数编码¶

Op/En 元组类型 操作数 1 操作数 2 操作数 3 操作数 4
A Tuple1 Scalar ModRM:reg (w) EVEX.vvvv (r) ModRM:r/m (r) N/A

描述¶

计算第二源操作数（第三操作数）低四字元素中标量双精度浮点值的平方根近似倒数，并根据写掩码将结果存储在目的操作数（第一操作数）的低四字元素中。该近似的最大相对误差小于 2-14。源操作数可以是 XMM 寄存器或 64 位内存位置。目的操作数是 XMM 寄存器。

目的 XMM 寄存器的 bits (127:64) 从第一源操作数的对应位复制。目的寄存器的 bits (MAXVL-1:128) 清零。

VRSQRT14SD 指令不受 MXCSR 寄存器中舍入控制位的影响。当源值为 0.0 时，返回带有源值符号的 ∞。当源操作数是 +∞ 时，返回 +ZERO。仅当 MXCSR 中设置了 DAZ 位时，渐近下溢 (denormal) 源值才会被视为零。否则它将被正确处理，并使用指定的掩码响应执行近似计算。当源值是负值（非 0.0）时，返回浮点 QNaN_indefinite。当源值是 SNaN 或 QNaN 时，SNaN 被转换为 QNaN，或者返回源 QNaN。

MXCSR 异常标志不受此指令影响，且不报告浮点异常。

操作¶

VRSQRT14SD (EVEX 版本)¶

IF k1[0] or *no writemask*
    THEN DEST[63:0] := APPROXIMATE(1.0/ SQRT(SRC2[63:0]))
    ELSE
        IF *merging-masking* ; merging-masking
            THEN *DEST[63:0] remains unchanged*
            ELSE
                    ; zeroing-masking
                THEN DEST[63:0] := 0
        FI;
FI;
DEST[127:64] := SRC1[127:64]
DEST[MAXVL-1:128] := 0
