描述¶

可变控制版本：

置换第一源操作数（第二操作数）中的四元组单精度浮点值，每个四元组使用第二源操作数对应双字元素中的 2 位控制字段。置换后的结果存储在目的操作数（第一操作数）中。

2 位控制字段位于每个双字元素的低两位（参见 图 5-26）。每个控制确定输入四元组中的哪个源元素被选择给目的元素。每个源四元组必须与目的地位于相同的 128 位区域中。

EVEX 版本：第二源操作数（第三操作数）是 ZMM/YMM/XMM 寄存器、512/256/128 位内存位置，或者是从 32 位内存位置广播的 512/256/128 位向量。置换后的结果在写掩码下写入目的地。

（立即数控制版本）

置换第一源操作数（第二操作数）中的四元组单精度浮点值，每个四元组使用 imm8 字节中的 2 位控制字段。目的操作数（第一操作数）中的每个 128 位通道使用同一个 imm8 字节的四个控制字段。

VEX 版本：源操作数是 YMM/XMM 寄存器或 256/128 位内存位置，并且目的操作数是 YMM/XMM 寄存器。

EVEX 版本：源操作数（第二操作数）是 ZMM/YMM/XMM 寄存器、512/256/128 位内存位置，或者是从 32 位内存位置广播的 512/256/128 位向量。置换后的结果在写掩码下写入目的地。

注意：对于 imm8 版本，VEX.vvvv 和 EVEX.vvvv 是保留的，必须为 1111b，否则指令会导致 #UD。

操作¶

Select4(SRC, control) {
CASE (control[1:0]) OF
    0: TMP := SRC[31:0];
    1: TMP := SRC[63:32];
    2: TMP := SRC[95:64];
    3: TMP := SRC[127:96];
ESAC;
RETURN TMP
}

SIMD 浮点异常¶

无。

其他异常¶

非 EVEX 编码指令，请参阅表 2-21，“第 4 类异常条件”。

此外：

#UD 如果 VEX.W = 1。

EVEX 编码指令，请参阅表 2-50，“第 E4NF 类异常条件”。

此外：

#UD 如果带 imm8 且 (E)VEX.vvvv != 1111B。
