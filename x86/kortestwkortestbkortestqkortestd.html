KORTESTW/KORTESTB/KORTESTQ/KORTESTD — 掩码按位或并设置标志位

操作码/指令 Op/En 64/32位模式支持 CPUID 特征标志 描述
VEX.L0.0F.W0 98 /r KORTESTW k1, k2 RR V/V AVX512F 对 16 位掩码 k1 和 k2 进行按位或，并相应地更新 ZF 和 CF。
VEX.L0.66.0F.W0 98 /r KORTESTB k1, k2 RR V/V AVX512DQ 对 8 位掩码 k1 和 k2 进行按位或，并相应地更新 ZF 和 CF。
VEX.L0.0F.W1 98 /r KORTESTQ k1, k2 RR V/V AVX512BW 对 64 位掩码 k1 和 k2 进行按位或，并相应地更新 ZF 和 CF。
VEX.L0.66.0F.W1 98 /r KORTESTD k1, k2 RR V/V AVX512BW 对 32 位掩码 k1 和 k2 进行按位或，并相应地更新 ZF 和 CF。

指令操作数编码¶

Op/En 操作数 1 操作数 2
RR ModRM:reg (w) ModRM:r/m (r, ModRM:[7:6] 必须是 11b)

描述¶

在向量掩码寄存器 k2 和向量掩码寄存器 k1 之间执行按位或，并根据操作结果设置 CF 和 ZF。

如果两个源都为 0x0，则设置 ZF 标志。如果在执行或操作后，操作结果全为 1，则设置 CF。

操作¶

KORTESTW¶

TMP[15:0] := DEST[15:0] BITWISE OR SRC[15:0]
IF(TMP[15:0]=0)
    THEN ZF := 1
    ELSE ZF := 0
FI;
IF(TMP[15:0]=FFFFh)
    THEN CF := 1
    ELSE CF := 0
FI;

TMP[7:0] := DEST[7:0] BITWISE OR SRC[7:0]

    THEN ZF := 1
    ELSE ZF := 0

KORTESTQ¶

TMP[63:0] := DEST[63:0] BITWISE OR SRC[63:0]
IF(TMP[63:0]=0)
    THEN ZF := 1
    ELSE ZF := 0
FI;
IF(TMP[63:0]==FFFFFFFF_FFFFFFFFh)
    THEN CF := 1
    ELSE CF := 0
FI;

    THEN ZF := 1

Intel C/C++ 编译器等价内部函数¶

KORTESTW __mmask16 _mm512_kortest[cz](__mmask16 a, __mmask16 b);

受影响的标志位¶

如果两个源的或运算结果全为 0，则设置 ZF 标志。

如果两个源的或运算结果全为 1，则设置 CF 标志。

OF、SF、AF 和 PF 标志设置为 0。
