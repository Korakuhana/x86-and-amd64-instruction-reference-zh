WRUSSD/WRUSSQ — 写入用户影子栈

操作码/指令 Op/En 64/32位模式支持 CPUID 功能标志 描述
66 0F 38 F5 !(11):rrr:bbb WRUSSD m32, r32 MR V/V CET_SS 向影子栈写入 4 字节。
66 REX.W 0F 38 F5 !(11):rrr:bbb WRUSSQ m64, r64 MR V/N.E. CET_SS 向影子栈写入 8 字节。

指令操作数编码¶

Op/En 操作数 1 操作数 2 操作数 3 操作数 4
MR ModRM:r/m (w) ModRM:reg (r) N/A N/A

描述¶

将源寄存器中的字节写入用户影子栈页面。WRUSS 指令只能在 CPL = 0 时执行，但处理器将其影子栈访问视为用户访问。

操作¶

IF CR4.CET = 0
    THEN #UD; FI;
IF CPL > 0
    THEN #GP(0); FI;
DEST_LA = Linear_Address(mem operand)
IF (operand size is 64 bit)
    THEN
        (* Destination not 8B aligned *)
        IF DEST_LA[2:0]
            THEN GP(0); FI;
        Shadow_stack_store 8 bytes of SRC to DEST_LA as user-mode access;
    ELSE
        (* Destination not 4B aligned *)
        IF DEST_LA[1:0]
            THEN GP(0); FI;
        Shadow_stack_store 4 bytes of SRC[31:0] to DEST_LA as user-mode access;
FI;

受影响的标志位¶

无。

保护模式异常¶

#UD 如果使用了 LOCK 前缀。
如果 CR4.CET = 0。
#GP(0) 如果内存操作数有效地址超出 CS、DS、ES、FS 或 GS 段限制。
如果目的操作数位于不可写段中。
如果使用 DS、ES、FS 或 GS 寄存器访问内存，且它包含一个空段选择子。
如果目的操作数的线性地址未 4 字节对齐。
如果 CPL 不为 0。
#SS(0) 如果内存操作数有效地址超出 SS 段限制。
#PF(错误码) 如果目的操作数不是用户影子栈。
其他终结性和非终结性故障。

实地址模式异常¶

#UD WRUSS（即 WRUSSD/WRUSSQ）指令在实地址模式下不被识别。

虚拟-8086 模式异常¶

#UD WRUSS（即 WRUSSD/WRUSSQ）指令在虚拟-8086 模式下不被识别。
