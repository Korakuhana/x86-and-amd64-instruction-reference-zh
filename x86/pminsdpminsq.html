描述¶

对第二个源操作数和第一个源操作数中的打包有符号双字或四字整数执行 SIMD 比较，并向目的操作数返回每对整数的最小值。

128位传统 SSE 版本：第一个源操作数和目的操作数是 XMM 寄存器。第二个源操作数是 XMM 寄存器或 128 位内存位置。对应目的寄存器的位（MAXVL-1:128）保持不变。

VEX.128 编码版本：第一个源操作数和目的操作数是 XMM 寄存器。第二个源操作数是 XMM 寄存器或 128 位内存位置。对应目的寄存器的位（MAXVL-1:128）清零。

VEX.256 编码版本：第二个源操作数可以是 YMM 寄存器或 256 位内存位置。第一个源操作数和目的操作数是 YMM 寄存器。对应目的寄存器的位（MAXVL-1:256）清零。

EVEX 编码版本：第一个源操作数是 ZMM/YMM/XMM 寄存器；第二个源操作数是 ZMM/YMM/XMM 寄存器、512/256/128 位内存位置或从 32/64 位内存位置广播的 512/256/128 位向量。根据写掩码 k1 条件性地更新目的操作数。

VPMINSD（EVEX 编码版本）¶

(KL, VL) = (4, 128), (8, 256), (16, 512)
FOR j := 0 TO KL-1
    i := j * 32
    IF k1[j] OR *no writemask* THEN
        IF (EVEX.b = 1) AND (SRC2 *is memory*)
            THEN
                IF SRC1[i+31:i] < SRC2[31:0]
                    THEN DEST[i+31:i] := SRC1[i+31:i];
                    ELSE DEST[i+31:i] := SRC2[31:0];
                FI;
            ELSE
                IF SRC1[i+31:i] < SRC2[i+31:i]
                    THEN DEST[i+31:i] := SRC1[i+31:i];
                    ELSE DEST[i+31:i] := SRC2[i+31:i];
            FI;
        FI;
        ELSE
            IF *merging-masking* ; merging-masking
                THEN *DEST[i+31:i] remains unchanged*
                ELSE ; zeroing-masking
                    DEST[i+31:i] := 0
            FI
    FI;
ENDFOR;
DEST[MAXVL-1:VL] := 0

VPMINSQ（EVEX 编码版本）¶

(KL, VL) = (2, 128), (4, 256), (8, 512)
FOR j := 0 TO KL-1
    i := j * 64
    IF k1[j] OR *no writemask* THEN
        IF (EVEX.b = 1) AND (SRC2 *is memory*)
            THEN
                IF SRC1[i+63:i] < SRC2[63:0]
                    THEN DEST[i+63:i] := SRC1[i+63:i];
                    ELSE DEST[i+63:i] := SRC2[63:0];
                FI;
            ELSE
                IF SRC1[i+63:i] < SRC2[i+63:i]
                    THEN DEST[i+63:i] := SRC1[i+63:i];
                    ELSE DEST[i+63:i] := SRC2[i+63:i];
            FI;
        FI;
        ELSE
            IF *merging-masking* ; merging-masking
                THEN *DEST[i+63:i] remains unchanged*
                ELSE
                        ; zeroing-masking
                    DEST[i+63:i] := 0
            FI
    FI;
ENDFOR;
DEST[MAXVL-1:VL] := 0
