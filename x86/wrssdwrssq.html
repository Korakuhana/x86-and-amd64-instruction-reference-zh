WRSSD/WRSSQ — 写入影子栈

操作码/指令 Op/En 64/32位模式支持 CPUID 功能标志 描述
0F 38 F6 !(11):rrr:bbb WRSSD m32, r32 MR V/V CET_SS 向影子栈写入 4 字节。
REX.W 0F 38 F6 !(11):rrr:bbb WRSSQ m64, r64 MR V/N.E. CET_SS 向影子栈写入 8 字节。

指令操作数编码¶

Op/En 操作数 1 操作数 2 操作数 3 操作数 4
MR ModRM:r/m (w) ModRM:reg (r) N/A N/A

描述¶

将源寄存器中的字节写入影子栈。

操作¶

IF CPL = 3
    IF (CR4.CET & IA32_U_CET.SH_STK_EN) = 0
        THEN #UD; FI;
    IF (IA32_U_CET.WR_SHSTK_EN) = 0
        THEN #UD; FI;
ELSE
    IF (CR4.CET & IA32_S_CET.SH_STK_EN) = 0
        THEN #UD; FI;
    IF (IA32_S_CET.WR_SHSTK_EN) = 0
        THEN #UD; FI;
FI;
DEST_LA = Linear_Address(mem operand)
IF (operand size is 64 bit)
    THEN
        (* Destination not 8B aligned *)
        IF DEST_LA[2:0]
            THEN GP(0); FI;
        Shadow_stack_store 8 bytes of SRC to DEST_LA;
    ELSE
        (* Destination not 4B aligned *)
        IF DEST_LA[1:0]
            THEN GP(0); FI;
        Shadow_stack_store 4 bytes of SRC[31:0] to DEST_LA;
FI;

受影响的标志位¶

无。

C/C++ 编译器等价内部函数¶

WRSSD void _wrssd(__int32, void *);

WRSSQ void _wrssq(__int64, void *);

保护模式异常¶

#UD 如果使用了 LOCK 前缀。
如果 CR4.CET = 0。
如果 CPL = 3 且 IA32_U_CET.SH_STK_EN = 0。
如果 CPL < 3 且 IA32_S_CET.SH_STK_EN = 0。
如果 CPL = 3 且 IA32_U_CET.WR_SHSTK_EN = 0。
如果 CPL < 3 且 IA32_S_CET.WR_SHSTK_EN = 0。
#GP(0) 如果内存操作数有效地址超出 CS、DS、ES、FS 或 GS 段限制。
如果目的操作数位于不可写段中。
如果使用 DS、ES、FS 或 GS 寄存器访问内存，且它包含一个空段选择子。
如果目的操作数的线性地址未 4 字节对齐。
#SS(0) 如果内存操作数有效地址超出 SS 段限制。
#PF(错误码) 如果发生缺页异常，且在 CPL = 3 时目的操作数不是用户影子栈，或在 CPL < 3 时不是主管影子栈。
其他终结性和非终结性故障。

实地址模式异常¶

#UD WRSS（即 WRSSD/WRSSQ）指令在实地址模式下不被识别。

虚拟-8086 模式异常¶

#UD WRSS（即 WRSSD/WRSSQ）指令在虚拟-8086 模式下不被识别。
