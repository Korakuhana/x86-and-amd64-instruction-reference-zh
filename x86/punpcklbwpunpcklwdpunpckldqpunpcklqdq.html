描述¶

将目的操作数（第一个操作数）和源操作数（第二个操作数）的低位数据元素（字节、字、双字和四字）交错并解包到目的操作数中。（图 4-22 显示了 64 位操作数中字节的解包操作。）。高位数据元素被忽略。

当源数据来自 128 位内存操作数时，实现可以仅获取适当的 64 位；然而，仍将强制执行对齐到 16 字节边界和正常的段检查。

(V)PUNPCKLBW 指令交错源操作数和目的操作数的低位字节，(V)PUNPCKLWD 指令交错源操作数和目的操作数的低位字，(V)PUNPCKLDQ 指令交错源操作数和目的操作数的低位双字（或多个双字），(V)PUNPCKLQDQ 指令交错源操作数和目的操作数的低位四字。

通过在源操作数中放置全 0，这些指令可以分别用于将字节转换为字、字转换为双字、双字转换为四字，以及四字转换为双四字。在此，如果源操作数包含全 0，则结果（存储在目的操作数中）包含目的操作数中原始值高位数据元素的零扩展。例如，使用 (V)PUNPCKLBW 指令时，高位字节被零扩展（即解包为无符号字整数），而使用 (V)PUNPCKLWD 指令时，高位字被零扩展（解包为无符号双字整数）。

在 64 位模式下且未在 VEX/EVEX 中编码时，使用 REX.R 形式的 REX 前缀允许此指令访问其他寄存器（XMM8-XMM15）。

传统 SSE 版本 64 位操作数：源操作数可以是 MMX 技术寄存器或 32 位内存位置。目的操作数是 MMX 技术寄存器。

128位传统 SSE 版本：第二个源操作数是 XMM 寄存器或 128 位内存位置。第一个源操作数和目的操作数是 XMM 寄存器。对应 YMM 目的寄存器的位（MAXVL-1:128）保持不变。

VEX.128 编码版本：第二个源操作数是 XMM 寄存器或 128 位内存位置。第一个源操作数和目的操作数是 XMM 寄存器。目的 YMM 寄存器的位（MAXVL-1:128）清零。

VEX.256 编码版本：第二个源操作数是 YMM 寄存器或 256 位内存位置。第一个源操作数和目的操作数是 YMM 寄存器。对应 ZMM 寄存器的位（MAXVL-1:256）清零。

EVEX 编码 VPUNPCKLDQ/QDQ：第二个源操作数是 ZMM/YMM/XMM 寄存器、512/256/128 位内存位置，或是从 32/64 位内存位置广播的 512/256/128 位向量。第一个源操作数和目的操作数是 ZMM/YMM/XMM 寄存器。目的操作数在写掩码 k1 下有条件地更新。

EVEX 编码 VPUNPCKLWD/BW：第二个源操作数是 ZMM/YMM/XMM 寄存器、512/256/128 位内存位置。第一个源操作数和目的操作数是 ZMM/YMM/XMM 寄存器。目的操作数在写掩码 k1 下有条件地更新。

VPUNPCKLBW（EVEX.512 编码指令）¶

(KL, VL) = (16, 128), (32, 256), (64, 512)
IF VL = 128
    TMP_DEST[VL-1:0] := INTERLEAVE_BYTES(SRC1[VL-1:0], SRC2[VL-1:0])
FI;
IF VL = 256
    TMP_DEST[VL-1:0] := INTERLEAVE_BYTES_256b(SRC1[VL-1:0], SRC2[VL-1:0])
FI;
IF VL = 512
    TMP_DEST[VL-1:0] := INTERLEAVE_BYTES_512b(SRC1[VL-1:0], SRC2[VL-1:0])
FI;
FOR j := 0 TO KL-1
    i := j * 8
    IF k1[j] OR *no writemask*
        THEN DEST[i+7:i] := TMP_DEST[i+7:i]
        ELSE
            IF *merging-masking*
                        ; merging-masking
                THEN *DEST[i+7:i] remains unchanged*
                ELSE *zeroing-masking*
                            ; zeroing-masking
                    DEST[i+7:i] := 0
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0
DEST[511:0] := INTERLEAVE_BYTES_512b(SRC1, SRC2)

(KL, VL) = (2, 128), (4, 256), (8, 512)
FOR j := 0 TO KL-1
    i := j * 64
    IF (EVEX.b = 1) AND (SRC2 *is memory*)
        THEN TMP_SRC2[i+63:i] := SRC2[63:0]
        ELSE TMP_SRC2[i+63:i] := SRC2[i+63:i]
    FI;
ENDFOR;
IF VL = 128
    TMP_DEST[VL-1:0] := INTERLEAVE_QWORDS(SRC1[VL-1:0], TMP_SRC2[VL-1:0])
FI;
IF VL = 256
    TMP_DEST[VL-1:0] := INTERLEAVE_QWORDS_256b(SRC1[VL-1:0], TMP_SRC2[VL-1:0])
FI;
IF VL = 512
    TMP_DEST[VL-1:0] := INTERLEAVE_QWORDS_512b(SRC1[VL-1:0], TMP_SRC2[VL-1:0])
FI;
FOR j := 0 TO KL-1
    i := j * 64
    IF k1[j] OR *no writemask*
        THEN DEST[i+63:i] := TMP_DEST[i+63:i]
        ELSE
            IF *merging-masking*
                        ; merging-masking
                THEN *DEST[i+63:i] remains unchanged*
                ELSE *zeroing-masking*
                            ; zeroing-masking
                    DEST[i+63:i] := 0
            FI
    FI;
ENDFOR
