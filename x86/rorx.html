RORX — 逻辑右移且不影响标志位

操作码/指令 Op/En 64/32位模式 CPUID 特征标志 描述
VEX.LZ.F2.0F3A.W0 F0 /r ib RORX r32, r/m32, imm8 RMI V/V BMI2 将 32 位 r/m32 循环右移 imm8 次，不影响算术标志位。
VEX.LZ.F2.0F3A.W1 F0 /r ib RORX r64, r/m64, imm8 RMI V/N.E. BMI2 将 64 位 r/m64 循环右移 imm8 次，不影响算术标志位。

指令操作数编码¶

Op/En 操作数 1 操作数 2 操作数 3 操作数 4
RMI ModRM:reg (w) ModRM:r/m (r) imm8 N/A

描述¶

将第二个操作数的位右移由 imm8 指定的计数值，且不影响算术标志位。RORX 指令不读取或写入算术标志位。

在实模式和虚拟-8086 模式下不支持此指令。如果不在 64 位模式下，操作数大小始终为 32 位。在 64 位模式下，操作数大小为 64 需要 VEX.W1。在非 64 位模式下，VEX.W1 被忽略。尝试在 VEX.L 不等于 0 的情况下执行此指令将导致 #UD。

操作¶

IF (OperandSize = 32)
    y := imm8 AND 1FH;
    DEST := (SRC >> y) | (SRC << (32-y));
ELSEIF (OperandSize = 64 )
    y := imm8 AND 3FH;
    DEST := (SRC >> y) | (SRC << (64-y));
FI;

受影响的标志位¶

无。
