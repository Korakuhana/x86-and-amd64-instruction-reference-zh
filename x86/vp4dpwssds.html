VP4DPWSSDS — 有符号字的点积与双字累加和饱和（4次迭代）

操作码/指令 Op/En 64/32位模式支持 CPUID 功能标志 描述
EVEX.512.F2.0F38.W0 53 /r VP4DPWSSDS zmm1{k1}{z}, zmm2+3, m128 A V/V AVX512_4VNNIW 将由 zmm2 指示的源寄存器块中的有符号字与来自 m128 的有符号字相乘，并在 zmm1 中以有符号饱和方式累加得到的双字结果。

指令操作数编码¶

Op/En 元组类型 操作数 1 操作数 2 操作数 3 操作数 4
A Tuple1_4X ModRM:reg (r, w) EVEX.vvvv (r) ModRM:r/m (r) N/A

描述¶

该指令计算两个有符号字操作数的 4 个连续寄存器源块点积，并进行双字累加和有符号饱和。在四个步骤的每一步中，顺序选择内存操作数。

在上框中，“+3” 的表示方法用于表示该指令根据该操作数访问 4 个源寄存器；源寄存器是连续的，从 4 的倍数边界开始，并包含编码 of 寄存器操作数。

此指令支持内存故障抑制。如果掩码的低 16 位中的任何一位被设置为 1，或者如果使用了“无掩码”编码，则加载整个内存操作数。

元组类型 Tuple1_4X 意味着此指令的内存操作部分引用了四个 32 位元素（16 字节）。

操作¶

src_reg_id is the 5 bit index of the vector register specified in the instruction as the src1 register.

VP4DPWSSDS dest, src1, src2¶

(KL,VL) = (16,512)
N := 4
ORIGDEST := DEST
src_base := src_reg_id & ~ (N-1) // for src1 operand
FOR i := 0 to KL-1:
    IF k1[i] or *no writemask*:
        FOR m := 0 to N-1:
            t := SRC2.dword[m]
            p1dword := reg[src_base+m].word[2*i] * t.word[0]
            p2dword := reg[src_base+m].word[2*i+1] * t.word[1]
            DEST.dword[i] := SIGNED_DWORD_SATURATE(DEST.dword[i] + p1dword + p2dword)
    ELSE IF *zeroing*:
        DEST.dword[i] := 0
    ELSE
        DEST.dword[i] := ORIGDEST.dword[i]
DEST[MAX_VL-1:VL] := 0
