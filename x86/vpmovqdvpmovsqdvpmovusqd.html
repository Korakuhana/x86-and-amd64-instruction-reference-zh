描述¶

VPMOVQD 使用截断将源操作数（第二操作数）中的 64 位整数元素向下转换为打包双字。VPMOVSQD 使用有符号饱和度将有符号 64 位整数转换为打包有符号双字。VPMOVUSQD 使用无符号饱和度将无符号四字值转换为无符号双字值。

源操作数是 ZMM/YMM/XMM 寄存器。目的操作数是 YMM/XMM/XMM 寄存器或 256/128/64 位内存位置。

向下转换的双字元素从最低有效双字写入目的操作数（第一操作数）。目的操作数的双字元素根据写掩码进行更新。寄存器目的地的 bits (MAXVL-1:256/128/64) 清零。

EVEX.vvvv 保留且必须为 1111b，否则指令将导致 #UD。

操作¶

当目的地是寄存器时 VPMOVQD 指令 (EVEX 编码版本)¶

(KL, VL) = (2, 128), (4, 256), (8, 512)
FOR j := 0 TO KL-1
    i := j * 32
    m := j * 64
    IF k1[j] OR *no writemask*
        THEN DEST[i+31:i] := TruncateQuadWordToDWord (SRC[m+63:m])
        ELSE
            IF *merging-masking* ; merging-masking
                THEN *DEST[i+31:i] remains unchanged*
                ELSE *zeroing-masking*
                        ; zeroing-masking
                    DEST[i+31:i] := 0
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL/2] := 0;
