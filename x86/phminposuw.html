PHMINPOSUW — 打包水平字最小值

操作码/指令 Op/En 64/32位模式支持 CPUID 特征标志 描述
66 0F 38 41 /r PHMINPOSUW xmm1, xmm2/m128 RM V/V SSE4_1 在 xmm2/m128 中查找最小的无符号字，并将其值放入 xmm1 的低字中，将其索引放入 xmm1 的次低字中。
VEX.128.66.0F38.WIG 41 /r VPHMINPOSUW xmm1, xmm2/m128 RM V/V AVX 在 xmm2/m128 中查找最小的无符号字，并将其值放入 xmm1 的低字中，将其索引放入 xmm1 的次低字中。

指令操作数编码¶

Op/En 操作数 1 操作数 2 操作数 3 操作数 4
RM ModRM:reg (w) ModRM:r/m (r) N/A N/A

描述¶

确定源操作数（第二个操作数）中的最小无符号字值，并将该无符号字放入目的操作数（第一个操作数）的低字（位 0-15）中。最小值所在字的索引存储在目的操作数的位 16-18 中。目的操作数剩余的高位被置零。

128位传统 SSE 版本：对应 XMM 目的寄存器的位（MAXVL-1:128）保持不变。

VEX.128 编码版本：目的 XMM 寄存器的位（MAXVL-1:128）清零。VEX.vvvv 是保留的且必须为 1111b，VEX.L 必须为 0，否则指令将导致 #UD。

操作¶

PHMINPOSUW（128位传统 SSE 版本）¶

INDEX := 0;
MIN := SRC[15:0]
IF (SRC[31:16] < MIN)
    THEN INDEX := 1; MIN := SRC[31:16]; FI;
IF (SRC[47:32] < MIN)
    THEN INDEX := 2; MIN := SRC[47:32]; FI;
* Repeat operation for words 3 through 6
IF (SRC[127:112] < MIN)
    THEN INDEX := 7; MIN := SRC[127:112]; FI;
DEST[15:0] := MIN;
DEST[18:16] := INDEX;
DEST[127:19] := 0000000000000000000000000000H;

VPHMINPOSUW（VEX.128 编码版本）¶

INDEX := 0
MIN := SRC[15:0]
IF (SRC[31:16] < MIN) THEN INDEX := 1; MIN := SRC[31:16]
IF (SRC[47:32] < MIN) THEN INDEX := 2; MIN := SRC[47:32]
* Repeat operation for words 3 through 6
IF (SRC[127:112] < MIN) THEN INDEX := 7; MIN := SRC[127:112]
DEST[15:0] := MIN
DEST[18:16] := INDEX
DEST[127:19] := 0000000000000000000000000000H
DEST[MAXVL-1:128] := 0

其他异常¶

请参阅表 2-21，“第 4 类异常条件”，此外：

#UD 如果 VEX.L = 1。
如果 VEX.vvvv ≠ 1111B。
