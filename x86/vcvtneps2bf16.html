描述¶

将一个包含打包单精度数据的 SIMD 寄存器转换为一个包含打包 BF16 数据的寄存器。

此指令使用“舍入到最接近的偶数 (Round to nearest (even))”舍入模式。输出的渐近下溢（反常数）总是被清零（flushed to zero），输入的渐近下溢总是被视为零。不参考也不更新 MXCSR。

如指令操作数编码表所示，EVEX.vvvv 字段不用于对操作数进行编码。EVEX.vvvv 保留，必须为 0b1111，否则指令会导致 #UD。

操作¶

Define convert_fp32_to_bfloat16(x):
    IF x is zero or denormal:
        dest[15] := x[31] // 保持符号位的零（反常数转为零）
        dest[14:0] := 0
    ELSE IF x is infinity:
        dest[15:0] := x[31:16]
    ELSE IF x is NAN:
        dest[15:0] := x[31:16] // 截断并置位尾数最高位（MSB）以强制产生 QNAN
        dest[6] := 1
    ELSE // 正常数
        LSB := x[16]
        rounding_bias := 0x00007FFF + LSB
        temp[31:0] := x[31:0] + rounding_bias // 整数加法
        dest[15:0] := temp[31:16]
    RETURN dest

VCVTNEPS2BF16 dest, src¶

VL = (128, 256, 512)
KL = VL/16
origdest := dest
FOR i := 0 to KL/2-1:
    IF k1[ i ] or *no writemask*:
        IF src is memory and evex.b == 1:
            t := src.fp32[0]
        ELSE:
            t := src.fp32[ i ]
        dest.word[i] := convert_fp32_to_bfloat16(t)
    ELSE IF *zeroing*:
        dest.word[ i ] := 0
    ELSE: // 合并掩码，目的元素保持不变
        dest.word[ i ] := origdest.word[ i ]
DEST[MAXVL-1:VL/2] := 0
