SHA256RNDS2 — 执行两轮 SHA256 操作

操作码/指令 Op/En 64/32位模式支持 CPUID 特征标志 描述
NP 0F 38 CB /r SHA256RNDS2 xmm1, xmm2/m128, <XMM0> RMI V/V SHA 使用来自 xmm1 的初始 SHA256 状态 (C,D,G,H)、来自 xmm2/m128 的初始 SHA256 状态 (A,B,E,F) 以及来自隐式操作数 XMM0 的下 2 轮消息双字和相应轮常数的预计算和，执行 2 轮 SHA256 操作，并将更新后的 SHA256 状态 (A,B,E,F) 结果存储在 xmm1 中。

指令操作数编码¶

Op/En 操作数 1 操作数 2 操作数 3
RMI ModRM:reg (r, w) ModRM:r/m (r) 隐式 XMM0 (r)

描述¶

SHA256RNDS2 指令执行 2 轮 SHA256 操作，使用来自第一个操作数的初始 SHA256 状态 (C,D,G,H)、来自第二个操作数的初始 SHA256 状态 (A,B,E,F) 以及来自隐式操作数 xmm0 的下 2 轮消息双字和相应轮常数的预计算和。请注意，该指令仅使用 XMM0 的低两个双字。

更新后的 SHA256 状态 (A,B,E,F) 被写入第一个操作数，而第二个操作数可以在后续轮次中用作更新后的状态 (C,D,G,H)。

操作¶

SHA256RNDS2¶

A_0 := SRC2[127:96];
B_0 := SRC2[95:64];
C_0 := SRC1[127:96];
D_0 := SRC1[95:64];
E_0 := SRC2[63:32];
F_0 := SRC2[31:0];
G_0 := SRC1[63:32];
H_0 := SRC1[31:0];
WK0 := XMM0[31: 0];
WK1 := XMM0[63: 32];
FOR i = 0 to 1
    A_(i +1) :=
        Ch (E_i, F_i, G_i) +Σ1( E_i) +WKi+ H_i + Maj(A_i , B_i, C_i) +Σ0( A_i);
    B_(i +1) :=
        A_i;
    C_(i +1) :=
        B_i ;
    D_(i +1) :=
        C_i;
    E_(i +1) :=
        Ch (E_i, F_i, G_i) +Σ1( E_i) +WKi+ H_i + D_i;
    F_(i +1) :=
        E_i ;
    G_(i +1) :=
        F_i;
    H_(i +1) :=
        G_i;
ENDFOR
DEST[127:96] := A_2;
DEST[95:64] := B_2;
DEST[63:32] := E_2;
DEST[31:0] := F_2;
