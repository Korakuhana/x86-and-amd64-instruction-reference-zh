MULSS — 标量单精度浮点值相乘

操作码/指令 Op / En 64/32位模式支持 CPUID 特征标志 描述
F3 0F 59 /r MULSS xmm1,xmm2/m32 A 有效/有效 SSE 将 xmm2/m32 中的低位单精度浮点值与 xmm1 中的低位单精度浮点值相乘。
VEX.LIG.F3.0F.WIG 59 /r VMULSS xmm1,xmm2, xmm3/m32 B 有效/有效 AVX 将 xmm3/m32 中的低位单精度浮点值与 xmm2 中的低位单精度浮点值相乘。
EVEX.LLIG.F3.0F.W0 59 /r VMULSS xmm1 {k1}{z}, xmm2, xmm3/m32 {er} C 有效/有效 AVX512F 将 xmm3/m32 中的低位单精度浮点值与 xmm2 中的低位单精度浮点值相乘。

描述¶

将第二个源操作数中的低位单精度浮点值与第一个源操作数中的低位单精度浮点值相乘，并将单精度浮点结果存储在目的操作数中。第二个源操作数可以是 XMM 寄存器或 32 位内存位置。第一个源操作数 and 目的操作数是 XMM 寄存器。

128 位传统 SSE 版本：第一个源操作数与目的操作数相同。对应 YMM 目的寄存器的位 (MAXVL-1:32) 保持不变。

VEX.128 和 EVEX 编码版本：第一个源操作数是通过 VEX.vvvv 编码的 XMM 寄存器。目的操作数的高三个双字复制自第一个源操作数。目的寄存器的位 (MAXVL-1:128) 被清零。

EVEX 编码版本：目的操作数的低双字元素根据写掩码进行更新。

软件应确保 VMULSS 用 VEX.L=0 编码。用 VEX.L=1 编码 VMULSS 可能会在不同的处理器世代中遇到不可预测的行为。

操作¶

VMULSS (EVEX Encoded Version)¶

IF (EVEX.b = 1) AND SRC2 *is a register*
    THEN
        SET_ROUNDING_MODE_FOR_THIS_INSTRUCTION(EVEX.RC);
    ELSE
        SET_ROUNDING_MODE_FOR_THIS_INSTRUCTION(MXCSR.RC);
FI;
IF k1[0] or *no writemask*
    THEN DEST[31:0] := SRC1[31:0] * SRC2[31:0]
    ELSE
        IF *merging-masking* ; merging-masking
            THEN *DEST[31:0] remains unchanged*
            ELSE ; zeroing-masking
                THEN DEST[31:0] := 0
            FI
    FI;
ENDFOR
DEST[127:32] := SRC1[127:32]
DEST[MAXVL-1:128] := 0

VMULSS (VEX.128 Encoded Version)¶

DEST[31:0] := SRC1[31:0] * SRC2[31:0]
DEST[127:32] := SRC1[127:32]
DEST[MAXVL-1:128] := 0

MULSS (128-bit Legacy SSE Version)¶

DEST[31:0] := DEST[31:0] * SRC[31:0]
DEST[MAXVL-1:32] (Unmodified)
