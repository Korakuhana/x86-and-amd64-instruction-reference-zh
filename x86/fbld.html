FBLD — 加载二进码十进数 (BCD)

操作码 指令 64位模式 兼容/传统模式 描述
DF /4 FBLD m80bcd 有效 有效 将 BCD 值转换为浮点数并压入 FPU 栈。

描述¶

将 BCD 源操作数转换为双精度扩展浮点格式，并将该值压入 FPU 栈。加载源操作数时不会发生舍入误差。保留源操作数的符号，包括 -0。

假设打包的 BCD 数位在 0 到 9 的范围内；该指令不会检查无效数位（AH 到 FH）。尝试加载无效的编码会产生未定义的结果。

此指令的操作在非 64 位模式 and 64 位模式下相同。

操作¶

TOP := TOP − 1;
ST(0) := ConvertToDoubleExtendedPrecisionFP(SRC);

影响的 FPU 标志寄存器¶

C1 如果发生栈溢出，则设置为 1；否则设置为 0。
C0, C2, C3 未定义。

浮点异常¶

#IS 发生栈溢出。

保护模式异常¶

#GP(0) 如果内存操作数有效地址超出 CS、DS、ES、FS 或 GS 段限制。
如果 DS、ES、FS 或 GS 寄存器包含 NULL 段选择子。
#SS(0) 如果内存操作数有效地址超出 SS 段限制。
#NM CR0.EM[bit 2] 或 CR0.TS[bit 3] = 1。
#PF(错误代码) 如果发生页面错误。
#AC(0) 如果启用了对齐检查，并且在当前特权级别为 3 时进行了未对齐的内存引用。
#UD 如果使用了 LOCK 前缀。

#SS 如果内存操作数有效地址超出 SS 段限制。
#NM CR0.EM[bit 2] 或 CR0.TS[bit 3] = 1。
#UD 如果使用了 LOCK 前缀。
