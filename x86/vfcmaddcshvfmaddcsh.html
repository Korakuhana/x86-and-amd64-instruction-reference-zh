VFCMADDCSH/VFMADDCSH — 复数乘加标量 FP16 值

操作码/指令 Op/En 64/32位模式支持 CPUID 特征标志 描述
EVEX.LLIG.F2.MAP6.W0 57 /r VFCMADDCSH xmm1{k1}{z}, xmm2, xmm3/m32 {er} A V/V AVX512-FP16 将 xmm2 中的一对 FP16 值与 xmm3/m32 的复共轭进行复数相乘，加到 xmm1 中，并在写掩码 k1 控制下将结果存储在 xmm1 中。xmm2 的 127:32 位被复制到 xmm1[127:32] 中。
EVEX.LLIG.F3.MAP6.W0 57 /r VFMADDCSH xmm1{k1}{z}, xmm2, xmm3/m32 {er} A V/V AVX512-FP16 将 xmm2 中的一对 FP16 值与 xmm3/m32 进行复数相乘，加到 xmm1 中，并在写掩码 k1 控制下将结果存储在 xmm1 中。xmm2 的 127:32 位被复制到 xmm1[127:32] 中。

描述¶

此指令执行复数乘法和累加操作。存在该操作的普通形式和共轭形式。

此操作的掩码处理是在代表一对 FP16 值的 32 位量上进行的。

目的操作数的 127:32 位从第一个源操作数的对应位复制。目的操作数的 MAXVL-1:128 位清零。目的操作数的低 FP16 元素根据写掩码进行更新。

在每个 FMA（融合乘加）边界处执行舍入。执行时就像所有 MXCSR 异常均被屏蔽一样。MXCSR 状态位被更新以反映异常条件。

操作¶

VFCMADDCSH dest{k1}, src1, src2 (AVX512)¶

IF k1[0] or *no writemask*:
    tmp[0] := dest.fp16[0] + src1.fp16[0] * src2.fp16[0]
    tmp[1] := dest.fp16[1] + src1.fp16[1] * src2.fp16[0]
    // conjugate version subtracts odd final term
    dest.fp16[0] := tmp[0] + src1.fp16[1] * src2.fp16[1]
    dest.fp16[1] := tmp[1] - src1.fp16[0] * src2.fp16[1]
ELSE IF *zeroing*:
    dest.fp16[0] := 0
    dest.fp16[1] := 0
DEST[127:32] := src1[127:32] // copy upper part of src1
DEST[MAXVL-1:128] := 0

VFMADDCSH dest{k1}, src1, src2 (AVX512)¶

IF k1[0] or *no writemask*:
    tmp[0] := dest.fp16[0] + src1.fp16[0] * src2.fp16[0]
    tmp[1] := dest.fp16[1] + src1.fp16[1] * src2.fp16[0]
    // non-conjugate version subtracts last even term
    dest.fp16[0] := tmp[0] - src1.fp16[1] * src2.fp16[1]
    dest.fp16[1] := tmp[1] + src1.fp16[0] * src2.fp16[1]
ELSE IF *zeroing*:
    dest.fp16[0] := 0
    dest.fp16[1] := 0
DEST[127:32] := src1[127:32] // copy upper part of src1
DEST[MAXVL-1:128] := 0
