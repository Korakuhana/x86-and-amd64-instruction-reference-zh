VRCP28SD — 相对误差小于 2^-28 的标量双精度浮点值倒数近似

操作码/指令 Op/En 64/32位模式支持 CPUID 功能标志 描述
EVEX.LLIG.66.0F38.W1 CB /r VRCP28SD xmm1 {k1}{z}, xmm2, xmm3/m64 {sae} A V/V AVX512ER 在写掩码控制下，计算 xmm3/m64 中标量双精度浮点值的近似倒数（相对误差 < 2-28），并将结果存储在 xmm1 中。此外，将 xmm2 的高双精度浮点值 (bits[127:64]) 复制到 xmm1[127:64]。

指令操作数编码¶

Op/En 元组类型 操作数 1 操作数 2 操作数 3 操作数 4
A Tuple1 Scalar ModRM:reg (w) EVEX.vvvv (r) ModRM:r/m (r) N/A

描述¶

计算第二源操作数（第三操作数）中低 float64 值的倒数近似值，并将结果存储在目的操作数（第一操作数）中。近似倒数估算的最大相对误差小于 2-28。在写掩码 k1 的控制下，结果被写入目的操作数的低 float64 元素中。目的操作数的 bits 127:64 从第一源操作数（第二操作数）的对应位复制。

渐近下溢 (denormal) 输入值被视为零且不发出 #DE 信号，这与 MXCSR.DAZ 无关。渐近下溢结果被刷新为零且不发出 #UE 信号，这与 MXCSR.FTZ 无关。

如果任何源元素是 NaN，则为该元素返回静默化 (quietized) 的 NaN 源值。如果任何源元素是 ±∞，则为该元素返回 ±0.0。此外，如果任何源元素是 ±0.0，则为该元素返回 ±∞。

第一源操作数是 XMM 寄存器。第二源操作数是 XMM 寄存器或 64 位内存位置。目的操作数是 XMM 寄存器，在写掩码 k1 控制下有条件地更新。

操作¶

VRCP28SD (EVEX 编码版本)¶

IF k1[0] OR *no writemask* THEN
        DEST[63: 0] := RCP_28_DP(1.0/SRC2[63: 0]);
ELSE
    IF *merging-masking* ; merging-masking
        THEN *DEST[63: 0] remains unchanged*
        ELSE ; zeroing-masking
            DEST[63: 0] := 0
    FI;
FI;
DEST[127:64] := SRC1[127: 64]
DEST[MAXVL-1:128] := 0
