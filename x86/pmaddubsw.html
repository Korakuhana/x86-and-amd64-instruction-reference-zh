描述¶

(V)PMADDUBSW 将目的操作数（第一个操作数）的每个无符号字节与源操作数（第二个操作数）的对应有符号字节进行垂直相乘，产生中间有符号 16 位整数。每个相邻的有符号字对相加，并将饱和的结果打包到目的操作数中。例如，源操作数和目的操作数中的最低位字节（位 7-0）相乘，中间有符号字结果与操作数次低位字节（位 15-8）的对应中间结果相加；符号饱和的结果存储在目的寄存器的最低字中（15-0）。对其他相邻字节对也执行相同的操作。两个操作数都可以是 MMX 寄存器或 XMM 寄存器。当源操作数是 128 位内存操作数时，操作数必须对齐到 16 字节边界，否则将产生通用保护异常 (#GP)。

在 64 位模式下且不使用 VEX/EVEX 编码时，使用 REX 前缀可访问 XMM8-XMM15。

128位传统 SSE 版本：第一个源操作数 and 目的操作数是 XMM 寄存器。第二个源操作数是 XMM 寄存器或 128 位内存位置。对应目的寄存器的位（MAXVL-1:128）保持不变。

VEX.128 和 EVEX.128 编码版本：第一个源操作数和目的操作数是 XMM 寄存器。第二个源操作数是 XMM 寄存器或 128 位内存位置。对应目的寄存器的位（MAXVL-1:128）清零。

VEX.256 和 EVEX.256 编码版本：第二个源操作数可以是 YMM 寄存器或 256 位内存位置。第一个源操作数和目的操作数是 YMM 寄存器。对应 ZMM 寄存器的位（MAXVL-1:256）清零。

EVEX.512 编码版本：第二个源操作数可以是 ZMM 寄存器或 512 位内存位置。第一个源操作数和目的操作数是 ZMM 寄存器。

操作¶

PMADDUBSW（具有 64 位操作数）¶

DEST[15-0] = SaturateToSignedWord(SRC[15-8]*DEST[15-8]+SRC[7-0]*DEST[7-0]);
DEST[31-16] = SaturateToSignedWord(SRC[31-24]*DEST[31-24]+SRC[23-16]*DEST[23-16]);
DEST[47-32] = SaturateToSignedWord(SRC[47-40]*DEST[47-40]+SRC[39-32]*DEST[39-32]);
DEST[63-48] = SaturateToSignedWord(SRC[63-56]*DEST[63-56]+SRC[55-48]*DEST[55-48]);

PMADDUBSW（具有 128 位操作数）¶

DEST[15-0] = SaturateToSignedWord(SRC[15-8]* DEST[15-8]+SRC[7-0]*DEST[7-0]);
// Repeat operation for 2nd through 7th word
SRC1/DEST[127-112] = SaturateToSignedWord(SRC[127-120]*DEST[127-120]+ SRC[119-112]* DEST[119-112]);

VPMADDUBSW（VEX.128 编码版本）¶

DEST[15:0] := SaturateToSignedWord(SRC2[15:8]* SRC1[15:8]+SRC2[7:0]*SRC1[7:0])
// Repeat operation for 2nd through 7th word
DEST[127:112] := SaturateToSignedWord(SRC2[127:120]*SRC1[127:120]+ SRC2[119:112]* SRC1[119:112])
DEST[MAXVL-1:128] := 0

VPMADDUBSW（VEX.256 编码版本）¶

DEST[15:0] := SaturateToSignedWord(SRC2[15:8]* SRC1[15:8]+SRC2[7:0]*SRC1[7:0])
// Repeat operation for 2nd through 15th word
DEST[255:240] := SaturateToSignedWord(SRC2[255:248]*SRC1[255:248]+ SRC2[247:240]* SRC1[247:240])
DEST[MAXVL-1:256] := 0

VPMADDUBSW（EVEX 编码版本）¶

(KL, VL) = (8, 128), (16, 256), (32, 512)
FOR j := 0 TO KL-1
    i := j * 16
    IF k1[j] OR *no writemask*
        THEN DEST[i+15:i] := SaturateToSignedWord(SRC2[i+15:i+8]* SRC1[i+15:i+8] + SRC2[i+7:i]*SRC1[i+7:i])
        ELSE
            IF *merging-masking* ; merging-masking
                THEN *DEST[i+15:i] remains unchanged*
                ELSE *zeroing-masking*
                        ; zeroing-masking
                    DEST[i+15:i] = 0
            FI
    FI;
ENDFOR;
DEST[MAXVL-1:VL] := 0
