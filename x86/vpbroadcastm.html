VPBROADCASTM — 广播掩码到向量寄存器

操作码/指令 Op/En 64/32位模式支持 CPUID 功能标志 描述
EVEX.128.F3.0F38.W1 2A /r VPBROADCASTMB2Q xmm1, k1 RM V/V AVX512VL AVX512CD 将 k1 中的低位字节值广播到 xmm1 中的两个位置。
EVEX.256.F3.0F38.W1 2A /r VPBROADCASTMB2Q ymm1, k1 RM V/V AVX512VL AVX512CD 将 k1 中的低位字节值广播到 ymm1 中的四个位置。
EVEX.512.F3.0F38.W1 2A /r VPBROADCASTMB2Q zmm1, k1 RM V/V AVX512CD 将 k1 中的低位字节值广播到 zmm1 中的八个位置。
EVEX.128.F3.0F38.W0 3A /r VPBROADCASTMW2D xmm1, k1 RM V/V AVX512VL AVX512CD 将 k1 中的低位字值广播到 xmm1 中的四个位置。
EVEX.256.F3.0F38.W0 3A /r VPBROADCASTMW2D ymm1, k1 RM V/V AVX512VL AVX512CD 将 k1 中的低位字值广播到 ymm1 中的八个位置。
EVEX.512.F3.0F38.W0 3A /r VPBROADCASTMW2D zmm1, k1 RM V/V AVX512CD 将 k1 中的低位字值广播到 zmm1 中的十六个位置。

指令操作数编码¶

Op/En 操作数 1 操作数 2 操作数 3 操作数 4
RM ModRM:reg (w) ModRM:r/m (r) N/A N/A

描述¶

将源操作数（第二操作数）低字节/字的零扩展 64/32 位值广播到目的操作数（第一操作数）的每个 64/32 位元素。源操作数是 opmask 寄存器。目的操作数是 ZMM 寄存器 (EVEX.512)、YMM 寄存器 (EVEX.256) 或 XMM 寄存器 (EVEX.128)。

EVEX.vvvv 是保留的，必须为 1111b，否则指令会导致 #UD。

操作¶

VPBROADCASTMB2Q¶

(KL, VL) = (2, 128), (4, 256), (8, 512)
FOR j := 0 TO KL-1
    i := j*64
    DEST[i+63:i] := ZeroExtend(SRC[7:0])
ENDFOR
DEST[MAXVL-1:VL] := 0

VPBROADCASTMW2D¶

(KL, VL) = (4, 128), (8, 256),(16, 512)
FOR j := 0 TO KL-1
    i := j*32
    DEST[i+31:i] := ZeroExtend(SRC[15:0])
ENDFOR
DEST[MAXVL-1:VL] := 0
