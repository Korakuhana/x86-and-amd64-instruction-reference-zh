描述¶

此指令使用三个源操作数对 FP16 值执行打包乘加（偶数元素）或乘减（奇数元素）计算，并将结果写入目的操作数。目的操作数也是第一个源操作数。记号“132”、“213”和“231”表示操作数在 A * B ± C 中的使用方式，其中每个数字对应操作数编号，目的操作数为操作数 1；请参阅 表 5-8。

目的元素根据写掩码进行更新。

操作¶

VFMSUBADD132PH DEST, SRC2, SRC3 (EVEX 编码版本) 当 src3 操作数为寄存器时¶

VL = 128, 256 or 512
KL := VL/16
IF (VL = 512) AND (EVEX.b = 1):
    SET_RM(EVEX.RC)
ELSE
    SET_RM(MXCSR.RC)
FOR j := 0 TO KL-1:
    IF k1[j] OR *no writemask*:
        IF *j is even*:
            DEST.fp16[j] := RoundFPControl(DEST.fp16[j]*SRC3.fp16[j] + SRC2.fp16[j])
        ELSE:
            DEST.fp16[j] := RoundFPControl(DEST.fp16[j]*SRC3.fp16[j] - SRC2.fp16[j])
    ELSE IF *zeroing*:
        DEST.fp16[j] := 0
    // else dest.fp16[j] remains unchanged
DEST[MAXVL-1:VL] := 0
