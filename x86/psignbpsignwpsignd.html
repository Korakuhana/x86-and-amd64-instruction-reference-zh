描述¶

(V)PSIGNB/(V)PSIGNW/(V)PSIGND 如果源操作数（第二个操作数）中对应数据元素的有符号整数值小于零，则对目的操作数（第一个操作数）的每个数据元素求反（取相反数）。如果源操作数中某个数据元素的有符号整数值为正，则目的操作数中对应的数据元素保持不变。如果源操作数中的某个数据元素为零，则目的操作数中的对应数据元素将被设置为零。

(V)PSIGNB 操作有符号字节。(V)PSIGNW 操作 16 位有符号字。(V)PSIGND 操作有符号 32 位整数。

传统 SSE 指令：两个操作数都可以是 MMX 寄存器。在 64 位模式下，使用 REX 前缀可以访问其他寄存器。

128位传统 SSE 版本：第一个源操作数与目的操作数是 XMM 寄存器。第二个源操作数是 XMM 寄存器或 128 位内存位置。对应 YMM 目的寄存器的位（MAXVL-1:128）保持不变。

VEX.128 编码版本：第一个源操作数与目的操作数是 XMM 寄存器。第二个源操作数是 XMM 寄存器或 128 位内存位置。目的 YMM 寄存器的位（MAXVL-1:128）清零。VEX.L 必须为 0，否则指令将产生 #UD。

VEX.256 编码版本：第一个源操作数与目的操作数是 YMM 寄存器。第二个源操作数是 YMM 寄存器或 256 位内存位置。

操作¶

def byte_sign(control, input_val):
    if control<0:
        return negate(input_val)
    elif control==0:
        return 0
    return input_val
def word_sign(control, input_val):
    if control<0:
        return negate(input_val)
    elif control==0:
        return 0
    return input_val
def dword_sign(control, input_val):
    if control<0:
        return negate(input_val)
    elif control==0:
        return 0
    return input_val
