描述¶

注意：VEX.vvvv 和 EVEX.vvvv 是保留的，必须为 1111b，否则指令将产生 #UD 异常。

EVEX.512 编码版本：

将 512 位打包双精度浮点值从源操作数（第二个操作数）移动到目的操作数（第一个操作数）。此指令可用于从 float64 内存位置加载 ZMM 寄存器，或将 ZMM 寄存器的内容存储到内存中。目的操作数根据写掩码进行更新。

VEX.256 编码版本：

将 256 位打包双精度浮点值从源操作数（第二个操作数）移动到目的操作数（第一个操作数）。此指令可用于从 256 位内存位置加载 YMM 寄存器，将 YMM 寄存器的内容存储到 256 位内存位置，或在两个 YMM 寄存器之间移动数据。目的寄存器的位 (MAXVL-1:256) 被清零。

128 位版本：

将 128 位打包双精度浮点值从源操作数（第二个操作数）移动到目的操作数（第一个操作数）。此指令可用于从 128 位内存位置加载 XMM 寄存器，将 XMM 寄存器的内容存储到 128 位内存位置，或在两个 XMM 寄存器之间移动数据。

128 位传统 SSE 版本：对应目的寄存器的位 (MAXVL-1:128) 保持不变。

当源操作数或目的操作数是内存操作数时，操作数可以不按 16 字节边界对齐，而不会导致产生一般保护性异常 (#GP)。

VEX.128 和 EVEX.128 编码版本：目的寄存器的位 (MAXVL-1:128) 被清零。

操作¶

VMOVUPD (EVEX Encoded Versions, Register-Copy Form)¶

(KL, VL) = (2, 128), (4, 256), (8, 512)
FOR j := 0 TO KL-1
    i := j * 64
    IF k1[j] OR *no writemask*
        THEN DEST[i+63:i] := SRC[i+63:i]
        ELSE
            IF *merging-masking*
                THEN *DEST[i+63:i] remains unchanged*
                ELSE DEST[i+63:i] := 0 ; zeroing-masking
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0

VMOVUPD (EVEX Encoded Versions, Store-Form)¶

(KL, VL) = (2, 128), (4, 256), (8, 512)
FOR j := 0 TO KL-1
    i := j * 64
    IF k1[j] OR *no writemask*
        THEN DEST[i+63:i] := SRC[i+63:i]
        ELSE *DEST[i+63:i] remains unchanged*
            ; merging-masking
    FI;
ENDFOR;

VMOVUPD (EVEX Encoded Versions, Load-Form)¶

(KL, VL) = (2, 128), (4, 256), (8, 512)
FOR j := 0 TO KL-1
    i := j * 64
    IF k1[j] OR *no writemask*
        THEN DEST[i+63:i] := SRC[i+63:i]
        ELSE
            IF *merging-masking*
                THEN *DEST[i+63:i] remains unchanged*
                ELSE DEST[i+63:i] := 0 ; zeroing-masking
            FI
    FI;
ENDFOR
DEST[MAXVL-1:VL] := 0

VMOVUPD (VEX.256 Encoded Version, Load - and Register Copy)¶

DEST[255:0] := SRC[255:0]
DEST[MAXVL-1:256] := 0

VMOVUPD (VEX.256 Encoded Version, Store-Form)¶

DEST[255:0] := SRC[255:0]

VMOVUPD (VEX.128 Encoded Version)¶

DEST[127:0] := SRC[127:0]
DEST[MAXVL-1:128] := 0

MOVUPD (128-bit Load- and Register-Copy- Form Legacy SSE Version)¶

DEST[127:0] := SRC[127:0]
DEST[MAXVL-1:128] (Unmodified)

(V)MOVUPD (128-bit Store-Form Version)¶

DEST[127:0] := SRC[127:0]
